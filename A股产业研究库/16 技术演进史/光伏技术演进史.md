# 光伏技术演进史

## 技术起源

光伏效应（Photovoltaic Effect）的发现可以追溯到1839年，法国物理学家A.E. Becquerel在电解液中观察到光照能产生电压。1954年，贝尔实验室的Chapin、Fuller和Pearson制造出第一块实用的硅太阳电池，效率约6%。这个起点拉开了光伏产业的序幕。

晶体硅太阳能电池的物理基础是PN结的光电效应。当光子能量大于硅的禁带宽度（1.12 eV），可以激发电子从价带跃迁到导带，产生电子-空穴对。PN结的内建电场将电子和空穴分离，形成光生电流。输出功率取决于电压（Voc，开路电压）和电流（Isc，短路电流）。

光伏技术的演进本质上是三件事的循环：**光吸收效率**的改善（更少的反射损失、更宽的光谱响应），**载流子收集效率**的提升（更少的复合损失），以及**成本的不断下降**（SWANSON定律——光伏组件价格每累计翻倍产能下降约20%，与摩尔定律形成对照）。技术代际的划分以电池结构变化和效率突破为标志。

## 第一代方案

### 铝背场电池（Al-BSF，1954-2010年代）

铝背场电池是第一代晶硅太阳电池的经典结构。其结构特征是：正面是硅片表面的N⁺发射极，覆盖减反射层和银栅线电极；背面是全面覆盖的铝层，烧结形成P⁺背电场（Back Surface Field，Al-BSF），通过杂质浓度梯度产生的内建电场降低背表面的载流子复合。

典型参数：单晶效率17-19%，多晶效率15-17%，硅片厚度180-200μm，铝背场厚度约10-30μm。

代表产品：尚德电力、英利、天合光能在2010年以前主要生产Al-BSF电池。2011年，单晶Al-BSF组件的量产效率约17-18%，多晶约15.5-16.5%。这一时期的全球光伏装机量增长主要由德国（EEG法案）和欧洲市场驱动。

Al-BSF的优点是工艺简单、成熟（只需要5-6道工序：制绒、扩散、刻蚀、镀膜、丝网印刷、烧结），设备国产化程度高。但它的效率天花板约20%，因为全背面铝层造成严重的复合损失——铝背场无法实现有效的表面钝化。

### 多晶vs单晶路线之争（2005-2015）

2010年代中期以前，多晶硅片凭借成本优势占据主导地位。多晶硅片用铸锭法制备（成本低于直拉单晶），电池效率低1-2个百分点，但成本低10-20%。2014年多晶组件全球市场份额约70-80%，单晶仅约20-30%。

转机出现在2015年。隆基绿能等中国企业大幅降低了单晶拉棒的切割和成本，同时PERC技术与单晶结合后的效率增益更显著。单晶的晶体缺陷密度远低于多晶，表面钝化效果更好，效率天花板更高。加上金刚线切割技术的引入（替代传统砂浆切割），单晶硅片的切割成本降幅超过50%。2017年单晶全球份额首次超过多晶，此后多晶逐渐边缘化。到2023年，多晶硅片市占率已降至不足3%。

## 第二代方案

### PERC（Passivated Emitter and Rear Cell，2015-2023）

PERC电池是光伏行业历史上最具影响力的中期技术。它由澳大利亚新南威尔士大学Martin Green团队在1983年首次提出，但真正量产化是2015年以后，原因在于背面钝化膜和激光开槽工艺的成熟。

PERC与Al-BSF的核心区别在背面。PERC在硅片背面覆盖一层钝化膜（Al₂O₃/SiNₓ叠层），然后通过激光开槽在钝化膜上形成局部接触孔，铝浆通过孔洞与硅形成接触。这使得背面的复合速率从Al-BSF的约1000 cm/s降低到数十cm/s级别。

| 参数 | Al-BSF | PERC |
|---|---|---|
| 量产效率 | 17-19% | 20-22.5% |
| 开路电压Voc | ~630 mV | ~670 mV |
| 短路电流Jsc | ~37 mA/cm² | ~40 mA/cm² |
| 填充因子FF | ~78% | ~81% |
| 背面钝化 | 无（Al背场） | Al₂O₃/SiNₓ叠层 |
| 制造工序 | 5-6道 | 8-10道 |

代表产品：隆基Hi-MO系列（单晶PERC，2017年起效率从21%升到22.5%+），晶科Eagle系列（多晶PERC，效率约19.5-20.5%）。2018-2022年间，PERC电池是全球绝对主流，2022年市占率超过90%。

PERC的意义在于它是一条"低投资换高效率"的路线——改造Al-BSF产线的成本约为新建的1/3，PERC产线的投资回收期仅1-2年。这个经济性推动了全球光伏产线的快速PERC化。

PERC的效率天花板约23-24%，受限于两类复合损失：正面发射极的俄歇复合（Auger Recombination）和背面接触区的局部高复合。要进一步突破就需要更先进的钝化接触技术。

### 单晶vs多晶的终局

PERC时代彻底终结了多晶的生命力。单晶PERC效率（22-22.5%）大幅领先多晶PERC（19.5-20.5%），加上单晶金刚线切割成本逼近多晶铸锭，单晶的经济性碾压了多晶。2020年后主要厂商均已放弃多晶PERC产线。市场只剩单晶。

## 当前主流方案

### TOPCon（Tunnel Oxide Passivated Contact，2023-）

TOPCon电池2013年由Fraunhofer ISE研究所提出。核心结构是在硅片背面沉积一层超薄氧化硅（SiOₓ，约1-2nm），然后覆盖一层掺杂多晶硅（Poly-Si）。氧化层起到隧穿钝化效应——空穴通过量子隧穿通过氧化层，而电子被氧化层阻挡，实现对背面的优异钝化。这种结构被称为"选择性载流子接触"。

| 参数 | PERC（典型值） | TOPCon（典型值） |
|---|---|---|
| 量产效率 | 22.0-22.5% | 24.5-25.5% |
| 开路电压Voc | ~680 mV | ~710-720 mV |
| 短路电流Jsc | ~40.5 mA/cm² | ~42 mA/cm² |
| 温度系数 | -0.34%/°C | -0.30%/°C |
| 双面率 | 70-75% | 80-85% |
| 工序数 | 8-10道 | 12-14道 |

TOPCon的核心竞争力来自三个维度。一是效率增益明显，比PERC高2-3个百分点。二是双面率高，背面也能发电（双面组件在电站中额外增益5-15%）。三是与PERC产线兼容度高——从PERC升级到TOPCon，约60-70%的产线设备可复用（扩散炉、PECVD、丝印等），新增设备主要是LPCVD或PECVD沉积多晶硅层。这个升级路径大幅降低了厂商的迁移门槛。

代表产品：晶科N型TOPCon（Tiger Neo系列，最高效率25.8%），天合光能Vertex N（25.5%+），晶澳DeepBlue 4.0 Pro（25.4%）。TOPCon在2023年超越PERC成为新增产线主流，2024年全球新增产能中TOPCon占比超过70%。

TOPCon同样使用N型硅片（掺磷），替代了PERC时代的P型硅片（掺硼）。原因是N型硅对金属杂质容忍度更高，光致衰减更低（无B-O对引起的LID衰减）。

### HJT（Heterojunction with Intrinsic Thin Layer）

HJT电池是日本三洋公司（现松下）在1990年代发明的技术，核心结构是在N型单晶硅片的两侧各沉积一层本征非晶硅（a-Si:H i-layer），然后再沉积掺杂非晶硅层。非晶硅层对硅片表面形成优异钝化——本征非晶硅的界面缺陷密度极低（约10¹⁰ cm⁻²级别），远低于TOPCon的多晶硅接触。

| 参数 | HJT（典型值） |
|---|---|
| 量产效率 | 24.5-25.5% |
| 开路电压Voc | ~740-750 mV |
| 工序数 | 4-5道 |
| 温度系数 | -0.25%/°C |
| 双面率 | 90-95%+ |

HJT的优势很明显：工序最少（仅制绒、清洗、PECVD双面镀非晶硅层、PVD/RPD沉积TCO、丝印烧结），温度系数最低（在高温环境下性能衰减更小，适合沙漠电站），双面率最高（背面也能达到正面的90%+效率）。

但HJT的普及瓶颈是成本。制备非晶硅的PECVD设备技术门槛高，ITO透明导电氧化物（TCO）靶材中的铟属于稀有元素；产线与PERC不兼容，需要完全新建产线。PERC→TOPCon的升级投资约0.5-0.7亿元/GW，而PERC→HJT的新建投资约2.5-3.5亿元/GW。

代表产品：华晟新能源（HJT，量产效率25.2%+），东方日升（伏羲系列），REC Alpha纯黑系列（效率22.5%+，面向屋顶分布式市场）。HJT在分布式和户用市场因其美观和温度系数优势而占据一席之地。

### BC（Back Contact，背面接触电池）

BC电池将所有电极置于电池背面，正面完全没有金属遮光。这使BC电池的短路电流Jsc极高（可达43 mA/cm²+），外观一致性好。

BC分为两类。IBC（Interdigitated Back Contact）结构最复杂——P⁺和N⁺掺杂区在背面交叉排列。TBC（Tunnel Oxide Back Contact）则是TOPCon和BC的融合——背面用隧穿氧化层和多晶硅实现钝化接触。

代表产品：Maxeon的IBC电池（量产效率24-24.5%，面向高端屋顶市场），隆基Hi-MO 9（采用HPBC技术，量产效率24.5%+）。隆基在2023年底宣布全面转向BC技术路线，在TOPCon热潮中保持差异化策略。

BC电池的封装难度大、银浆用量多、制程控制严，但目前正向耐用电极设计和更低成本的银包铜浆料方向突破。

### N型替代P型

2023-2024年，N型硅片（掺磷/掺砷）全面替代P型硅片（掺硼）成为光伏行业的主旋律。原因是P型晶硅的B-O对在光照下会产生复合中心，引起光致衰减（LID），衰减幅度可达2-3%。N型没有B-O对，LID几乎为零，且少子寿命更长。到2024年，新增产能的90%以上使用N型硅片。

## 未来技术路线

### 钙钛矿/硅叠层电池

钙钛矿是当前光伏技术的最大热点。钙钛矿材料（ABX₃结构的金属卤化物，如CH₃NH₃PbI₃）具有优异的吸收系数（约10⁵ cm⁻¹，是晶体硅的3倍）和可调的带隙（1.5-2.3 eV），非常适合与晶体硅组成叠层电池。

叠层电池的原理是让宽带隙的钙钛矿（~1.7 eV）吸收高能光子，窄带隙的硅（1.12 eV）吸收低能光子，消除了单结电池的"热载流子弛豫损失"，理论上限可达理论Shockley-Queisser极限的约45-50%。

| 类型 | 实验室效率 | 产业化预期效率 |
|---|---|---|
| 单结钙钛矿 | 26.1%（UNIST, 2023） | 20-24% |
| 钙钛矿/硅叠层（两端） | 33.9%（隆基, 2023） | 28-32% |
| 钙钛矿/钙钛矿叠层（全钙钛矿） | 29.1%（南京大学, 2023） | 25-28% |

代表企业：晶科能源与牛津PV合作开发叠层电池，隆基在2023年以33.9%的效率创造了叠层世界纪录。协鑫光电、纤纳光电在单结钙钛矿产业化上走在前列。2024年，多家企业宣布建成钙钛矿中试线或百兆瓦级实验线。

钙钛矿的商业化核心瓶颈是**稳定性**和**面积放大**。钙钛矿材料在高温（85°C）、高湿（85%RH）和持续光照下容易分解。目前典型寿命仅数千小时，远低于晶硅组件的25年质保。大面积制备（>1m²）的均匀性也面临挑战，大面积电池的效率和填充因子显著低于小面积（0.1cm²）器件。

### 背接触异质结（HBC）和无主栅技术

HBC是HJT和IBC的融合——用HJT的非晶硅钝化实现高Voc，用IBC的背接触实现高Jsc。实验室效率已超过26%，但制程复杂度和设备成本使其离量产较远。

无主栅技术（SmartWire、0BB）通过用密集的细栅线（铜丝）直接接触电池的TCO层，无需焊接主栅，降低银用量、提高导电性和组件可靠性。无主栅与HJT搭配时经济性最好，已在华晟、东方日升等HJT产线上采用。

### 钙钛矿组件走向市场（2027-2030预期）

钙钛矿组件的量产出货预计在2027-2028年达到第一次规模化拐点。初期将以单结钙钛矿组件（用于轻量化和BIPV场景）和钙钛矿/晶硅叠层组件（用于大型电站）两条路线并行。当组件效率突破30%、工作寿命超过15年时，钙钛矿技术将真正挑战晶硅的主导地位。

## 核心瓶颈

**效率天花板接近理论极限**：单结晶体硅电池的理论效率极限（Shockley-Queisser极限）约29.4%。当前TOPCon和HJT的量产效率已接近25.5%，实验室效率约26-27%，距离理论极限仅2-3个百分点。进一步提升效率的边际成本急剧上升，每提升0.1%所需研发投入和技术难度大幅增加。

**成本下降速度放缓**：光伏的Swanson定律（成本每累计翻倍产能下降约20%）在硅料、硅片和组件环节已经接近物理极限。非硅成本（玻璃、胶膜、背板、铝边框、银浆）占比越来越高，而银浆成本（特别是正银的银粉价格）受全球银价影响明显。产业已经出现了"每瓦成本难以继续下降"的声音。

**银浆消耗和贵金属依赖**：TOPCon的正背银消耗约100-130mg/片，BC和HJT的银浆用量更高。银价波动（2024年突破30美元/盎司）直接影响电池成本。银包铜浆料和无主栅技术正在推进，但完全无银化方案（如铜电镀）在量产可靠性上还未充分验证。

**钙钛矿稳定性**：IEC 61215标准要求组件在85°C/85%RH条件下运行2000小时性能衰减<5%，目前钙钛矿组件很少能达到。温度循环、UV照射、湿气渗透是主要失效机制。封装材料和界面工程仍在快速迭代。

**产能过剩与行业洗牌**：2022-2024年间中国光伏行业经历了巨大的产能扩张。TOPCon新增产能（300GW+）远超实际需求，组件价格从2022年的1.9元/W下降到2024年的0.7-0.8元/W，全行业出现大面积亏损。产能过剩加速了落后产线的淘汰，但也挤压了新技术（HJT、钙钛矿）的投资空间。

## 产业影响

光伏技术演进的每一个代际都深刻影响了下游的投资决策和行业格局。

**效率提升驱动LCOE下降**：电池效率每提升1个百分点，光伏电站的LCOE（平准化度电成本）下降约4-6%。TOPCon对比PERC的2-2.5个百分点效率提升，意味着LCOE降幅约10-15%。这使得在多数地区光伏已成为最便宜的电力来源（约0.02-0.04美元/kWh）。

**中国光伏产业链的绝对主导**：中国企业在硅料、硅片、电池片、组件四个环节的全球产能占比分别约85%、97%、85%和80%。技术迭代（特别是PERC→TOPCon的迁移）进一步强化了中国企业在制造端的领先地位。隆基、晶科、天合、晶澳形成四强格局。

**N型时代的设备国产化**：TOPCon的关键设备（LPCVD、PECVD、激光开槽设备）已经基本实现国产化。捷佳伟创、先导智能、帝尔激光等中国设备商深度参与了TOPCon的产能扩张。钙钛矿涂布/镀膜设备也有多家国内企业布局。

**技术路线的分化投资风险**：大型厂商在TOPCon、HJT、BC三条路线上进行了不同方向的投资。错误的路线选择可能带来数百亿的沉没成本。例如隆基在2023年从PERC转向BC，放弃了其已有的TOPCon产能布局。晶科和天合在TOPCon上投入最大。这种分化意味着2025-2027年会有新一轮的技术路线洗牌。

**钙钛矿带来的产业重构可能**：钙钛矿一旦量产成功，将颠覆现有光伏产业链。晶硅产业链（硅料→拉棒/铸锭→切片→电池→组件）的硅片加工环节可能被跳过，代之的是玻璃镀膜→组件封装。这意味着连城数控、晶盛机电等硅片设备商面临转型压力，而玻璃深加工企业（福莱特、信义光能）可能获得新增长点。
