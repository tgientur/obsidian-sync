## 二、招聘方视角的自我定位

### 硬伤（需正视）

| 问题 | 影响 | 应对 |
|------|------|------|
| 学校双非 | 大厂/央企简历机筛不过 | 主攻中小厂，走内推 |
| GPA 3.0 | 金融岗敏感 | 简历不写，面试问到再说 |
| 无实习 | 最大弱点 | 暑假前必须搞定 |

### 优势

| 优势 | 面试价值 |
|------|---------|
| 省 A 类金奖（队长） | 证明带队+答辩+研究能力 |
| 自建信息筛选体系 | 差异化——多数金融学生做不到 |
| Python + AI 工具 + 独立部署 | 商科生里稀缺 |
| 持续关注市场 | 面试能讲出具体标的和判断逻辑 |

**一句话：面试型选手，不是简历型选手。** 优势在面聊，问题在纸面过不了关。

## 三、职业方向

### 主攻：中小厂数据分析 / BI 分析

- **地点：** 杭州
- **公司规模：** 50-500 人
- **匹配点：** Excel + Python + 金融背景 + 信息处理
- **需补充：** SQL（2 周刷 70 题）、1 个数据分析项目（用 A 股真实数据）

### 备选
- 地方国企（杭州本地国企对本地学校不歧视）
- 技术实施 / 解决方案（偏 SaaS 部署和客户沟通）

### 排除
- 大厂（简历机筛不过）
- 央企正式编（卡学校）
- 纯销售/客服（与技术含量目标不符）
