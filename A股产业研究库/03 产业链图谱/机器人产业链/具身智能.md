# 具身智能

## 定义

具身智能（Embodied Intelligence）是指能够在物理世界中感知、理解、推理并自主行动的智能系统。与传统AI（大语言模型、CV等"数字智能"）的核心区别在于：具身智能体拥有物理身体，能在真实三维空间中进行移动、操作和交互。

## 技术框架

```
感知（视觉/触觉/力觉/IMU） 
    ↓ 多模态融合
世界模型（空间理解 + 物理规律学习）
    ↓ 决策规划
灵巧操作/移动控制（全身运动控制）
    ↑
大语言模型提供高层语义理解和任务拆解
```

## 核心赛道

| 方向 | 代表公司 | 核心能力 |
|:----|:---------|:--------|
| 人形机器人本体 | 特斯拉Optimus、Figure AI、宇树科技、智元机器人 | 通用双足运动 + 灵巧手操作 |
| 灵巧手 | 因时机器人、腾讯Robotics X | 微型电机驱动的多自由度仿人手 |
| 世界模型 | 谷歌DeepMind（RT-2/RT-X）、李飞飞空间智能 | 从视觉输入直接生成动作 |
| 双臂协作 | 库卡、ABB | 高精度力控装配 |
| 具身大模型 | 谷歌PaLM-E、清华CoPa | 用LLM做任务规划，连接底层控制 |
数据来源：[1][2] 相关年报/行业报告

## 投资意义

具身智能是2024-2025年机器人领域最热的方向，核心催化来自：

1. **特斯拉Optimus量产预期**：2025-2026年逐步进入工厂应用
2. **Figure AI与OpenAI深度合作**：LLM+机器人，Figure 01已展示对话+操作能力
3. **国内政策推动**：北京、上海、深圳均出台具身智能专项政策
4. **AI赋能加速**：大模型让"通用机器人"从专用工具变为可能
