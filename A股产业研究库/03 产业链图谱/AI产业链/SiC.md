# SiC（碳化硅）

## 定义与产业定位

SiC（Silicon Carbide，碳化硅）是第三代宽禁带半导体的核心材料。与传统的硅（Si）材料相比，SiC具有更高的禁带宽度（3.26eV vs 1.12eV）、更高的击穿电场强度、更高的热导率和更高的电子饱和漂移速度。这些物理特性决定了SiC在高压、高频、高温应用场景下的性能显著优于硅基器件。

**核心价值**：SiC功率器件（MOSFET/肖特基二极管）相比硅基IGBT/Si MOSFET，可降低开关损耗50-80%（数据来源：[1]）、提升系统效率3-5%、缩小模块体积30-50%。在新能源汽车主驱逆变器、充电桩、光伏逆变器、数据中心电源等大功率场景中，SiC正加速替代传统硅基器件。

AI算力高速增长的背景下，SiC的价值有了新的维度——**AI数据中心的供电效率直接受功率半导体性能影响**。以GB200 NVL72机柜为例，单机柜功耗120kW，供电架构从传统的12V向48V/400V演进，SiC器件在高压DC-DC转换和UPS中发挥关键作用。

## 产业链位置

### 双产业链交叉

SiC同时涉及两条产业链，且核心制造环节重合：

**路径一：半导体产业链（制造端）**

```
SiC衬底 → SiC外延 → SiC芯片制造 → 功率封装 → 模块应用
 天岳先进    天岳先进     士兰微/华润微    SiC模块    逆变器/电源
```

**路径二：AI/新能源汽车产业链（应用端）**

```
SiC器件 → 主驱逆变器/OBC/DC-DC → 新能源车电驱系统/充电桩
SiC器件 → 服务器电源/UPS → AI数据中心供电
SiC器件 → 光伏逆变器/储能变流器 → 新能源发电
```

**产业定位**：SiC属于**功率半导体**细分赛道，与IGBT同属"电力电子核心器件"。但SiC不是IGBT的直接替代——SiC MOSFET在中高压（600-1700V）和高频场景有显著优势，而IGBT在低成本/低频率场景仍有长期生存空间。

**交叉说明**：本文件放在AI产业链路径下，是因为SiC在AI数据中心供电环节（高压DC-DC/UPS）的应用是新增结构增量，且与GPU/服务器功耗增长直接相关。[[A股产业研究库/03 产业链图谱/半导体产业链/SiC|半导体产业链的SiC节点]]聚焦衬底/外延/器件制造端的国产替代逻辑。

## 核心技术

### SiC vs Si vs GaN 对比

| 参数 | Si | SiC | GaN |
|:----|:--:|:---:|:---:|
| 禁带宽度（eV） | 1.12 | 3.26 | 3.44 |
| 击穿电场（MV/cm） | 0.3 | 2.8 | 3.3 |
| 电子迁移率（cm²/Vs） | 1500 | 900 | 2000 |
| 热导率（W/cm·K） | 1.5 | 4.9 | 1.3 |
| 适用电压范围 | ≤900V | 600-1700V | ≤650V |
| 成本（相对） | 1x | 3-5x | 2-3x |
数据来源：[1][2] 相关年报/行业报告

### SiC成本结构（以SiC MOSFET为例）

| 成本项 | 占比 | 说明 |
|:------|:---:|:-----|
| SiC衬底 | 40-50% | 衬底成本占比最高，良率和尺寸是关键 |
| SiC外延 | 10-15% | 外延缺陷密度直接影响器件性能 |
| 芯片制造 | 20-25% | 离子注入/高温退火/栅氧化等特殊工艺 |
| 封装/测试 | 15-20% | 高温/高压封装，可靠性要求极高 |
数据来源：[1][2] 相关年报/行业报告

**核心矛盾**：SiC衬底的良率和产能是产业链最大瓶颈——6英寸SiC衬底良率仅60-70%（硅片>95%），8英寸衬底良率更低。衬底成本下降速度决定了SiC渗透率提升速度。

## 投资逻辑简析

1. **新能源车主驱渗透加速**：800V高压平台是SiC上车的最强驱动力。800V平台逆变器必须用SiC MOSFET（硅IGBT在800V下的开关损耗过大）。2026年国内800V车型渗透率预计达到25-30%（数据来源：[1]），对应SiC MOSFET需求翻倍。

2. **AI数据中心供电新需求**：AI算力增长拉动数据中心功率密度提升，UPS和DC-DC转换中SiC的渗透率从2024年的5%（数据来源：[1]）提升至2026年的20%。英维克/高澜的液冷方案需要搭配高效率的SiC电源。

3. **国产替代窗口**：SiC衬底/外延/器件制造几乎全链条被海外封锁（Wolfspeed/ST/英飞凌对华出口受限）。天岳先进8英寸导电型衬底突破意味着国产SiC产业链的核心瓶颈正在被攻克。

4. **成本下降的S型曲线**：SiC衬底市场价格每年下降10-15%（数据来源：[1]），当SiC MOSFET成本接近硅IGBT的2-3倍时（预计2026-2027年），SiC在大量场景将进入性价比拐点。

**风险提示**：SiC赛道技术迭代快，Wolfspeed/ST的8英寸良率优势可能拉开差距；国内SiC衬底良率和一致性仍有差距，客户导入周期长。

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期
