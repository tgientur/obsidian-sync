# 金融科技、银行风险与市场挤出效应

- **作者**：金洪飞、李弘基、刘音露
- **期刊**：《财经研究》2020年第46卷第5期
- **关键词**：金融科技；银行风险；小微企业贷款；挤出效应
- **DOI**：10.16538/j.cnki.jfe.2020.05.004

---

## 研究问题

金融科技运用对不同类型银行风险的异质性影响？大银行运用金融科技是否对中小银行产生"挤出效应"？

## 数据与方法

- **样本**：261家国内银行，2010—2018年
- **核心解释变量**：**银行个体层面的金融科技运用程度指标**——用Python网络爬虫技术改进文本挖掘法，收集每家银行在新闻报道中被提及的次数
- **被解释变量**：银行风险水平（RiskCa，风险资产占比）、核心一级资本充足率
- **模型**：面板固定效应模型

## 核心发现

### 1. 金融科技降低银行风险，但异质性显著

| 银行类型 | 金融科技对风险的影响 |
|---------|-------------------|
| 所有银行 | 显著降低风险水平 |
| 大型银行 | 风险显著下降，改善明显 |
| 中小银行 | 改善程度较弱 |

### 2. 市场挤出效应（核心发现）

**大银行运用金融科技的行为刺激了中小银行风险水平的上升。**

| 关系 | 发现 |
|------|------|
| 大银行FinTech→大银行风险 | 不显著（风险可控） |
| 大银行FinTech→中小银行风险 | 显著升高（竞争效应） |
| 大银行FinTech→大银行小微贷款 | 显著增加 |
| 大银行FinTech→中小银行小微贷款 | 被动增加但风险上升 |

### 3. 机制解释

> **大银行凭借资金成本优势+金融科技降低信息不对称，抢占中小银行的优质低风险小微企业客户。中小银行优质客户流失后，只能"业务下沉"拓展风险更高的客户，导致风险水平上升。**

- 金融科技缩小了大银行与中小银行在获取软信息方面的能力差距
- 大银行原来受限于信息劣势无法服务小微企业，FinTech弥补了这一短板
- 大银行提供更低利率，吸引中小银行的头部小微客户

### 4. 理论基础

论文建立了**贷款定价模型**：

$$P_L = r_f + TC_L + R_L(q_L, θ)$$
$$P_S = r_f + TC_S + R_S(q_S, θ)$$

- L=大银行，S=中小银行
- θ ∈ [0,1] 表示软信息比重（企业不透明程度）
- FinTech水平η↑ → 软信息生产成本f'(η)↓
- 大银行资金成本优势 → 报价更低 → 吸引中小银行的优质客户

## 方法论亮点

1. **网络爬虫改进文本挖掘法**：从新闻文本中提取每家银行个体的FinTech运用程度（而非全国/省级指数），这是对沈悦郭品(2015)方法的重要改进
2. **挤出效应的定量识别**：首次定量证明大银行FinTech运用对中小银行的负面溢出
3. 稳健性检验充分：标准化处理、银行重新分组、滞后一期、GMM

## 对学年论文的参考价值

1. **"市场挤出效应"是对FinTech研究的独特贡献**——大多数文献只讨论FinTech的正面效应，这篇揭示了负面影响
2. 可用于文献综述中讨论FinTech对银行间竞争的"马太效应"
3. 银行个体层面的FinTech指标构建方法值得参考
4. 小微贷款作为机制检验的桥接变量设计巧妙

## 局限性

- 大银行界定标准（资产规模前15位）有一定主观性
- 新闻爬虫指标可能受银行知名度影响（规模大的银行天然更容易被报道）
- 数据截至2018年，未覆盖后续变化
