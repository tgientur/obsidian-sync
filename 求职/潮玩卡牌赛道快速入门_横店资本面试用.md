## 二、收藏卡牌赛道（横店资本重点布局）

### 关键公司：杰森娱乐集团（JASON）
- 横店资本2025年6月战略投资
- 旗下子品牌：
  - **集卡社**（Jikashe）：核心卡牌品牌
  - **森罗万象**：潮玩/IP衍生品
  - **造极工作室**：精品雕像/手办
- 覆盖500+IP合作（迪士尼、光线传媒、国漫等）
- 商品覆盖全国超100,000家线下终端

### 现象级案例：《哪吒2》收藏卡
- 2025年春节上映的《哪吒之魔童闹海》票房破百亿
- 集卡社推出电影典藏版收藏卡牌
- **上线一周全渠道销售额破1亿元**
- 同时登上天猫、抖音、B站、小红书收藏卡类目销量第一
- 被称为"2025年春节档最受欢迎IP衍生品"

### 卡牌赛道趋势
- 收藏卡牌从小众破圈成大众消费，Z世代为主要消费群体
- 头部公司：**卡游（Kayou）**（奥特曼卡牌）、**集卡社**（国漫IP）
- 卡牌+盲盒模式：拆卡的不确定性和稀有卡驱动复购
- 二手交易活跃，稀有卡溢价空间大

## 五、Z世代消费趋势关键词

| 关键词 | 解释 |
|:---|:---|
| **情绪价值** | 为"开心""治愈""陪伴"买单，不只看功能 |
| **IP经济** | 愿意为喜欢的IP角色付费，衍生品市场爆发 |
| **盲盒经济** | 不确定性驱动购买，拆盒的刺激感 |
| **收藏价值** | 从消费到收藏，二手交易流通 |
| **谷子经济** | （Goods的谐音）二次元周边，徽章/立牌/吧唧 |
| **出海** | 泡泡玛特在东南亚/欧美爆火，中国潮玩走向全球 |
| **AI+IP** | AI生成内容降低IP创作门槛，个性化定制 |
