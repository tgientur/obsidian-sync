# 先进封装

## 定义与产业定位

先进封装（Advanced Packaging）是指超越传统引线键合（Wire Bonding）的封装技术体系，核心特征是在封装层面实现更高密度的互联、更短的信号路径和更优的散热性能。与只承担"保护+连线"功能的传统封装不同，先进封装已成为延续摩尔定律的关键路径——在晶体管微缩趋近物理极限的背景下，通过异构集成（Heterogeneous Integration）在封装层面提升芯片性能。

**核心技术体系**：

| 技术 | 说明 | 应用场景 |
|:----|:-----|:---------|
| CoWoS（Chip-on-Wafer-on-Substrate） | 芯片通过硅中介层（Si Interposer）互联，TSV导通至基板 | GPU+HB M异构集成，NVIDIA H100/B200标配 |
| 2.5D封装 | 芯片并排放置在硅中介层上，通过微凸块+TSV互联 | AI芯片、HPC处理器 |
| 3D封装 | 芯片垂直堆叠，通过TSV直接上下互联 | HBM堆叠、CIS传感器 |
| Fan-Out（扇出型封装） | 芯片嵌入模塑层，RDL重分布向外扇出 | 射频前端、电源管理IC |
| Chiplet（芯粒） | 将大芯片拆分为多个小芯粒，通过Die-to-Die互联集成 | 大规模SoC、AI加速器 |
数据来源：[1][2] 相关年报/行业报告

**产业定位**：先进封装是从"制造后道工序"升级为"AI芯片性能的决定性环节"。AI大模型对算力的持续需求推动GPU面积不断增大（B200 Die size超800mm²），单晶圆良率制约迫使GPU采用Chiplet架构+先进封装方案。CoWoS产能已成为AI芯片出货量的硬约束——台积电CoWoS产能供不应求的程度决定了NVIDIA/AMD的GPU出货上限。

## 产业链位置

```
上游（封装材料/设备） → 先进封装代工 → AI芯片/SoC → 系统集成
      ↑                       ↑               ↑
  硅中介层/TSV            CoWoS产能          Chiplet设计
  EMC/Underfill           Fan-Out/WLCSP      Die-to-Die接口
  封装基板(FC-BGA)       3D堆叠
```

先进封装承上启下：上游连接封装材料（华海诚科的EMC/Underfill、联瑞新材的球硅填料、深南电路的FC-BGA载板）和封装设备（日本Disco/东京电子、国产芯源微）；下游服务AI芯片（NVIDIA/AMD/寒武纪）、HPC、消费电子SoC。

### 关键技术指标

- **互连密度**：CoWoS的微凸块间距已从40μm缩小至25μm，下一代目标<10μm
- **TSV孔径**：从10μm缩小至1-3μm，深宽比从10:1提升至20:1
- **中介层尺寸**：从1倍光罩（约800mm²）提升至3倍光罩（约2400mm²），支持更大芯片集成
- **热管理**：多芯片热耦合效应要求更高效的散热方案，液冷+先进封装绑定

## 投资逻辑简析

1. **AI芯片的"产能瓶颈"**：CoWoS产能紧缺是当前AI产业链最大的供给制约因素。台积电CoWoS产能从2024年30万片/年扩至2025年60万片/年，但仍无法满足NVIDIA/AMD订单需求。产能紧张=封装代工涨价=封装企业利润率提升。

2. **Chiplet带来的结构增量**：AI芯片从单芯片向Chiplet架构演进（B200由两颗Die组成，Rubin将集成更多Die），意味着每颗GPU需要的封装价值量持续提升——先进封装在芯片总成本中的占比从10%（数据来源：[1]）向20-30%演进。

3. **国产替代确定性**：先进封装设备（减薄机/划片机/临时键合机）国产化率<10%（数据来源：[1]），材料和基板也高度依赖日美供应商。大基金三期3440亿中先进封装是重点投资方向。

4. **投资节奏**：当前阶段（2026H1）关注CoWoS产能扩张节奏→封装材料订单确认→长电/通富先进封装收入占比提升。先进封装业绩兑现周期晚于GPU/光模块约2-3个季度。

**风险提示**：台积电CoWoS一家独大的格局短期不会改变，大陆先进封装与台积电差距2-3代以上，高端客户获取存在认证壁垒。

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期
