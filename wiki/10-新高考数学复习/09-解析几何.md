# 解析几何

> 难度：★★★★★ | 分值：17~27 分 | 题型：单选/多选 + 解答题第18题（压轴题）
> 考查特点：新高考 I 卷中**计算量最大**的板块。小题考性质，大题考"联立方程 + 韦达定理 + 几何条件代数化"

---

## 一、核心考点

### 1. 直线与圆

- **直线方程**：五种形式（点斜、斜截、两点、截距、一般式）
- **距离公式**：点到直线 $d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2+B^2}}$
- **圆的标准方程**：$(x-a)^2 + (y-b)^2 = r^2$
- **直线与圆位置关系**：$d < r$ 相交，$d = r$ 相切，$d > r$ 相离
- **弦长公式**：$|AB| = 2\sqrt{r^2 - d^2}$

### 2. 圆锥曲线定义

| 曲线 | 定义 | 标准方程 |
|------|------|---------|
| 椭圆 | 到两焦点距离之和为常数 | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ |
| 双曲线 | 到两焦点距离之差为常数 | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ |
| 抛物线 | 到焦点与到准线距离相等 | $y^2 = 2px$ |

### 3. 核心运算框架

- **联立方程**：直线 $y = kx + m$ 代入圆锥曲线方程
- **韦达定理**：$x_1 + x_2$、$x_1x_2$ 的表达式
- **判别式**：$\Delta > 0$（相交），$\Delta = 0$（相切），$\Delta < 0$（相离）
- **弦长公式**：$|AB| = \sqrt{1+k^2} \cdot \sqrt{(x_1+x_2)^2 - 4x_1x_2}$
- **面积表达**：三角形面积 $S = \frac12 |\text{底}||\text{高}|$ 或坐标公式

## 三、常用技巧

### 技巧 1：点差法（解决弦中点问题）
若 $A(x_1,y_1)$、$B(x_2,y_2)$ 在椭圆上，$M$ 为 $AB$ 中点：
$$\frac{y_1 - y_2}{x_1 - x_2} = -\frac{b^2 x_0}{a^2 y_0}$$

**推导**：$A, B$ 代入椭圆方程 → 两式相减 → 平方差公式 → 整理得斜率与中点坐标的关系。

**椭圆**：$k_{AB} = -\frac{b^2 x_0}{a^2 y_0}$（$x_0,y_0$ 为中点坐标）
**双曲线**：$k_{AB} = \frac{b^2 x_0}{a^2 y_0}$
**抛物线** $y^2 = 2px$：$k_{AB} = \frac{p}{y_0}$

> 点差法的好处：**不需要联立方程**，直接得到弦的斜率与中点坐标的关系。

### 技巧 2：联立运算的"降维模板"
解析几何大题 90% 的运算量在联立化简。**标准化流程**：

1. 设直线 $l: y = kx + m$（注意斜率不存在的情况要单独讨论）
2. 代入圆锥曲线方程
3. 消去 $y$，得到关于 $x$ 的二次方程
4. 写出韦达定理（$x_1+x_2$，$x_1x_2$）
5. 写出 $\Delta > 0$ 的条件（参数范围由此得出）
6. 将题目条件翻译成 $x_1+x_2$ 和 $x_1x_2$ 的表达式
7. 代入韦达定理求解

**关键**：把"垂直""平分""共线""面积比"等几何条件用 $x_1+x_2$ 和 $x_1x_2$ 表示。

### 技巧 3：设直线不设斜率
当直线过 $x$ 轴上定点 $(t, 0)$ 时，设 $x = ny + t$ 代入：
- 避免讨论斜率不存在的情况
- 计算量通常比 $y = k(x-t)$ 更小

### 技巧 4：离心率的计算技巧
- **椭圆离心率** $e = \frac{c}{a} = \sqrt{1 - \frac{b^2}{a^2}}$（$0 < e < 1$）
- **双曲线离心率** $e = \frac{c}{a} = \sqrt{1 + \frac{b^2}{a^2}}$（$e > 1$）
- **利用定义**：通过几何关系求 $a$ 和 $c$ 的比值

### 技巧 5：抛物线焦点弦的性质（选填秒杀）
抛物线 $y^2 = 2px$ 中，过焦点 $F(\frac{p}{2}, 0)$ 的弦 $AB$：
- $x_1 x_2 = \frac{p^2}{4}$，$y_1 y_2 = -p^2$
- $|AB| = x_1 + x_2 + p$
- $\frac{1}{|AF|} + \frac{1}{|BF|} = \frac{2}{p}$（定值）
- 以 $AB$ 为直径的圆与准线相切
- $\angle AFB = \pi - (\angle AFx + \angle BFx)$ 恒为直角

## 四、二级结论

1. **椭圆/双曲线焦点三角形面积**：
   $P$ 在椭圆上，$\triangle PF_1F_2$ 面积 $S = b^2 \tan\frac{\theta}{2}$（$\theta = \angle F_1PF_2$）
   双曲线：$S = b^2 \cot\frac{\theta}{2}$

2. **圆锥曲线的切线方程**（已知切点 $(x_0,y_0)$）：
   - 椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$：$\frac{x x_0}{a^2} + \frac{y y_0}{b^2} = 1$
   - 双曲线 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$：$\frac{x x_0}{a^2} - \frac{y y_0}{b^2} = 1$
   - 抛物线 $y^2 = 2px$：$y y_0 = p(x + x_0)$
   
   **记住公式**：将 $x^2$ 换成 $x x_0$，$y^2$ 换成 $y y_0$，$x$ 换成 $\frac{x+x_0}{2}$

3. **椭圆的光学性质**：从一焦点发出的光线经椭圆反射后经过另一焦点

4. **双曲线的渐近线**：
   $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 的渐近线为 $y = \pm \frac{b}{a}x$

5. **抛物线顶点处的切线**：$y^2 = 2px$ 在原点处的切线为 $x = 0$（即 y 轴）

6. **椭圆中 $b^2$ 的几何意义**：$b$ 是短半轴长，也是焦点到对应准线的距离

7. **圆锥曲线系**（处理公共焦点的多个曲线有用）

## 五、易错清单

- [ ] 联立方程后忘记检查 $\Delta > 0$（这往往给出了参数范围）
- [ ] 椭圆中 $a > b > 0$，$c^2 = a^2 - b^2$；双曲线中 $c^2 = a^2 + b^2$，**不要混淆**
- [ ] 双曲线的渐近线方程和焦点位置关系
- [ ] 焦点弦公式只适用于过焦点的弦
- [ ] 设直线 $y = kx + m$ 时漏了 $k$ 不存在（竖直直线）的情况
- [ ] $x = ny + t$ 设法的 $\Delta$ 表达式不同
- [ ] 圆锥曲线大题的计算结果往往是整数或简单分数，算出了很复杂的数要检查


> **总评**：解析几何是"得计算者得天下"。真正拉开差距的不是思路，而是**在有限时间内把联立韦达定理算对**。建议每天练一道解析大题保持手感和熟练度。点差法和焦点弦性质是选填提速的关键。
