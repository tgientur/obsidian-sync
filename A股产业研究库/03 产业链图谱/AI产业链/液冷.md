# 液冷

## 定义与产业定位

液冷（Liquid Cooling）是以液体为介质的散热方案，核心原理是利用液体比热容和导热系数远高于空气的物理特性，将发热元件（GPU/CPU）的热量高效传递至冷却端。在当前AI算力爆发的背景下，液冷已从"可选节能方案"变为"强制散热手段"。

推动液冷从可选变刚需的核心驱动力是GPU功耗的指数级增长：NVIDIA H100 TDP（热设计功耗）约700W，B200约1000W，下一代Rubin预计达1200-1500W。风冷散热能力上限约为800-1000W/芯片（受限于空气比热容和散热器体积），这意味着B200及以上代际的GPU必须依赖液冷才能稳定运行。

**产业定位**：液冷是AI算力基础设施中的"散热瓶颈环节"，处于GPU/AI服务器与数据中心之间的技术桥梁位置。上游连接液冷板、管路、冷却液、CDU（冷量分配单元）、水泵、阀件等零部件；下游服务于AI服务器、数据中心、超算中心、智算中心。液冷的产业地位随GPU功耗提升而持续增强——当风冷不可行时，液冷从"选配"变为"必配"。

产业规模：根据IDC数据，2024年中国液冷数据中心市场约80亿元，预计2026年将超300亿元（CAGR约94%）。渗透率从2023年的约15-20%提升至2026年预计的50-60%。

## 产业链分析

### 价值流动与利润分配

```
上游（液冷板/管路/冷却液/CDU/泵阀） → 液冷系统集成 → 数据中心液冷方案 → AI服务器散热
       利润份额：零部件~35%            利润份额：系统集成~25%    利润份额：方案部署~40%
```

液冷产业链的利润分配呈现"两头大中间小"的特征——上游核心零部件（液冷板、CDU、泵阀）和下游部署运维利润份额较高，中游系统集成竞争激烈、毛利偏低。核心零部件中，液冷板（冷板式）和CDU（冷量分配单元）是利润最厚的环节。

### 定价权分析

1. **上游零部件**（液冷板/CDU/泵阀）：掌握定价权。液冷板需要与GPU/AI服务器的散热设计深度定制，技术壁垒高。CDU中的核心元件（微通道换热器、电子膨胀阀）由少数供应商（三花智控、丹佛斯等）掌控
2. **中游系统集成**：定价权弱。英维克、高澜股份等系统集成商数量多且竞争激烈，毛利率被压缩至20-30%（数据来源：[1]）
3. **下游部署运维**：定价权中等。液冷系统部署涉及数据中心改造，需要专业工程资质和经验积累

### 成本传导

液冷系统的初始投资在AI数据中心建设成本中占比约10-15%（传统风冷约3-5%），但运营层面液冷可将PUE从1.4降至1.1以下，年省电费约30-40%。因此，虽然液冷增加了初始投资（单机柜液冷改造成本约3-5万元），但全生命周期成本低于风冷。这种"初期投资增加但全周期成本降低"的成本结构有利于液冷推广。

### 产业分工

液冷产业的分工模式逐步从"全链条自己做"走向"专业化分工"：

- **综合方案商**（英维克/高澜股份/申菱环境）：提供液冷系统集成方案，覆盖设计→制造→部署→运维全链条
- **核心零部件商**（三花智控/盾安环境）：聚焦液冷阀件/换热器/泵，技术壁垒高，客户包括英维克等集成商及云厂商直采
- **冷却液供应商**：基于去离子水或氟化液，单吨价格几万至几十万，占有一定利润份额
- **云厂商/IDC自研**：阿里/字节/BAT部分自研液冷方案，与供应商深度联合开发

### 利润来源与现金流特点

- **毛利率**：系统集成20-30%，零部件35-50%，运维服务50-60%
- **净利率**：系统集成8-12%，零部件15-25%
- **现金流**：项目制模式，单项目金额几百万元到几千万元，回款周期受结算节点影响（设备进场30%+验收30%+最终验收40%），现金流波动较大
- **规模效应**：液冷目前仍处于"定制化"阶段，批量生产的规模效应尚未充分释放，随着标准化推进，成本有较大下降空间

## 行业壁垒分析

### 技术壁垒

液冷系统设计涉及热力学、流体力学、材料科学的交叉应用：
- **冷板设计**：需要精确匹配GPU芯片的发热热点分布，微通道/翅片结构优化，流阻与散热效率的平衡——设计经验积累需要3-5年工程实践
- **CDU控制**：冷量按需分配、闭环温控、防凝露控制，需要软硬件协同能力
- **冷却液兼容性**：液冷板材料（铜/铝/不锈钢）与冷却液的电化学腐蚀控制，长期运行可靠性验证
- **浸没式液冷**：电子元件与冷却液长期接触的介电性能、材料溶胀、寿命验证，复杂度高于冷板式

### 客户壁垒

液冷方案涉及GPU服务器的散热适配和可靠性——与AI服务器的耦合程度极高：
- NVIDIA的GPU散热设计规格（热阻/流阻/接口）需要液冷方案商通过认证（NVIDIA认证周期6-12个月）
- 云厂商液冷方案的验证流程：首个样品测试（1-2月）→小批量部署验证（3-6月）→大规模部署（6-12月）。切换供应商成本极高
- 一旦液冷方案被验证并纳入数据中心设计方案，后续扩建大概率延续同一方案

### 资本壁垒

液冷领域进入资金门槛相对低于GPU/芯片设计，但也不低：
- 建立热测试实验室、风洞、流阻测试平台需投入5000万元-1亿元
- 建设生产线（钎焊炉、焊接线、检测线）需1-2亿元
- 项目前期需要垫资（设备采购+原材料+人工），占压资金量大

### 供应链壁垒

核心零部件供应集中度高：
- 微通道换热器：主要供应商为三花智控/丹佛斯/松下，国产替代率低
- 电子膨胀阀/温控阀：三花智控/盾安环境等少数企业掌握核心工艺
- 水泵（磁力泵/屏蔽泵）：格兰富/威乐等外资品牌主导，国产替代空间大
- 冷却液（氟化液）：3M退出后，国产替代方案（巨化股份等）处于验证期

### 规模壁垒

液冷系统的规模效应体现在两个方面：一是方案复用——为一家云厂商开发成熟的液冷方案可直接复制到该客户的新项目中，边际成本递减；二是供应链议价——批量采购液冷板/CDU核心件可降低单位成本。目前液冷市场仍处于渗透率快速提升期，先发企业通过项目经验积累构筑护城河。

## 历史行情复盘

### 第一期：2023年——液冷概念从0到1

**事件**：2023年H100大规模出货，其700W功耗使数据中心散热压力骤升。2023年6月中国移动/中国电信AI服务器集采中均明确提出液冷散热要求。工信部在2023年10月《算力基础设施高质量发展行动计划》中要求新建DC PUE<1.3。液冷概念在A股正式诞生。

**市场表现**：英维克从2023年初的12元涨至6月的25元（涨幅108%），高澜股份从8元涨至20元（涨幅150%）。申菱环境从20元涨至40元（涨幅100%）。

**上涨原因**：ChatGPT引发的AI算力预期首次传导至液冷方向——GPU功耗提升→风冷不可行→液冷刚需。市场将液冷定位为"算力卖水人"逻辑的延伸，与GPU/光模块同节奏上涨。

**资金结构**：游资主导，小市值液冷标的（高澜股份/依米康）弹性最大。公募基金配置较少，液冷尚未进入主流机构视线。

**龙头表现**：高澜股份（300499，+150%）、英维克（002837，+108%）、申菱环境（301018，+100%）。

**最终结果**：2023年下半年的调整中，液冷板块回落40-60%（数据来源：[1]）。核心原因是液冷订单落地滞后于市场预期——AI数据中心尚处于规划阶段，液冷招标集中在2024年，2023年业绩没有兑现。概念先行、业绩后到，典型的第一波投机。

### 第二期：2024年2月-7月——液冷订单落地验证

**事件**：2024年Q1英维克/高澜股份陆续公告数据中心液冷订单——英维克中标阿里云张北液冷项目（规模约2亿元），高澜股份中标字节跳动数据中心液冷项目。2024年3月上海市出台智算中心液冷部署要求（新建项目强制液冷）。英维克/高澜股份2024年Q1液冷收入同比增长200-300%（数据来源：[1]）。

**市场表现**：英维克从2024年初的18元涨至7月的35元（涨幅94%）。高澜股份从12元涨至25元（涨幅108%）。申菱环境从25元涨至40元（涨幅60%）。三花智控（液冷阀件逻辑）从20元涨至32元（涨幅60%）。

**上涨原因**：液冷订单从"概念"到"落地"的跨越。订单公告+收入确认+政策强制三条线索同时确认。市场开始给液冷2025年利润预期定价，估值切换至PEG框架。

**资金结构**：公募基金开始配置（英维克/三花智控进入多只成长基金重仓）。北向资金净买入英维克/三花智控。游资活跃度下降——小市值标的阶段结束，机构主导行情。

**龙头表现**：高澜股份（+108%）、英维克（+94%）、三花智控（+60%）。

**最终结果**：2024年8-12月液冷板块进入调整，英维克回落至25元（回撤29%），高澜股份回落至18元（回撤28%）。调整原因：2024年液冷板块整体涨幅已充分反映当年订单，但2025年业绩预期尚不清晰。

### 第三期：2025年至今——液冷成为数据中心标配

**事件**：B200（1000W+）在2025年规模出货，风冷彻底退出AI数据中心选项。液冷渗透率从2024年的20-25%（数据来源：[1]）跳升至2025年的35-40%。三大运营商算力集采中液冷方案占比超过80%。英维克/高澜股份液冷业务收入占比首次超过50%。

**市场表现**：英维克从2025年初的25元涨至40元（涨幅60%），高澜股份从18元涨至30元（涨幅67%）。三花智控液冷阀件业务放量，股价从30元涨至50元（涨幅67%）。

**上涨原因**：液冷从"可选"到"必选"的产业地位确认。渗透率加速提升（23年15%→24年25%→25年40%）的确定性增强。英维克/高澜的业绩可预测性提升——从项目制开始转向经常性收入模式。

**资金结构**：机构主导，北向/公募重仓英维克/三花智控作为AI基础设施长期配置。液冷板块的投资逻辑从"弹性博弈"变为"确定性成长"。

**龙头表现**：三花智控（+67%）、高澜股份（+67%）、英维克（+60%）。

**经验总结**：液冷板块经历了完整的产业投资三阶段——概念（2023年）→订单验证（2024年）→业绩兑现（2025年至今）。核心规律：标的选择从游资偏好的小市值弹性标的，逐步过渡到机构偏好的龙头（英维克/三花智控）。液冷是AI产业链中少有的"从0到1并且确认从1到N"的方向。

## 投资逻辑分析

### 为什么资本市场关注液冷？

液冷是AI算力基础设施中"渗透率最低、弹性最大"的环节之一：
1. **渗透率翻倍空间**：当前约35-40%（数据来源：[1]），未来3年有望提升至70-80%，对应2倍增长空间
2. **GPU功耗决定刚需**：B200/Rubin功耗持续攀升，液冷由可选变为必选
3. **AI数据中心建设高峰**：2025-2027年是中国AI数据中心大规模建设期，液冷直接受益
4. **国产替代空间**：核心零部件（微通道换热器/泵/电子膨胀阀）国产化率低

### 超额收益来源

1. **渗透率提升β**：液冷渗透率从15%（数据来源：[1]）提升至70%+的过程，是整个板块最大的贝塔。渗透率每提升10个百分点，对应市场规模增长40-50亿
2. **技术路线选择α**：冷板式vs浸没式的路线选择。当前冷板式占主导（85%+），但浸没式有颠覆性潜力（PUE更低、空间利用更优）。重仓冷板vs提前布局浸没是获取阿尔法的关键方向
3. **零部件国产替代α**：微通道换热器/电子膨胀阀/磁力泵等核心零部件国产化率不到30%（数据来源：[1]），国产替代弹性大

### 景气度传导

```
GPU功耗升级（H100 700W→B200 1000W→Rubin 1500W）
    ↓
AI服务器散热方案切换（风冷→液冷）
    ↓
数据中心新建/改造液冷需求爆发
    ↓
液冷方案商订单增长 → 收入确认
    ↓
核心零部件需求拉动（液冷板/CDU/泵阀）
```

### 未来催化剂

1. **Rubin架构发布**：NVIDIA下一代GPU（1200-1500W）发布进一步确认液冷刚需
2. **中国液冷渗透率超预期**：当前35-40%（数据来源：[1]），若加速突破50%将触发新一轮估值重估
3. **浸没式液冷突破**：若浸没式方案成本降至冷板式水平，将开启液冷第二成长曲线
4. **信创+液冷绑定**：政企/运营商算力建设中液冷成为标配

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期

## 舆论风向

### 雪球/股吧

液冷是AI算力板块的"性感方向"——渗透率低、增速快、故事清晰。看多者强调"GPU功耗无止境→液冷不可替代"的刚性逻辑，看空者强调"技术门槛不高、竞争加剧、毛利率可能持续压缩"。英维克/高澜股份的多空辩论最激烈——估值40x PE是合理还是高估，是核心分歧点。

### 知乎

技术讨论深度最高。冷板式vs浸没式的对比分析；液冷板的微通道设计优化；CDU控制算法；冷却液的介电性能与长期可靠性。有系统性的技术科普长文（如"详解冷板式液冷如何在数据中心落地"）。

### 微博/财经大V

液冷作为AI算力的"卖水人"叙事被广泛传播。"GPU越热，液冷越赚"是核心标签。新订单公告通常被快速传播，驱动短线情绪。

**关键话语特征**：液冷在投资者认知中已从"概念"升级为"业绩"——讨论焦点从"要不要买液冷"转向"买谁"（英维克vs高澜vs三花智控），表明板块认知成熟度提升。
