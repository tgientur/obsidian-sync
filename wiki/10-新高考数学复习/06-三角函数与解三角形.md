# 三角函数与解三角形

> 难度：★★★ | 分值：15~20 分 | 题型：单选 + 多选 + 解答题（第15题常客）
> 考查特点：公式多、变换灵活，小题考图像性质，大题考正余弦定理解三角形

---

## 一、核心考点

### 1. 三角函数的定义与公式

- **同角关系**：$\sin^2\alpha + \cos^2\alpha = 1$，$\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$
- **诱导公式**：口诀"奇变偶不变，符号看象限"
- **和差公式**：
  $$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$
  $$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$
  $$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$
- **倍角公式**：
  $$\sin 2\alpha = 2\sin\alpha\cos\alpha$$
  $$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$$
- **辅助角公式**：
  $$a\sin x + b\cos x = \sqrt{a^2+b^2}\,\sin(x+\varphi) \quad (\tan\varphi = \frac{b}{a})$$

### 2. 三角函数的图像与性质

$y = A\sin(\omega x + \varphi) + B$
- **振幅** $|A|$，**周期** $T = \frac{2\pi}{|\omega|}$，**频率** $f = \frac{1}{T}$
- **相位** $\omega x + \varphi$，**初相** $\varphi$
- **平移**：左加右减（$x$ 方向），上加下减（$y$ 方向）

### 3. 解三角形（正弦定理 & 余弦定理）

- **正弦定理**：$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$
  - 适用：两角一边 / 两边一对角 / 面积结合
- **余弦定理**：$a^2 = b^2 + c^2 - 2bc\cos A$
  - 适用：三边 / 两边一夹角
- **面积公式**：
  $$S = \frac12 ab\sin C = \frac12 bc\sin A = \frac12 ac\sin B$$

## 三、常用技巧

### 技巧 1：辅助角公式统一化
将 $a\sin x + b\cos x$ 统一成单一三角函数：
$$\sqrt{a^2+b^2}\,\sin(x+\varphi) \quad \text{或} \quad \sqrt{a^2+b^2}\,\cos(x-\varphi)$$
**这样做的好处**：求最值、周期、单调区间全部转化为单一三角函数的性质问题。

> 注意 $\varphi$ 所在象限由 $(a,b)$ 决定，不要只看 $\tan\varphi = \frac{b}{a}$ 就写。

### 技巧 2：解三角形的"边化角"与"角化边"
- **边化角**（更常用）：用正弦定理将边化为角的正弦，然后化简
- **角化边**：用余弦定理将角化为边的关系，通常用于已知三边关系时
- 判断标准：所给条件中边的关系复杂还是角的关系复杂

### 技巧 3：三角形中的不等关系
- $A + B + C = \pi$（角度和为 $\pi$）
- $a + b > c$（三角形成立条件）
- $\sin A > \sin B \Leftrightarrow A > B$
- 大边对大角，小边对小角
- 锐角三角形条件：$\cos A > 0,\; \cos B > 0,\; \cos C > 0$

### 技巧 4：射影定理（解三角形秒杀）
在 $\triangle ABC$ 中：
$$a = b\cos C + c\cos B$$
$$b = a\cos C + c\cos A$$
$$c = a\cos B + b\cos A$$
**应用**：看到 $b\cos C + c\cos B$ 直接写作 $a$。

### 技巧 5：1 的妙用
将 $1$ 写成 $\sin^2 x + \cos^2 x$ 或 $\frac{\sin\alpha}{\sin\alpha}$ 代入分式化简。

## 四、二级结论

1. **三角形内角平分线定理**：
   $\triangle ABC$ 中，$AD$ 平分 $\angle A$，则 $\frac{BD}{DC} = \frac{AB}{AC}$

2. **三角形的中线公式**：
   $m_a^2 = \frac{2b^2 + 2c^2 - a^2}{4}$（中线长的平方）

3. **海伦公式**（超纲，但用于快速求面积）：
   $$S = \sqrt{p(p-a)(p-b)(p-c)} \quad (p = \frac{a+b+c}{2})$$

4. **三角恒等式**（化简神器）：
   $$\sin x \pm \sin y = 2\sin\frac{x \pm y}{2}\cos\frac{x \mp y}{2}$$
   $$\cos x + \cos y = 2\cos\frac{x+y}{2}\cos\frac{x-y}{2}$$
   $$\cos x - \cos y = -2\sin\frac{x+y}{2}\sin\frac{x-y}{2}$$

5. **三角形中的射影恒等式**（比射影定理更强）：
   在 $\triangle ABC$ 中，$\sin A = \sin(B+C)$，$\cos A = -\cos(B+C)$

6. **$a\sin x + b\cos x$ 型函数的值域**：
   $$-\sqrt{a^2+b^2} \leq a\sin x + b\cos x \leq \sqrt{a^2+b^2}$$

7. **$\sin x$ 与 $\cos x$ 的对称轴**：
   $\sin x$ 对称轴 $x = \frac{\pi}{2} + k\pi$，$\cos x$ 对称轴 $x = k\pi$

## 五、易错清单

- [ ] 诱导公式"符号看象限"：把 $\alpha$ 看作锐角来确定符号
- [ ] 余弦定理中注意是 $a^2 = b^2 + c^2 - 2bc\cos A$，**$-$ 号不要忘**
- [ ] 辅助角公式中 $\varphi$ 的值：$\tan\varphi = \frac{b}{a}$，但要注意所在象限
- [ ] 三角形中 $\sin A = \sin B \not\Rightarrow A = B$（可能 $A + B = \pi$，但三角形中不可能）
- [ ] $\cos x = \frac{1}{2}$ 的解：$x = \pm\frac{\pi}{3} + 2k\pi$，不是只有 $\frac{\pi}{3}$
- [ ] 三角函数图像平移：平移 $|\varphi|$ 是对 $x$ 操作，不是对 $\omega x$ 操作


> **总评**：三角函数公式多但套路明确。小题记住辅助角公式和图像变换规则，大题掌握边化角/角化边的选择策略和面积公式应用。射影定理和和差化积公式是提速利器。
