### Token Dashboard 插件迭代

从初版到 v2 经历了多次修正：

- **上下文窗口**：初版写死 128K，用户指出模型实际为 1M。改为动态从 session 数据读取模型名 + 映射表匹配。
- **Token 数据源**：初版 bus API 返回 0，改为直接读 session JSONL 文件逐行解析 `usage` 字段。
- **上下文用量算法**：初版把缓存命中（cacheRead）也计入上下文用量，导致显示 1800%。改为只计 `inputTokens + outputTokens`。
- **UI 重设计**：环形图 + 四格指标卡 + sparkline + 逐条消息明细，深色厂商风格。
- **Route 文件语法**：初版用 `require()` 导致 route 404，改为 ESM `import` 语法。
- **缓存优化**：添加 mtime 缓存，文件未变时不重复读盘。
- **Widget iframe 认证**：从页面 URL 提取 token 参数加入 fetch，解决 iframe 内请求认证问题。
