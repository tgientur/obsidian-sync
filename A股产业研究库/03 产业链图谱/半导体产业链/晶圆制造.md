# 晶圆制造

## 定义与产业定位

晶圆制造（Wafer Fabrication）是将芯片设计公司的GDSII版图数据，通过光刻、刻蚀、薄膜沉积、离子注入、化学机械抛光（CMP）等数百道工艺步骤，在硅片上逐层构建晶体管和互连线路的过程。一颗7nm制程的芯片需要经过约1000道工序、耗时3-4个月完成制造。

晶圆制造的商业模式以Foundry（代工）为主——代工厂不设计芯片，只为客户（Fabless设计公司）提供制造服务。全球晶圆代工市场中，台积电独占约60%（数据来源：[1]）份额，三星约12%，联电约7%，中芯国际约5%。在先进制程领域（7nm以下），台积电与三星双头垄断。

**产业定位**：晶圆制造是半导体产业链中"资产最重、门槛最高、周期效应最显著"的环节。上游连接半导体设备、半导体材料、EDA工具；下游对接封装测试和芯片成品。晶圆制造环节的产能利用率直接决定半导体行业的景气度，被视为半导体产业链的"温度计"。

中国晶圆制造的特殊性在于：先进制程（7nm/5nm/3nm）受美国出口管制限制无法获取EUV光刻机，因此大陆代工厂（中芯国际/华虹）主力深耕成熟制程（28nm/40nm/55nm/90nm+）。这决定了大陆晶圆制造的投资逻辑不同于全球代工龙头——不是争取最先进制程的竞争，而是在成熟制程上通过产能扩张和国产替代获取增长。

## 产业链分析

### 价值流动与利润分配

```
半导体设备/材料 → 晶圆制造/代工 → 封装测试 → 芯片成品
    利润份额：~20%     利润份额：~30%   利润份额：~15%   利润份额：~35%
```

晶圆制造环节在营收规模上是半导体产业链最大的环节（全球代工市场约$1200亿），但净利率并非最高——台积电净利率约35-40%（数据来源：[1]），但中芯国际仅15-20%（成熟制程竞争激烈），三星代工甚至亏损。

### 定价权分析

1. **台积电**：拥有强定价权——先进制程（5nm/3nm）全球唯一稳定量产供应商（三星良率偏低），客户（苹果/英伟达/AMD/高通）没有替代选择。台积电每年代工价格上调3-5%（数据来源：[1]）
2. **中芯国际（成熟制程）**：定价权弱——28nm/40nm/55nm制程竞争者众多（联电/华虹/台积电成熟制程），客户对价格敏感。中芯国际的定价更多跟随市场供需
3. **产能利用率是核心变量**：利用率>95%（数据来源：[1]）时，代工厂有涨价底气；<80%时，降价接单是常态

### 成本传导

晶圆制造成本中，设备折旧占比最高（约40-50%）。一条5nm产线的设备投资达100-150亿美元，折旧年限5-7年。其次是硅片成本（15-20%）、化学品/气体（10-15%）、电力（8-12%）、人工（5-8%）。中芯国际的优势在于——成熟制程的设备已基本完成折旧，后续晶圆制造成本显著低于新建产线。

### 产业分工

晶圆制造的分工模式分为Foundry和IDM：
- **Foundry代工**：中芯国际（SMIC，大陆最大代工，成熟制程）、华虹半导体（特色工艺，BCD/eNVM）
- **IDM模式**：士兰微、华润微（设计+制造一体，主要覆盖功率/模拟）
- **先进制程**（7nm以下）：台积电/三星垄断，中芯国际仅能提供有限的14nm产能（受EUV管制）

### 利润来源与现金流特点

- **毛利率**：中芯国际成熟制程约30-35%（满产时），台积电约50-60%
- **净利率**：中芯国际15-20%，台积电35-40%
- **现金流**：晶圆厂是资本开支最大的半导体环节。中芯国际每年Capex 50-80亿美元（新建产线），自由现金流常年为负。经营现金流虽大（折旧高），但远不够覆盖扩产投资
- **ROE**：中芯国际约8-12%，受扩产期的低资产周转率拖累

## 行业壁垒分析

### 技术壁垒

晶圆制造是"技术密度最高的制造业"：
- **光刻**：EUV光刻系统由ASML独家供应，单台售价超$3亿。7nm/5nm制程需要多重曝光技术（LELE/LELELE），工艺复杂度指数级上升
- **刻蚀**：3D NAND/Nanowire/GAA结构的刻蚀需要极高的各向异性控制，刻蚀深宽比从50:1提升至100:1
- **良率控制**：一颗芯片1000道工序中，每一步的缺陷密度都影响最终良率。成熟制程良率>95%，先进制程良率从爬坡期的60-70%提升至稳定期的90%+需要2-3年
- **工艺库积累**：各代工厂的工艺库（PDK/Design Rule）积累需要10年以上，新进入者即使买齐设备也难以快速建立

### 资本壁垒

晶圆制造是全世界资本壁垒最高的行业之一：
- 一条12英寸成熟制程（28nm）产线投资约30-50亿美元
- 一条5nm/3nm先进制程产线投资超150亿美元
- 一座晶圆厂的建设周期2-3年，达到盈亏平衡点还需要3-5年
- 新进入者即使有500亿资金，也需要5-7年才能进入量产

### 供应链壁垒

- **光刻机**：ASML是EUV光刻机唯一供应商，对中国出口受限
- **刻蚀/沉积设备**：应用材料/Lam Research/TEL主导，国产替代率10-30%
- **硅片**：日本信越/SUMCO/Siltronic垄断12英寸硅片，沪硅产业产能快速提升
- **高纯气体/化学品**：林德/大阳日酸/昭和电工主导，华特气体/南大光电等国产替代推进中

### 客户壁垒（认证壁垒）

- 每一颗芯片在代工厂量产前需要经过**工程批验证**，周期6-12个月
- 客户（Fabless公司）的量产方案与特定代工厂的工艺库深度绑定，迁移需要重新验证
- 汽车芯片的PPAP认证周期18-24个月，认证后难切换

### 规模壁垒

晶圆厂产能利用率与利润弹性高度相关——利用率从80%（数据来源：[1]）提升至95%，利润可能增长100-200%（固定成本占比高）。中芯国际2023年约80%的利用率对应净利率约15%；若利用率提升至95%，净利率可提升至25-30%。因此，产能越大的晶圆厂在景气下行期存活能力越强。

## 历史行情复盘

### 第一期：2020年-2021年——中芯国际上市与国产替代狂热

**事件**：2020年7月中芯国际科创板上市，募资532亿（超额配售前），创A股近十年最大IPO。华为海思被迫转单中芯国际14nm。2020年9月美国BIS向中芯国际发布出口管制（限制10nm以下设备）。

**市场表现**：中芯国际（A股）上市后从27元涨至2020年8月的95元（涨幅251%），随后回落至50元。华虹半导体（港股）从15港元涨至2021年7月的55港元（涨幅267%）。

**上涨原因**：国产替代最大叙事——"中芯国际承载中国半导体制造希望"。华为海思转单带来订单确定性。科创板流动性溢价。

**资金结构**：公募基金标配+科创板ETF被动配置。游资炒作上市初期的短期博弈。

**龙头表现**：中芯国际（688981，+251%）、华虹半导体（01347.HK，+267%）。

**最终结果**：中芯国际从95元跌至2022年10月的30元（回撤68%）。逻辑被证伪——14nm产能受限（美国设备管制），先进制程扩张受阻。

### 第二期：2022年-2024年——成熟制程扩张，国产化加速

**事件**：中芯国际策略调整——从"追赶先进制程"转向"做大成熟制程"。中芯京城/中芯深圳/中芯临港等新建12英寸产线（均聚焦28nm/40nm/55nm）。美国管制持续加码，但成熟制程设备国产化率提升。

**市场表现**：中芯国际在2022年10月的30元触底后逐步反弹至2024年的50元（涨幅67%）。华虹半导体科创板上市后震荡。

**上涨原因**：成熟制程产能扩张符合产业逻辑+国产化率提升。28nm/40nm产能供不应求（车规/工业/IoT需求增长）。

**资金结构**：半导体ETF持续净申购。北向资金从2023年开始重新流入中芯国际。

**龙头表现**：中芯国际（+67%）。

**最终结果**：中芯国际在40-55元区间震荡，未形成趋势。原因：尽管收入增长，但利润受扩产带来的高折旧拖累，市场等待产能利用率提升带来的利润拐点。

### 第三期：2025年至今——产能利用率修复与利润拐点

**事件**：2024年下半年半导体周期进入上行期，晶圆代工产能利用率从80%（数据来源：[1]）提升至90%+。中芯国际成熟制程满产，单季度净利润创历史新高。AI芯片需求拉动基板及配套芯片代工需求。

**市场表现**：中芯国际从2025年初的45元涨至70元（涨幅56%）。华虹半导体从20元涨至35元（涨幅75%）。

**上涨原因**：产能利用率提升→利润弹性释放。中芯国际2025年净利率从15%（数据来源：[1]）提升至25%。成熟制程代工的周期性红利确认。

**资金结构**：机构开始从中周期视角配置代工龙头，中芯国际成为半导体周期投资核心标的。

**龙头表现**：华虹半导体（+75%）、中芯国际（+56%）。

**经验总结**：晶圆制造的投资核心是"产能利用率周期"。中芯国际上市以来的股价走势完全围绕产能利用率和美国管制展开——管制加压时下跌，产能利用率提升时上涨。

## 投资逻辑分析

### 为什么资本市场关注晶圆制造？

晶圆制造是半导体产业链的"风向标"：
1. **产能利用率是周期最敏感的先行指标**——利用率变化1个百分点，利润变化3-5个百分点
2. **成熟制程国产替代空间**：中国是全球最大半导体消费市场，但本土代工产能满足率不到30%（数据来源：[1]）
3. **管制倒逼自主产线建设**：美国管制下，中国必须建设自主晶圆产能，大基金三期重点支持制造

### 超额收益来源

1. **周期弹性β**：晶圆制造是半导体周期中最具弹性的环节——产能利用率提升时利润弹性最大
2. **成熟制程国产替代α**：28nm/40nm/55nm制程国产化率提升，是中国晶圆制造最确定的增长来源
3. **产能扩张α**：新建产能贡献收入增量——每新建一条3万片/月的12英寸产线，年收入增加30-50亿

### 景气度传导

```
全球半导体销售额↑

```

### 未来催化剂

1. **半导体周期上行**：全球半导体销售额创历史新高
2. **成熟制程产能利用率超95%（数据来源：[1]）**：中芯国际利润弹性释放
3. **国产设备突破**：去A化产线验证成功，产能扩张加速
4. **大基金三期投资落地**：新建晶圆厂资本注入

## 主要风险

1. **周期下行风险**：全球半导体周期可能进入下行阶段
2. **技术追赶风险**：国产替代进度低于预期
3. **设备管制风险**：荷兰/日本对华半导体设备出口管制加码
4. **产能过剩风险**：成熟制程产能扩张可能导致价格战

## 舆论风向

### 雪球/股吧

中芯国际是A股半导体板块"话题之王"。核心辩论："成熟制程能否支撑长期增长"vs"先进制程被卡死没有未来"。华虹半导体的"特色工艺"定位获得部分机构投资者认可。中芯国际股价与美国管制消息高度相关——每轮制裁升级都会引发恐慌性抛售和随后的"利空出尽"反弹。

### 知乎

关于"中芯国际能否突破7nm"的技术讨论长期热度高——EUV光刻机被禁后的多重曝光替代方案（SAQP/LELE）的技术可行性和成本可行性。晶圆厂的选址和当地电力/水供应也是热门话题。

### 微博/财经大V

中芯国际是中概科技股的代表性叙事。美国制裁相关消息是最大催化剂。

**关键话语特征**：晶圆制造的讨论带有强烈的"国家意志"色彩——中芯国际被视为"中国芯"的代名词，投资决策中的非理性情绪高于其他半导体环节。
