## 是什么

随机前沿分析（Stochastic Frontier Analysis, SFA）是一种参数化的效率测度方法。核心思想：实际产出 = 前沿产出（最大可能产出） − 无效率损失。

## 数学模型

基本模型：

```
Q_it = X'_it β + ε_it
ε_it = ν_it − μ_it
```

其中：
- ν_it ~ N(0, σ²_ν)：随机误差（运气、环境因素）
- μ_it ~ N⁺(ϖ_it, σ²_it)：无效率项（非负，单边分布）

## 异质性扩展

传统 SFA 假设所有样本的无效率分布相同。异质性 SFA 允许 μ_it 的均值和方差随银行特征变化：

```
ϖ_it = exp(b₀ + Z'_it δ)
σ²_it = exp(b₁ + ω'_it γ)
```

这样不同资产规模、不同区域的银行可以有不同水平的效率分布。

## 效率值计算

```
TE_it = exp(−μ_it)
```

介于 0 和 1 之间，越接近 1 效率越高。

## 为什么用 SFA 而不是 DEA

| SFA | DEA |
|-----|-----|
| 参数方法，需设定函数形式 | 非参数，无需设定 |
| 区分随机误差和无效率 | 将所有偏离视为无效率 |
| 可做统计检验 | 无统计推断基础 |
| 适合面板数据 | 适合小样本 |
