# 政策专题

本页面用于梳理重要政策专题，按政策脉络和行业影响展开分析。

## 政策分析框架

### 供需分析四层框架（来源：知乎@不熬夜睡觉冠军）

1. **实体经济供需** — 影响商品/服务的实际供给与需求（如新能源补贴退坡）
2. **金融资产供需** — 影响资本市场流动性与资产配置（如降准降息）
3. **地缘冲突** — 重塑全球供应链与贸易格局（如出口管制）
4. **政治周期** — 决定政策力度与节奏（如换届年、五年规划）

### 政策新闻解读法

- 看会场：什么级别、什么机构、什么场合发布
- 分钱权袋：政策给了多少钱、谁有权执行、执行工具是什么
- 时间差：政策发布 → 细则落地 → 招标执行 → 业绩兑现
- 串连续剧：同一主题的系列政策组合拳

## 政策传导机制

```
政策文件发布
  → 部委细则/地方试点
    → 补贴/税收/准入/标准
      → 企业行为调整
        → 产业链供需变化
          → 上市公司业绩
            → 二级市场定价
```
