# 中国A股全产业链投研知识库

> 版本：v1.0 初始构建
> 构建时间：2026-06-20
> 定位：达到券商行业深度报告水平的产业链知识图谱

## 体系架构

本知识库按照券商卖方研究所的研究框架组织，从宏观策略到微观公司，形成完整的投研分析链条：宏观策略 → 行业研究 → 产业链图谱 → 技术路线图 → 商业模式分析 → 主题投资 → 上市公司研究 → 投资框架。
