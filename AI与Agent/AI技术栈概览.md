### 分层说明

| 层级 | 说明 |
|------|------|
| **能源层** | 算力的物理基础——电力和散热 |
| **芯片层** | GPU / TPU / NPU 等 AI 加速硬件 |
| **基础设施层** | 云服务、分布式计算、网络 |
| **大型语言模型** | 预训练基础模型（LLM）|
| **上下文窗口** | 模型能一次处理的输入长度 |
| **长期存储** | 超出上下文窗口的记忆/知识持久化 |
| **工具集成** | 模型调用外部 API、数据库、代码执行等能力 |
| **智能体框架** | Agent 编排、多步推理、自主决策 |
| **AI Native 部门** | 组织内的 AI-first 团队 |
| **AI Native 公司** | 以 AI 为核心竞争力的公司 |
| **AI Native 经济生态** | 宏观层面 AI 重塑整个经济系统 |
