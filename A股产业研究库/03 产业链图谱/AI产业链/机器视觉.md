# 机器视觉

## 定义与产业定位

机器视觉（Machine Vision）是指利用摄像头、图像传感器和图像处理算法，使机器获得"看"和"理解"的能力——从图像/视频中提取信息、做出判断、指导行动。作为AI技术落地的核心感知层，机器视觉是连接数字世界与物理世界的"眼睛"。

核心任务是让机器完成三件事：**看得见**（图像采集）、**看得准**（图像识别）、**看得懂**（语义理解）。从工业质检到自动驾驶、从医疗影像到安防监控，机器视觉正在渗透各行各业。

AI大模型时代的到来（尤其是多模态大模型CLIP/SAM/GPT-4V）大幅提升了机器视觉的通用性——从过去"一个场景一个模型"的碎片化模式，走向"一个基础模型适配多个场景"的通用视觉智能。

## 产业链位置

```
上游（核心零部件） → 中游（视觉系统/解决方案） → 下游（应用场景）
    光源/镜头             相机/智能相机              工业质检
    CMOS传感器             图像采集卡               自动驾驶
    AI芯片（推理）        视觉软件/算法             医疗影像
    镜头模组              视觉控制器               安防监控
                                                       消费电子
```

机器视觉承接上游的CMOS传感器（韦尔股份）、镜头（舜宇光学）、AI推理芯片（寒武纪/海光），通过中游的视觉算法和系统集成，服务下游众多行业。

### 关键趋势：视觉大模型（VLM）

2024-2026年，多模态大模型（Visual Language Model）正在重构机器视觉的技术范式：
- **零样本/少样本检测**：不再需要大量标注数据，VLM可直接理解"找什么"
- **通用感知**：一个模型覆盖检测/分割/识别/OCR等多种任务
- **交互式视觉**：自然语言描述即可驱动视觉任务，降低部署门槛

代表模型：GPT-4V/o、Claude视觉能力、华为盘古CV、智谱CogVLM。

## 投资逻辑简析

1. **工业自动化"眼睛"**：中国制造业升级（智能制造2025）驱动机器视觉在检测/测量/引导环节的渗透率从当前15-20%（数据来源：[1]）提升至30%+。每台自动化设备都需要视觉系统。

2. **AI降低部署门槛**：传统机器视觉需要大量标注数据和定制开发，AI大模型实现零样本/少样本部署，降低使用成本30-50%（数据来源：[1]），加速下游中小企业应用。

3. **替代人力检验**：中国电子/半导体/新能源产业对品控精度要求持续提升，人工目检被替代的确定性高。AOI检测设备在PCB/半导体前道/锂电隔膜等领域的渗透率快速提升。

4. **国产替代空间**：基恩士/康耐视占中国机器视觉市场>50%（数据来源：[1]）份额，国产替代空间约200-300亿。奥普特/天准在光源/镜头环节已突破，视觉软件和智能相机是下一步攻坚方向。

**风险提示**：机器视觉下游分散，单一客户占比低，业绩爆发性不如AI算力链。高估值需谨慎。

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期
