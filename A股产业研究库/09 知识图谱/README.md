## 全产业链通联图谱

三条隐形的连接线贯穿所有产业链——资金、技术、政策。每一轮大行情，往往都是三条线中的某一条共振。

### 资金流：资本开支的轮动轨迹

资本不会同时流向所有产业链，而是有迹可循地轮动。理解资金的流向变化，比研究单个产业链的景气度更重要。

**AI（2023-2025）→ 半导体（2024-2026）**
- AI军备竞赛拉动GPU/HBM/HBM封装需求
- 半导体资本开支暴增，设备/材料受益
- 逻辑：AI是需求端爆发，半导体是供给端响应
- 关联节点：[[A股产业研究库/03 产业链图谱/AI产业链/AI服务器|AI服务器]] → [[A股产业研究库/03 产业链图谱/半导体产业链/HBM|HBM]] → [[A股产业研究库/03 产业链图谱/半导体产业链/半导体设备|半导体设备]]

**半导体（2024-2026）→ 新能源（未来）**
- 半导体技术成熟后，制造成本下降
- 智能化电动化渗透加速
- 逻辑：半导体是底层赋能者，新能源是最大应用场景
- 关联节点：[[A股产业研究库/03 产业链图谱/半导体产业链/SiC|SiC]] → [[A股产业研究库/03 产业链图谱/新能源产业链/电驱电控|电驱电控]] → [[A股产业研究库/03 产业链图谱/新能源产业链/总纲|新能源车]]

**新能源（2020-2023）→ 高端制造（2023-2026）**
- 新能源产业链的规模效应溢出到高端装备
- 锂电设备→光伏设备→机器人部件，技术同源性在增强
- 逻辑：制造业升级是长期主线，新能源是先行者
- 关联节点：[[A股产业研究库/03 产业链图谱/新能源产业链/动力电池|动力电池]] → 精密制造 → [[A股产业研究库/03 产业链图谱/机器人产业链/总纲|机器人]]

**资本开支轮动框架**
1. 当AI产业链的资本开支增速见顶时，半导体设备的需求会滞后1-2个季度受益
2. 当半导体设备订单增速见顶时，上游材料会滞后1-2个季度受益
3. 当上游产能释放、成本下降时，下游应用（智能终端、机器人、智驾）成本门槛降低
4. 每一轮技术升级的末期，资金会从上游制造回流到下游应用和内容

### 技术流：底层技术的共享与迁移

不同产业链之间的技术共享不是巧合，而是物理定律和数学能力的自然延伸。

**GPU → 智驾 → 机器人**
- 核心计算单元都是GPU/ASIC，高性能计算是三者共同的基础
- 自动驾驶感知算法（BEV+Transformer）直接迁移到机器人视觉
- 仿真环境（Isaac Sim/CARLA）也高度共用
- 关联：AI芯片→[[A股产业研究库/03 产业链图谱/高端制造产业链/智能驾驶|智能驾驶]]→人形机器人控制器

**SiC → 新能源车 → 光伏**
- SiC器件在高压高频场景效率远超Si基器件
- 新能源车主驱逆变器是SiC最大应用场景
- 光伏逆变器紧随其后，储能变流器是增量市场
- 三代半导体技术的成本下降曲线同时推动多个产业的效率提升
- 关联：[[A股产业研究库/03 产业链图谱/半导体产业链/SiC|SiC衬底/外延]]→OBC/电驱→[[A股产业研究库/03 产业链图谱/新能源产业链/光伏逆变器|光伏逆变器]]

**云计算 → 金融科技 → 数据要素**
- 云基础设施降低金融IT的部署成本
- 金融IT积累的海量数据成为数据要素交易的天然基础
- 数据要素确权和定价技术反过来赋能金融风控
- 关联：[[A股产业研究库/03 产业链图谱/AI产业链/云计算|云计算]] → 银行IT/证券IT → 数据交易所

**传感器 → 消费电子 → 机器人 → 智驾**
- MEMS传感器产线高度兼容，消费电子走量降本后向车规/工规迁移
- 激光雷达、毫米波雷达、摄像头三者的供应链高度重合
- 关联：消费电子传感器 → [[A股产业研究库/03 产业链图谱/高端制造产业链/智能驾驶|智驾传感器]] → 机器人触觉传感器

### 政策流：顶层设计的传导链

中国产业政策的传导路径清晰可预测，理解它就找到了投资的时间轴。

**数据要素 → AI → 信创**
- 数据要素是生产资料：确权、交易、定价政策先行
- AI是生产力：大模型需要高质量数据训练
- 信创是生产关系：数据安全/AI安全/自主可控的底层要求
- 政策传导路标：数据二十条→数据资产入表→AI安全监管→信创加速落地
- 关联节点：[[A股产业研究库/05 政策跟踪/数据要素|数据要素]] → [[A股产业研究库/03 产业链图谱/AI产业链/总纲|AI]] → [[A股产业研究库/03 产业链图谱/信创产业链/总纲|信创]]

**碳中和 → 新能源 → 储能 → 电网**
- 双碳目标是源头：顶层约束碳排放总量
- 新能源发电是手段：风光装机量目标层层分解
- 储能是瓶颈：新能源波动性倒逼储能政策出台
- 电网改造是终点：分布式/特高压/数字化是最后一块拼图
- 政策传导路标：双碳政策→新能源补贴退坡→储能独立市场主体→电力市场化改革
- 关联节点：[[A股产业研究库/05 政策跟踪/碳中和|碳中和]] → [[A股产业研究库/03 产业链图谱/新能源产业链/总纲|新能源发电]] → [[A股产业研究库/03 产业链图谱/新能源产业链/储能|储能]] → [[A股产业研究库/03 产业链图谱/新能源产业链/电网设备|电网设备]]

**制造强国 → 高端制造 → 工业母机 → 工业软件**
- 制造强国2035是顶层战略
- 高端装备是核心抓手
- 工业母机（数控机床/精密加工）是最卡脖子的环节
- 工业软件是制造业的大脑
- 政策传导路标：制造强国战略→专项基金→国产生产线验收→工业软件国产化率目标
- 关联节点：[[A股产业研究库/03 产业链图谱/高端制造产业链/总纲|高端装备]] → [[A股产业研究库/03 产业链图谱/高端制造产业链/工业母机|工业母机]] → [[A股产业研究库/03 产业链图谱/高端制造产业链/工业软件|工业软件]]

## 产业链投资节奏日历

投资节奏的核心矛盾是：当信息来临时，你有没有完整的产业链视角判断它的影响。整理出全年关键节点，提前布局，而不是等新闻出来再追。

### 各产业链的业绩披露节奏

| 时间窗口 | 披露内容 | 关注的产业链 |
|:---------|:---------|:------------|
| 1月底前 | 上年年报预告（有条件强制披露） | 所有产业链，重点关注超预期方向 |
| 4月底前 | 年报+一季报 | 全年最重要的验证窗口，决定全年配置方向 |
| 4月-5月 | 一季报分析季 | 行业景气度拐点的最佳确认窗口 |
| 7月-8月 | 中报/半年报 | 验证上半年逻辑，修正下半年预期 |
| 8月-9月 | 中报分析季 | 产业链内部格局分化确认 |
| 10月底前 | 三季报 | 确认全年趋势，布局次年方向 |

### 政策事件日历

| 时间 | 事件 | 影响产业链 | 提前布局窗口 |
|:----|:----|:----------|:------------|
| 3月初 | 两会（政府工作报告+预算） | 所有政策驱动产业 | 1月底-2月（预期发酵期） |
| 4月中 | 政治局会议（经济工作） | 宏观预期→顺周期/地产/消费 | 3月-4月初 |
| 7月底 | 政治局会议（年中经济） | 下半年政策定调 | 6月-7月 |
| 10月-11月 | 三中全会/中央金融工作会议/其他重大会议 | 长期改革方向 | 8月-10月 |
| 12月中 | 中央经济工作会议 | 次年经济工作重点+产业方向 | 11月初-12月初 |
| 1月-2月 | 各部委年度工作会议 | 细分产业方向具体目标 | 12月下旬-1月 |
| 全年 | 行业展会/产品发布会 | 电子/CES（1月）/MWC（2月）/AWE（3月）/北京车展（4月）/COMPUTEX（5月）/世界AI大会（7月）/IAA（9月） | 展前2-4周 |

**关键原则**
- 预期在事件发生前2-4周开始发酵，事件当天往往是阶段高点
- 超预期的政策会在事后1-2周内继续驱动行情
- 低于预期的政策要快速离场，不要等待验证

### 产业链景气度确认时间窗口

**一季度（1月-3月）**
- 年报预告检验上年逻辑：业绩兑现率低于60%的产业链需要重新评估
- 春节后开工数据（复工率、物流、用电量）是判断制造业景气度的最早信号
- 两会政策方向确认政策驱动的产业链

**二季度（4月-6月）**
- 年报+一季报是最重要的景气度确认窗口
- 各类展会（北京车展、COMPUTEX等）的新品/订单信号
- 618消费数据验证消费电子的景气度
- 光伏、储能在年中并网冲刺窗口的装机量

**三季度（7月-9月）**
- 中报验证上半年逻辑，产业链分化最明显的窗口
- 下半年排产数据更新，旺季备货节奏开始
- 暑期出行/消费数据验证消费复苏强度
- 消费电子进入新品发布密集期

**四季度（10月-12月）**
- 三季报确认全年业绩趋势，提前布局次年
- 中央经济工作会议确定次年产业方向，是最重要的次年布局窗口
- 光伏/新能源车年末冲量数据
- 机构调仓换股的高峰期，资金流动向需关注

## 知识图谱使用指南

### 如何从一条链穿越到另一条链

知识图谱的价值不在于记住每条链的节点，而在于发现链与链之间的连接点，找到穿越的路径。

**三步穿越法**

1. **找共同技术**：从当前产业链中找出与目标产业链共享的底层技术
   - 例：从[[A股产业研究库/03 产业链图谱/AI产业链/总纲|AI]]到[[A股产业研究库/03 产业链图谱/机器人产业链/总纲|机器人]]，看GPU→控制器→执行器
   - 例：从[[A股产业研究库/03 产业链图谱/新能源产业链/总纲|新能源]]到[[A股产业研究库/03 产业链图谱/半导体产业链/总纲|半导体]]，看SiC→功率器件→衬底

2. **找资金流向**：从资本开支的边际变化判断资金下一步流向
   - 例：AI资本开支做多的阶段，优先看上游设备/材料
   - 例：上游产能过剩后，资金会向下游应用迁移

3. **找政策传导**：从政策文件的递进关系判断哪些产业链会依次受益
   - 例：算力基建→AI大模型→AI应用→AI终端硬件
   - 例：双碳→风光→储能→电网

**常见穿越路径**
- AI算力 → 半导体设备 → 半导体材料 → 先进封装
- 新能源车渗透率见顶 → 充电桩/换电站 → 电池回收 → 储能
- 消费电子企稳 → 上游芯片/面板库存回补 → 半导体周期上行
- 机器人量产规模效应 → 精密减速器降本 → 智驾传感器受益

### 如何从总纲找到核心公司再到具体数据

知识图谱的结构是分层的，正确的使用路径是从宏观到微观，从概览到细节。

**第一步：读总纲**
- 查看产业链的总纲页面，理解产业的逻辑链和核心驱动因子
- 确认当前产业链处于生命周期的哪个阶段
- 找到关键的上下游关系和瓶颈环节

**第二步：看核心公司**
- 从总纲的关联公司/子节点跳转到对应的具体标的
- 查看各环节的竞争格局和龙头公司
- 关注龙头的市占率变化和毛利率趋势

**第三步：查具体数据**
- 从核心公司页面链接到具体的数据跟踪页面
- 价格数据：产品单价、原材料成本、价差走势
- 供需数据：产能、产量、出货量、库存（参见[[A股产业研究库/04 数据跟踪/README|数据跟踪目录]]）
- 估值数据：历史PE/PB分位、PEG、PS

**第四步：验证逻辑**
- 看政策库里对应的政策文件是否落地
- 看技术突破节点的验证进度
- 看产业链调研纪要是否印证判断

### 知识库更新频率建议

| 更新频率 | 内容 | 操作人 |
|:--------|:----|:------|
| 每日 | 产业链重要新闻、产品发布、企业公告 | L本人（晨间简单扫描） |
| 每周 | 上游价格数据、高频景气指标更新 | L本人（周日更新） |
| 每月 | 产业链中观数据（出货量/装机量/招标量） | L本人（月初集中更新） |
| 每季 | 业绩更新、产业链逻辑重新梳理 | L本人（财报季重点更新） |
| 每半年 | 产业链总纲重构、框架迭代 | 需要系统性复盘 |
| 事件驱动 | 政策文件发布/管制升级/重大技术突破 | 消息出来后24h内录入 |

**更新原则**
- 核心逻辑链不要频繁改，半年审视一次足矣
- 高频数据存放在[[A股产业研究库/04 数据跟踪/README|数据跟踪]]目录，不要在产业链中重复记录，用链接关联
- 所有更新都带时间戳，方便回溯判断当时观点是否正确
- 如发现自己判断错误，不要删改旧数据，用新笔记打补丁，保留思考痕迹
