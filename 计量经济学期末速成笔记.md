## Ch1 计量经济学的性质与经济数据

**计量经济学定义**：以经济理论为指导，以经济数据为依据，以数理统计为手段，通过建立、估计、检验经济模型，揭示经济变量间的随机因果关系。

**数据四种类型**：

| 类型 | 特征 | 典型问题 |
|------|------|----------|
| 横截面数据 | 同一时点多个个体，独立 | 工资-教育 |
| 时间序列数据 | 同一对象随时间变化，序列相关 | GDP、股价 |
| 混合横截面数据 | 多个独立横截面的合并 | 政策评估前后对比 |
| 面板数据 | 同组个体跟踪多期，有横截面+时间双重维度 | 城市犯罪两年面板 |

**因果关系（ceteris paribus）**：X对Y的因果效应 = "保持其他所有相关因素不变，改变X时Y的变化量"。核心挑战：不可观测因素的干扰（遗漏变量、反向因果）。

**计量分析步骤**：理论模型设定 → 计量经济模型 → 参数估计 → 模型检验 → 模型应用

## Ch2 简单回归模型

### 模型形式
`y = β₀ + β₁x + u`

- y：因变量（被解释变量）
- x：自变量（解释变量）
- u：随机误差项（扰动项）

**误差项u的来源**：①人类经济行为的随机性 ②遗漏变量 ③模型设定误差 ④测量误差 ⑤其他随机因素

### 总体回归 vs 样本回归

- **总体回归函数（PRF）**：E(y|x) = β₀ + β₁x
- **样本回归函数（SRF）**：ŷ = β̂₀ + β̂₁x

**"线性"的含义**：参数线性（β以一次方出现），变量可以非线性（如x², lnx）

### OLS估计量推导

最小化残差平方和：min Σ(yᵢ - β̂₀ - β̂₁xᵢ)²

一阶条件：
- Σ(yᵢ - β̂₀ - β̂₁xᵢ) = 0
- Σxᵢ(yᵢ - β̂₀ - β̂₁xᵢ) = 0

解得：
```
β̂₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²
β̂₀ = ȳ - β̂₁x̄
```

### 关键假设（SLR.1-SLR.5）

| 假设 | 内容 | 意义 |
|------|------|------|
| SLR.1 参数线性 | y = β₀ + β₁x + u | 模型在参数上线性 |
| SLR.2 随机抽样 | {(xᵢ,yᵢ): i=1,...,n} 来自总体的随机样本 | 每个观测独立同分布 |
| SLR.3 样本变异 | Σ(xᵢ-x̄)² > 0 | x必须有变化 |
| SLR.4 零条件均值 | E(u|x) = 0 | x外生，与u不相关 |
| SLR.5 同方差 | Var(u|x) = σ² | 误差方差不依赖x |

### OLS估计量的性质

**无偏性**（SLR.1-SLR.4）：
E(β̂₁) = β₁ , E(β̂₀) = β₀

**方差**（SLR.1-SLR.5）：
```
Var(β̂₁) = σ² / Σ(xᵢ-x̄)²
Var(β̂₀) = σ²·(Σxᵢ²/n) / Σ(xᵢ-x̄)²
```

**误差方差估计**：
```
σ̂² = Σûᵢ² / (n-2)   (df = n-2)
```

**标准误**：se(β̂₁) = σ̂ / √[Σ(xᵢ-x̄)²]

### 高斯-马尔可夫定理
SLR.1-SLR.5下，OLS估计量是最佳线性无偏估计量（BLUE）——在所有线性无偏估计量中方差最小。

### OLS代数性质
① Σûᵢ = 0  ② Σxᵢûᵢ = 0  ③ 点(x̄,ȳ)落在回归线上

### R²（拟合优度）
```
SST = Σ(yᵢ-ȳ)²   总平方和
SSE = Σ(ŷᵢ-ȳ)²   解释平方和（回归平方和）
SSR = Σûᵢ²       残差平方和
SST = SSE + SSR
R² = SSE/SST = 1 - SSR/SST
```
R² = yᵢ与ŷᵢ之间相关系数的平方。加入新变量R²只增不减。

## Ch3 多元回归分析：估计

### 模型形式
`y = β₀ + β₁x₁ + β₂x₂ + ... + βₖxₖ + u`

**偏效应解释**：βⱼ = 保持其他解释变量不变，xⱼ增加一个单位时y的平均变化量。

### OLS估计

最小化 Σ(yᵢ - β̂₀ - β̂₁xᵢ₁ - ... - β̂ₖxᵢₖ)²

**分块回归解释**（Frisch-Waugh定理）：
1. 把xⱼ对其他所有x做回归，得到残差r̂ⱼ
2. y对r̂ⱼ做回归，β̂ⱼ = Σr̂ⱼᵢyᵢ / Σr̂ⱼᵢ²
→ β̂ⱼ衡量的是xⱼ中与其他x不相关的部分对y的影响

### 拟合优度（多元）
```
SST = SSE + SSR  （同上）
R² = SSE/SST = 1 - SSR/SST
```

### 六个假设（MLR.1-MLR.6）

| 假设 | 内容 |
|------|------|
| MLR.1 参数线性 | 总体模型参数线性 |
| MLR.2 随机抽样 | {(xᵢ₁,...,xᵢₖ,yᵢ)} 随机样本 |
| MLR.3 无完全共线性 | 无解释变量为常数，无精确线性关系 |
| MLR.4 零条件均值 | E(u|x₁,...,xₖ) = 0 |
| MLR.5 同方差 | Var(u|x₁,...,xₖ) = σ² |
| MLR.6 正态性 | u ~ N(0,σ²)，独立于x（用于推断，大样本可放宽） |

**MLR.1-MLR.5 → 高斯-马尔可夫假设**（OLS为BLUE）
**MLR.1-MLR.6 → 经典线性模型（CLM）假设**（t/F检验有效）

### 定理3.1 OLS无偏性（MLR.1-MLR.4）
E(β̂ⱼ) = βⱼ

### 定理3.2 OLS方差

```
Var(β̂ⱼ) = σ² / [SSTⱼ·(1-Rⱼ²)]
```
其中：
- SSTⱼ = Σ(xᵢⱼ - x̄ⱼ)²
- Rⱼ² = xⱼ对其他所有x（含常数）回归的R²
- 1/(1-Rⱼ²) = **方差膨胀因子（VIF）**

**VIF经验法则**：VIF > 10 表明存在严重多重共线性

**影响方差的三个因素**：
1. σ²↑ → Var↑ （噪声大）
2. SSTⱼ↑ → Var↓ （x变异大更好）
3. Rⱼ²↑ → Var↑ （多重共线性）

### 定理3.3 误差方差的无偏估计
```
σ̂² = SSR / (n-k-1)     df = n-k-1
```
se(β̂ⱼ) = σ̂ / √[SSTⱼ·(1-Rⱼ²)]

### 定理3.4 高斯-马尔可夫定理
MLR.1-MLR.5下，OLS是**BLUE**（最佳线性无偏估计量）。

### 遗漏变量偏误

真实模型：y = β₀ + β₁x₁ + β₂x₂ + u
估计模型：y = β₀ + β₁x₁ + v  （遗漏x₂）

```
β̃₁ = β̂₁ + β̂₂·δ̃₁
```
偏误方向 = β₂与δ̃₁（x₂对x₁回归的斜率）的符号乘积。

- 若β₂=0（x₂无关）或δ̃₁=0（x₁与x₂不相关）→ 无偏误
- 偏误不随样本增大而消失

### 包含无关变量
在总体中βⱼ=0的变量仍包含在模型中 → OLS仍无偏，但方差增大（降低估计精度）。

## Ch4 多元回归分析：推断

### t检验

**t统计量**：
```
t_βⱼ = (β̂ⱼ - aⱼ) / se(β̂ⱼ)
```
检验 H₀: βⱼ = aⱼ  （常用 aⱼ=0）

**H₀: βⱼ=0 时**，t = β̂ⱼ / se(β̂ⱼ) ~ t(n-k-1)

### 单边检验 vs 双边检验

| 检验类型 | H₀ | H₁ | 拒绝条件 |
|----------|----|----|----------|
| 单边（上尾） | βⱼ=0 | βⱼ>0 | t > t_α |
| 单边（下尾） | βⱼ=0 | βⱼ<0 | t < -t_α |
| 双边 | βⱼ=0 | βⱼ≠0 | \|t\| > t_α/2 |

**经验法则**（大样本正态近似）：
- |t| > 1.645 → 10%水平显著
- |t| > 1.96  → 5%水平显著
- |t| > 2.576 → 1%水平显著

### p值
p值 = 在H₀下观察到当前及更极端t值的概率。p < α 则拒绝H₀。

### 置信区间
```
β̂ⱼ ± t_α/2 · se(β̂ⱼ)
```
- 95% CI：β̂ⱼ ± 1.96·se(β̂ⱼ)（大样本）
- 若0不在CI内 → 在α水平上显著

### 参数线性组合的检验

例：H₀: β₁ = β₂  → 设 θ = β₁ - β₂，t = (β̂₁ - β̂₂) / se(β̂₁ - β̂₂)

**替代方法**：重新参数化，令 γ₁ = β₁ + β₂，则原模型变为
y = β₀ + γ₁x₁ + β₂(x₂ - x₁) + ...  ，
直接检验 γ₁的系数

### F检验（多重线性限制）

H₀: q个线性限制同时成立（如 β₃=0, β₄=0, β₅=0）

```
F = [(SSRᵣ - SSRᵤᵣ)/q] / [SSRᵤᵣ/(n-k-1)]
  = [(R²ᵤᵣ - R²ᵣ)/q] / [(1-R²ᵤᵣ)/(n-k-1)]
```
其中下标ur=无约束模型，r=受约束模型。

- F ~ F(q, n-k-1)
- 拒绝条件：F > F_临界值

**整体显著性F检验**（所有斜率系数=0）：
```
F = (R²/k) / [(1-R²)/(n-k-1)]
```
H₀: β₁ = β₂ = ... = βₖ = 0

### 统计显著 vs 经济显著
- 统计显著 ≠ 经济显著（大样本下微小效应也可能统计显著）
- 需同时关注系数大小（经济含义）
- 若系数符号与经济理论相反，可能存在模型设定偏误

### 大数定理与中心极限定理
- **大数定理**：n→∞时，样本均值 → 总体期望
- **中心极限定理**：n→∞时，样本均值 → 正态分布（不论总体分布）
→ 大样本下不必要求MLR.6（误差正态性），t/F检验渐近有效

## Ch6 进一步的问题

### 函数形式

四种常见模型及其解释：

| 模型形式 | 方程 | β₁的解释 |
|----------|------|----------|
| 水平-水平 | y = β₀ + β₁x + u | Δy = β₁Δx （绝对变化） |
| 对数-水平 | ln(y) = β₀ + β₁x + u | %Δy ≈ 100β₁·Δx |
| 水平-对数 | y = β₀ + β₁ln(x) + u | Δy = (β₁/100)%Δx |
| 对数-对数 | ln(y) = β₀ + β₁ln(x) + u | %Δy = β₁%Δx （弹性） |

**对数使用的原则**：
- 取对数可缓解异常值，帮助正态性和同方差
- 取值可为正数的变量取对数（工资、价格、销售额）
- 以年份计量的变量不取对数
- 以百分点计量的变量可选
- 变量有零或负值时不可取对数

### 二次函数
```
y = β₀ + β₁x + β₂x² + u
```
x的**边际效应**：Δy/Δx = β₁ + 2β₂x

**转折点**（极值点）：x* = -β₁/(2β₂)

- β₂ < 0 → 凹函数（先增后减），存在最大值
- β₂ > 0 → 凸函数（先减后增），存在最小值

### 交互项
```
y = β₀ + β₁x₁ + β₂x₂ + β₃x₁x₂ + u
```
- x₁的边际效应（依赖于x₂）：Δy/Δx₁ = β₁ + β₃x₂
- x₂的边际效应（依赖于x₁）：Δy/Δx₂ = β₂ + β₃x₁

**重新参数化**（使交互项系数在均值处有意义）：
令 z₁ = x₁ - x̄₁, z₂ = x₂ - x̄₂，回归 y = β₀ + γ₁z₁ + γ₂z₂ + β₃z₁z₂ + u
- γ₁ = 在均值处x₁的偏效应
- γ₂ = 在均值处x₂的偏效应

**平均偏效应（APE）**：对每个观测计算偏效应后求平均

### 调整R²
```
R̄² = 1 - [SSR/(n-k-1)] / [SST/(n-1)]
    = 1 - (1-R²)(n-1)/(n-k-1)
```
- 加入新变量时可能上升也可能下降
- 仅当新变量t统计量绝对值>1时R̄²才增加
- 用于比较非嵌套模型（因变量必须相同）

**不能**用R²比较因变量不同的模型（如y与ln(y)）

### 当因变量为ln(y)时的预测
```
预测ŷ = exp(σ̂²/2) · exp(ln(y)的预测值)
```
前提：u独立于x（否则有偏）

### 控制过度
- 不应控制内生中间变量（如研究啤酒税→交通死亡，不应控制啤酒消费量）
- 不同回归目的决定是否包含某些变量

## Ch8 异方差性

### 定义
同方差：Var(u|x₁,...,xₖ) = σ²
异方差：Var(u|x₁,...,xₖ) ≠ 常数（随x变化）

### 异方差对OLS的影响
- **不影响**：无偏性、一致性、R²的解释
- **影响**：
  1. OLS方差公式无效（标准误不正确）
  2. t检验、F检验失效
  3. OLS不再是BLUE（存在更有效的线性估计量）

### 异方差检验

**1. 布罗施-帕甘（Breusch-Pagan）检验**

步骤：
1. OLS估计原模型，得残差û²
2. û²对所有x做辅助回归：û² = δ₀ + δ₁x₁ + ... + δₖxₖ + v
3. 检验H₀: δ₁=δ₂=...=δₖ=0

检验统计量：
- F统计量（常规F检验）
- LM统计量 = n·R²_û² ~ χ²(k)

**2. 怀特（White）检验**

辅助回归包含所有x、x²、交叉项（更一般，检测更多形式异方差）

**简化版本**：û²对ŷ和ŷ²做回归（节省参数）

### 加权最小二乘法（WLS）

当异方差形式已知（如 Var(u|x) = σ²·h(x)）：

1. 原模型除以√h(x)：
   y/√h = β₀/√h + β₁x₁/√h + ... + u/√h
2. 变换后模型误差同方差
3. 对变换后模型做OLS = WLS（权重 1/h）

**WLS核心思想**：方差大的观测信息少 → 权重小

### 可行GLS（FGLS）

异方差形式未知时的两步法：
1. 假设 Var(u|x) = σ²·exp(δ₀ + δ₁x₁ + ...)
2. OLS得残差û，取ln(û²)对x做回归得估计δ̂
3. 用 ĥ = exp(xδ̂) 作为权重做WLS

**特点**：一致且渐近比OLS更有效

### 异方差稳健标准误（White/Huber标准误）

- 无论异方差形式如何，大样本下有效
- 无需知道异方差的具体形式
- **建议始终报告**稳健标准误

稳健标准误公式（以简单回归为例）：
```
se_robust(β̂₁) = √[ Σ(xᵢ-x̄)²ûᵢ² ] / Σ(xᵢ-x̄)²
```

### 线性概率模型（LPM）

当y为0/1二值变量时：
- 异方差必然存在：Var(y|x) = p(x)[1-p(x)]
- 已知异方差形式：h(x) = ŷ(1-ŷ)
- 若ŷ<0或>1，WLS不可行，改用OLS+稳健标准误

## 假设总结

| 编号 | 内容 | 违反后果 |
|------|------|----------|
| MLR.1 | 参数线性 | 模型设定错误 |
| MLR.2 | 随机抽样 | 可能有偏 |
| MLR.3 | 无完全共线性 | 无法估计 |
| MLR.4 | E(u|x)=0 | OLS有偏且不一致（内生性） |
| MLR.5 | Var(u|x)=σ² | OLS非有效，标准误无效 |
| MLR.6 | u~N(0,σ²) | 大样本下不影响推断 |

**核心逻辑链**：
MLR.1-MLR.4 → 无偏性 → MLR.1-MLR.5 → BLUE（高斯-马尔可夫）→ MLR.1-MLR.6 → t/F检验精确有效（CLM）

> 小样本靠假设，大样本靠渐近（CLT+大数定理，可放松MLR.6）
