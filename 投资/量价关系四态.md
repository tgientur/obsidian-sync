# 量价关系四态

> 来源：东方财富 2026年3月-6月 A股银行股真实数据

---

## 底层逻辑

任何资产的交易，价格和成交量是联动的。成交量的本质是「分歧程度」，价格的变动方向是「分歧的结果」。

**一个公式帮你去理解所有量价关系：**

```
成交量 = 想买的人 × 想卖的人
```

- 成交量放大 → **多空分歧加剧**，双方都在用力
- 成交量缩小 → **多空一致**，要么都不想买，要么都不想卖

价格变动的方向和成交量的状态组合起来，就得到四个基本信号。

---

## 一、放量上涨 —— 多头发力，共识形成中

**定义**：价格上涨，成交量显著放大（通常达到前一日或均量的 1.5 倍以上）。

**底层含义**：
想买的人多且坚决，想卖的人也愿意在这个价格出手。买方愿意不断出高价吃掉卖单，价格被推上去。这是一个**健康的趋势信号**——说明上涨由真金白银推动，而不是少数人的自弹自唱。

**但注意**：放量不一定代表"继续涨"。爆量（比如日常均量的 3 倍以上）往往意味着**分歧达到极端**，短期可能见顶。

### 真实案例

**工商银行 2026-03-03**：前一天涨 0.59%（量 373万），当天涨 **2.36%**（量 **570万**，放大 1.5 倍）。

```
3/2: 收盘 6.79 涨 0.59%  量 373万  ← 温和
3/3: 收盘 6.95 涨 2.36%  量 570万  ← 放量突破
3/4: 收盘 6.91 跌 0.58%  量 378万  ← 随后回踩但缩量
```

这组典型：前一日温和上涨，次日放量突破，之后缩量回踩支撑。

**农业银行 2026-04-02**：涨 **3.51%**（量 **562万**），是前一日 331万的 1.7倍。

```
4/1: 收盘 6.56 跌 0.15%  量 331万
4/2: 收盘 6.79 涨 3.51%  量 562万  ← 放量大阳线
4/3: 收盘 6.71 跌 1.18%  量 365万  ← 缩量回踩
```

农业银行在这天一根 3.5% 的大阳线，量放大到 562万，这是**主力资金确定性进场**的信号。

---

## 二、缩量上涨 —— 分歧消失，一致看多

**定义**：价格上涨，但成交量比之前明显缩小。

**底层含义**：
想卖的人少了——持筹者不愿意在这个价格出售（惜售），少量买盘就能把价格推上去。这种状态的潜台词是：**市场上没有抛压**。

缩量上涨是一个**看多延续信号**，但需要区分位置：
- **初期/中途缩量上涨** → 良性，筹码锁定良好，上涨还能持续
- **高位缩量上涨** → 危险，买盘在枯竭，随时可能转头向下，俗称"量价背离"

### 真实案例

**工商银行 2026-03-25 至 2026-03-26**：

```
3/24: 收盘 7.15  涨 0.99%  量 376万  ← 调整后的反弹
3/25: 收盘 7.17  涨 0.28%  量 337万  ← 量开始缩
3/26: 收盘 7.25  涨 1.12%  量 263万  ← 缩量上涨！量比前日少了22%
```

这一天 3/26 涨了 1.12% 但量从 337万缩到 263万。说明这个位置的持筹者不想卖，买盘不重也能涨。这是一个**筹码锁定健康的缩量上涨**，之后的走势也验证了——价格从 7.25 一路涨到 3/31 的 7.47。

**招商银行 2026-03-10 至 2026-03-13**：

```
3/9:  收盘 38.79 跌 1.05%  量 76万  ← 调整
3/10: 收盘 39.22 涨 1.11%  量 61万  ← 缩量反弹
3/11: 收盘 39.50 涨 0.71%  量 68万  ← 继续缩量
3/12: 收盘 39.50 涨 0.00%  量 52万  ← 量最萎缩
3/13: 收盘 39.76 涨 0.66%  量 76万  ← 开始放量
```

招行经过 3/9 的调整后，从 3/10 开始连续 3 天缩量反弹。这说明调整的抛压已经释放完了，不需要大量买盘就能把价格拉回——**抛压枯竭的特征**。

---

## 三、放量下跌 —— 恐慌出逃，多空激战

**定义**：价格下跌，成交量显著放大。

**底层含义**：
想卖的人很多且不顾价格地出逃，想买的人试图接盘但接不住，价格被打下去。这是**最弱的信号**——说明抛压巨大，有资金在主动离场。

主要场景：
- **利空消息**恐慌砸盘
- **主力出货**（高位放量下跌最危险）
- **止损盘**集中出逃

### 真实案例

**农业银行 2026-03-23**：跌 **4.37%**（量 **548万**，前一日只有 368万，放量 1.5倍）。

```
3/22(五): 收盘 6.63  跌 1.04%  量 368万
3/23(一): 收盘 6.34  跌 4.37%  量 548万  ← 放量暴跌！
```

一天跌了 4.37%，量放大到 548万——这是非常典型的**恐慌性放量下跌**。之后用了将近两周时间才从 6.34 回到 6.57，修复很慢。

**工商银行 2026-03-23**：同一天跌 **3.41%**（量 **502万**，是前一日 391万的 1.3倍）。

```
3/20: 收盘 7.33  跌 0.41%  量 391万
3/23: 收盘 7.08  跌 3.41%  量 502万  ← 放量下跌
同日农业银行跌 4.37%，招商银行跌 2.69%
```

这是全市场的银行板块共振下跌，大概率跟宏观利空相关。全板块同日放量下跌，性质比个股单独放量下跌更严重——**系统性风险**。

**招商银行 2026-04-29**：跌 **2.78%**（量 **253万**，是前一日 66万的 **3.8倍**！）

```
4/28: 收盘 39.60  涨 0.56%  量 66万   ← 正常
4/29: 收盘 38.50  跌 2.78%  量 253万  ← 爆量下跌！
```

量突然放大到前一天的 3.8 倍——**异常放量**。这种爆量下跌往往是有特定利空（可能是招行年报/季报不及预期）。之后价格从 38.50 一路跌到 36.94（约 4%），持续了半个月的弱势。

## 四、缩量下跌 —— 抛压衰竭，下跌动能减弱

**定义**：价格下跌，但成交量显著缩小。

**底层含义**：
想卖的人少了，但买盘也不积极，价格在惯性滑落。缩量下跌意味着**抛压在自然衰减**——该卖的人都卖了，剩下的都是不愿意卖的人。

信号意义：
- 下跌**初期**的缩量 → 买盘不足，还可能继续跌
- 下跌**末期**的缩量 → 筑底信号，空头力量衰竭，随时可能反弹

**缩量下跌 + 之后的放量上涨 = 最经典的底部反转形态。**

### 真实案例

**农业银行 2026-04-09 至 2026-04-10**：

```
4/7:  收盘 6.56  跌 2.24%  量 304万  ← 放量下跌
4/8:  收盘 6.50  跌 0.91%  量 370万  ← 继续放量
4/9:  收盘 6.47  跌 0.46%  量 247万  ← 缩量！抛压减
4/10: 收盘 6.46  跌 0.15%  量 220万  ← 进一步缩量
4/13: 收盘 6.51  涨 0.77%  量 283万  ← 开始放量反弹
```

这是一个**完整的底部反转过程**：放量下跌（恐慌出逃）→ 缩量下跌（抛压衰竭）→ 放量上涨（买盘回归）。之后的涨幅：6.46 → 6.93（约 7%）。

**工商银行 2026-03-19 至 2026-03-20**：

```
3/19: 收盘 7.36  涨 1.94%  量 560万  ← 放量大涨
3/20: 收盘 7.33  跌 0.41%  量 391万  ← 缩量回调
```

涨了 1.94% 后的回调只有 0.41%，而且量从 560万缩到 391万——说明没有人在获利后大规模出逃。这是一个**健康的缩量回调**，不是趋势反转。之后确实继续涨到了 3/31 的 7.47。

## 汇总对照表

| 形态 | 价格方向 | 成交量 | 核心含义 | 信号强度 | 主要风险 |
|---|---|---|---|---|---|
| 放量上涨 | ↑ | ↑↑ | 多头真金白银进场 | ✅ 强看多 | 爆量可能见顶 |
| 缩量上涨 | ↑ | ↓ | 惜售，少量买盘即可推升 | ✅ 温和看多 | 高位时是顶背离 |
| 放量下跌 | ↓ | ↑↑ | 空头恐慌出逃 | ❌ 强烈看空 | 系统性风险 |
| 缩量下跌 | ↓ | ↓ | 抛压衰竭，空头力量减弱 | ⚠️ 看空减弱 | 初期可能持续阴跌 |

## 实战口诀

```
放量上攻跟主力，缩量推升别下车
放量下砸先撤离，缩量阴跌等反转
爆量不论涨跌都要警惕——极端分歧意味着变盘
```

## 按场景组合

### 最安全的多头信号
**缩量下跌（底部）→ 放量上涨（突破）**

真实案例：农业银行 2026-04-10 缩量见底 → 4/14 放量 1.84% 突破

### 最危险的空头信号
**放量上涨（高位）→ 放量下跌（出逃）**

真实案例：工商银行 2026-03-19 放量大涨 1.94%（560万） → 3/20 缩量回调（391万）→ 3/23 放量暴跌 3.41%（502万）。高潮之后放量暴跌，是最经典的顶部形态。

### 最容易误判的场景
**缩量上涨（高位）**：看着价格还在涨，但量在萎缩。这不是"筹码锁定的好"，而是"买盘跟不上"。高位缩量上涨后一旦放量下跌，就是双顶/头部的确认信号。
