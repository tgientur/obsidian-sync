# 金融科技对传统银行行为的影响——基于互联网理财的视角

- **作者**：邱晗、黄益平、纪洋
- **期刊**：《金融研究》2018年第11期（总第461期）
- **关键词**：金融科技；银行负债结构；银行风险承担
- **JEL**：E40, G18, G21

---

## 研究问题

金融科技发展（以互联网理财为切入点）如何影响银行的负债结构、资产风险和利差？

## 数据与方法

- **样本**：263家银行，2011—2015年
- **核心解释变量**：**北京大学数字普惠金融指数**（地级市层面），来自北京大学数字金融研究中心
- **被解释变量**：负债结构（NIM净息差、LR贷存比、CRA同业负债占比）、资产风险（RISK风险加权资产比）、贷款利率等
- **模型**：面板固定效应 + GMM

## 核心发现

### 1. 负债端：FinTech推动存款利率市场化

| 效应 | 发现 |
|------|------|
| 负债结构变化 | 零售存款占比↓，同业融资等批发融资占比↑ |
| 负债成本上升 | 存款成本提高（余额宝等替代效应） |
| 解释 | FinTech实际上推动了**存款端的利率市场化** |

### 2. 资产端：风险偏好上升但利差下降

| 效应 | 发现 |
|------|------|
| 资产风险上升 | 银行选择更高风险的资产 |
| 贷款利率下降 | 资产平均利率和净息差均下降 |
| 含义 | 银行**没有把成本转嫁给借款企业**，而是自己承担，通过配置高收益资产弥补 |

### 3. 规模异质性

**大型银行受FinTech影响较小**——大银行客群更稳定、负债来源更多元，对存款竞争的敏感度较低。

### 4. 对比分析：2013年前后

以余额宝上线（2013年6月）为界分样本回归：
- 2011—2012年：FinTech对银行行为影响不显著
- 2013—2015年：影响显著增强（余额宝效应）

## 核心结论

> **FinTech发展实质上推动了存款利率市场化 → 银行负债结构变化（零售存款减少、同业融资增加）→ 负债成本上升 → 银行选择更高风险资产弥补损失 → 但贷款利率和净息差反而下降。**

## 对学年论文的参考价值

1. **"负债成本驱动风险承担"机制**被杨望等(2020)和宋科等(2023)引用和拓展，是理解FinTech-银行效率关系的关键一环
2. 北大数字普惠金融指数首次被用于银行微观研究——后续多篇论文沿用
3. 2013年余额宝的政策断点可作为制度背景

## 局限性

- 数据止于2015年（较早），未覆盖2016年后的金融科技监管变化
- 仅使用地级市层面指数，银行层面变量控制较少
- 互联网金融理财产品（余额宝等）只是FinTech的一部分，结论外部性有限
