# 立体几何

> 难度：★★★★ | 分值：15~22 分 | 题型：单选/多选 + 解答题（第16题或第17题）
> 考查特点：小题考空间几何体体积/表面积和位置关系判断，大题考空间向量法求角与距离

---

## 一、核心考点

### 1. 空间几何体

- **柱、锥、台、球的体积与表面积**
  - $V_{\text{柱}} = Sh$，$V_{\text{锥}} = \frac13 Sh$，$V_{\text{球}} = \frac43 \pi R^3$
  - $S_{\text{球}} = 4\pi R^2$
- **三视图与直观图**（新高考考频下降，但仍需掌握）
- **展开图与最短路径**

### 2. 点线面的位置关系

- **平行**：线线平行 → 线面平行 → 面面平行（递推关系）
- **垂直**：线线垂直 → 线面垂直 → 面面垂直（递推关系）
- **判定定理与性质定理**的熟练运用（高考大题第一问通常考证明）

### 3. 空间向量与立体几何（大题核心方法）

- **建系原则**：找三条两两垂直的直线作为坐标轴
- **方向向量与法向量**
- **线面角**：$\sin\theta = |\cos\langle \vec{l}, \vec{n} \rangle|$
- **二面角**：$\cos\theta = \pm \cos\langle \vec{n}_1, \vec{n}_2 \rangle$（注意锐钝判断）
- **点到平面的距离**：$d = \frac{|\overrightarrow{AP} \cdot \vec{n}|}{|\vec{n}|}$

## 三、常用技巧

### 技巧 1：建系的最佳选择
- **长方体/正方体**：直接以三条棱作为坐标轴
- **直棱柱**：底面互相垂直的两条线 + 侧棱为 z 轴
- **正四面体**：底面正三角形中心为原点
- **三棱锥**：找底面上的高线为 z 轴

> 最理想的建系：让尽量多的关键点落在坐标轴上，减少坐标运算量。

### 技巧 2：法向量的快速求法
已知平面内两个不共线向量 $\vec{a} = (a_1,a_2,a_3)$，$\vec{b} = (b_1,b_2,b_3)$：
$$\vec{n} = \vec{a} \times \vec{b} = (a_2b_3 - a_3b_2,\; a_3b_1 - a_1b_3,\; a_1b_2 - a_2b_1)$$
**口诀**：交叉相乘再相减，顺序是 $yz, zx, xy$

### 技巧 3：二面角锐钝判断
- **定义法**：在交线上取一点，分别在两个面内作交线的垂线，看两条垂线的夹角
- **法向量法**：观察法向量的方向，两个法向量同时指向角内部或同时指向外部 → 角为 $\pi - \theta$；一内一外 → 角为 $\theta$
- **经验法**：大部分高考立体几何的二面角是锐角或直角

### 技巧 4：存在性问题的统一解法
设点 $P$ 在某线段上，参数表示为 $P = A + t\overrightarrow{AB}$（$t \in [0,1]$）→ 代入条件（平行/垂直/等角/距离）→ 解方程求 $t$

### 技巧 5：外接球问题的通法
- **长方体模型**：若几何体可补成长方体，外接球直径 = 体对角线长
- **直棱柱**：球心在底面外接圆圆心与上底面外接圆圆心的中点
- **三棱锥**：找两个面的外心 → 过外心作面的垂线 → 垂线交点即为球心
- **通用公式**：$R^2 = r^2 + \left(\frac{h}{2}\right)^2$（$r$ 为底面外接圆半径，$h$ 为高）

## 四、二级结论

1. **正四面体的高与体积**：
   棱长为 $a$ 的正四面体，高 $h = \frac{\sqrt{6}}{3}a$，体积 $V = \frac{\sqrt{2}}{12}a^3$

2. **长方体外接球直径**：
   长方体长宽高 $a,b,c$，外接球半径 $R = \frac{\sqrt{a^2+b^2+c^2}}{2}$

3. **三余弦定理**（最小角定理）：
   $\cos\theta = \cos\alpha \cdot \cos\beta$（斜线与平面内一条线夹角 $\theta$ = 斜线与平面夹角 $\alpha$ + 该线在平面内与斜线投影的夹角 $\beta$）

4. **三垂线定理**：
   平面内一条线垂直于斜线的投影 → 这条线垂直于斜线本身

5. **等体积法**（求点面距离）：
   $V_{A-BCD} = V_{B-ACD}$，利用两种方式计算体积，列方程求高

6. **正方体中的常见截面**（选填题常考）：
   - 对角线截面是矩形
   - 中点截面是正六边形

## 五、易错清单

- [ ] 建系后注意坐标轴方向的选择（左/右手系不影响结果，但需一致）
- [ ] 法向量的计算不要漏掉负号
- [ ] 线面角是 $\sin\theta = |\cos\langle \vec{l}, \vec{n} \rangle|$，**不是**直接等于夹角余弦
- [ ] 二面角是两法向量夹角或其补角，需根据图形判断
- [ ] 点到平面距离公式中，分子是**绝对值**，结果为正值
- [ ] 立体几何证明题先写条件再下结论，不可跳跃


> **总评**：立体几何大题是"套路化"程度最高的板块——建系、求法向量、公式代入三步走。拿满分的关键在于运算准确和文字表述完整。建系前的几何证明（第一问）不要忽视。
