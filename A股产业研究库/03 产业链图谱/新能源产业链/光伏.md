# 光伏

## 定义与产业定位

光伏（Photovoltaic，PV）是利用半导体材料的光生伏特效应，将太阳光直接转化为电能的发电技术。光伏产业链以硅材料为核心——从高纯度多晶硅（纯度99.9999%+）拉晶/铸锭、切割成硅片，制成电池片，封装为组件，配合逆变器等电力电子设备实现并网发电。

**产业定位**：光伏是全球能源转型的核心方向之一，也是中国制造业"全球第一"的代表性产业。中国企业在硅料、硅片、电池片、组件四大环节的全球产能占比均超过80%（数据来源：[1]）。2025年全球光伏新增装机约650GW，中国贡献约300GW。

光伏产业经历了从"政策补贴驱动"到"平价上网驱动"的根本性转变。中国自2021年进入光伏平价时代（光伏发电成本已低于燃煤标杆电价），光伏从"需要补贴的新能源"变成"最便宜的电源"。当前行业处于产量过剩→产能出清→格局重塑的周期底部。

从A股投资视角看，光伏是典型的重资产周期性制造业——在供不应求阶段暴涨如2020-2021年，在产能过剩阶段暴跌如2023-2024年。理解光伏产业周期的位置（出清进度、盈利底部、拐点信号）是投资的核心。

## 产业链分析

### 产业链结构

```
硅料 → 硅片 → 电池片 → 组件 → 逆变器 → 电站/EPC → 运营维护
  ↑       ↑       ↑       ↑        ↑
 多晶硅   拉晶    制绒    封装     DC/AC
 硅烷气   切片    PERC   层压     MPPT
 电力    清洗    TOPCon 接线盒   MLPE
 IC级硅  分选    HJT/BC  边框     智能优化
```

### 价值流动与利润分配（正常周期下）

|:----|:------------:|:---------------:|:-------:|:-----:|:--------|
| 硅料 | 50-70%（高峰） | 5-15% | ~15% | 中（化工属性，供给弹性低） | 寡头（通威/协鑫/大全/新特） |

### 定价权分析

逆变器是光伏产业链中定价权最强的环节。阳光电源和华为数字能源在全球逆变器市场合计市占率超过50%（数据来源：[1]），技术壁垒（大功率逆变器的MPPT算法、散热设计、电网接入兼容性）和品牌壁垒（大型电站业主对逆变器可靠性的极度重视）共同构成护城河。

硅料环节在上升周期中定价权极强（供给刚性+需求弹性），在下降周期中定价权快速丧失（2023-2024年硅料价格暴跌约80%）。

组件环节定价权最弱——产品同质化严重（所有组件封装结构和功能几乎一致），品牌忠诚度有限，行业长期处于价格战状态。

### 成本传导

光伏产业链的成本传导具有鲜明的"由下往上"特征：

```
组件降价压力 → 电池片降价 → 硅片降价 → 硅料降价
```
下游的降价压力从组件端开始，逐级向上游传导，最终由硅料环节承担最大的价格弹性。2023-2024年组件价格从￥1.8/W跌至￥0.7/W（跌幅61%），硅料从￥30万/吨跌至￥4万/吨（跌幅87%），上游弹性远大于下游。

### 产业分工演变

光伏产业分工经历了从"垂直一体化→专业化→垂直一体化"的轮回：
- **2020年前**：行业高度专业化——通威（硅料+电池）、隆基（硅片+组件）、爱旭（电池）、晶科/天合/晶澳（组件）
- **2021-2022年**：隆基等龙头转向硅片+电池+组件垂直一体化
- **2023-2025年**：产能过剩导致专业化厂商亏损加剧，只有通威（从硅料到组件全链条）和隆基（一体化+BC技术）等少数企业能维持盈利。垂直一体化成为穿越周期的重要能力

### 利润来源与现金流特点

光伏制造是典型的"重资产、强周期、资本密集"模式：
- **毛利率**：正常周期下硅料/电池片/逆变器25-40%，硅片/组件10-20%。2023-2024年行业整体毛利率大幅收缩，除逆变器外各环节毛利率普遍降至10%以下
- **净利率**：正常周期硅料15-30%、逆变器12-18%、硅片/组件5-8%。2024年大量企业亏损
- **经营现金流**：2020-2022年强劲，2023-2024年随利润下滑大幅收缩，部分企业出现经营性现金流净流出
- **资本开支**：极高（产能建设），2020-2024年行业累计资本开支超￥1万亿，导致大量折旧拖累利润

## 行业壁垒分析

### 技术壁垒

**多晶硅**：改良西门子法/硅烷流化床法的工艺积累需要10年+，电力电耗（kg-Si耗电约45-55度）和纯度控制（电子级/太阳能级）决定了生产成本。

**电池片**：从P型PERC到N型TOPCon/HJT/BC的技术迁移速度决定了企业的生存能力。2025年TOPCon已成为主流（市占率约60%），HJT约10%，BC约15%。技术路线的选择错误是致命的——隆基2022年押注BC路线导致TOPCon红利期市占率下降。

**逆变器**：大功率组串式逆变器的MPPT算法、I/V曲线扫描、EMS能效管理、电网适应性（弱电网/高谐波/SCR<1.1）、安全功能（AFCI电弧检测）等综合技术壁垒。

### 资本壁垒

光伏是超级重资产行业：10万吨硅料产能投资约￥60-80亿（建设周期18-24个月），10GW电池片投资约￥25-35亿（12-18个月），10GW组件投资约￥5-8亿（6-12个月）。2020-2024年行业累计资本开支预计超￥1.2万亿。

### 规模壁垒

光伏制造的规模经济效应极强：通威2025年硅料产能约58万吨（全球第一），单吨成本约￥3.5万，较10万吨以下的小厂低15-20%（数据来源：[1]）。隆基硅片产能超150GW，单W制造费用较二线厂商低30%。小企业在大规模价格战中完全无法生存。

### 客户/品牌壁垒

逆变器环节品牌壁垒最强——大型光伏电站业主对逆变器有长达10-15年的质保要求，逆变器故障会导致整站发电损失。阳光电源和华为在全球建立了极强的品牌信任。

组件环节品牌壁垒有限——价格是招标的决定性因素，溢价空间仅约0.01-0.02元/W。

### 认证壁垒

光伏产品出口需要IEC 61215（组件性能）、IEC 61730（组件安全）、IEC 62477（逆变器安全）等国际认证。欧洲还需要CE/ TUV认证，美国需要UL认证/反倾销税评估。出口到美国市场面临的201/301关税和反倾销税是额外壁垒。

### 供应链壁垒

光伏制造的上游供应链（EVA粒子/光伏玻璃/铝边框）已充分国产化，不是核心壁垒。设备供应链（拉晶炉/切片机/PECVD/丝网印刷）的国产化率超过95%（数据来源：[1]），且设备商（晶盛机电/迈为股份/捷佳伟创/帝尔激光）充分竞争。供应链本身不构成壁垒，反而加剧了行业的可复制性和产能过剩。

## 历史行情复盘

### 第一期：2020年4月-2022年1月——供需两旺与全面繁荣

**事件**：中国2020年9月提出"2030碳达峰、2060碳中和"，光伏进入景气巅峰。2020-2021年全球光伏新增装机从134GW增至175GW。硅料因产能刚性（建设周期18-24个月）成为最紧缺的环节，价格从￥6万/吨暴涨至2021年底￥28万/吨。

**市场表现**：通威股份从2020年4月的￥10涨至2021年9月的￥62（复权，涨幅520%）。隆基绿能从￥14涨至￥103（复权，涨幅636%）。阳光电源从￥8涨至2021年10月的￥167（涨幅1988%）。锦浪科技从￥15涨至￥280（涨幅1767%）。福斯特（胶膜）从￥12涨至￥76（涨幅533%）。

**上涨原因**：碳中和大叙事+光伏平价上网+硅料紧缺三重驱动。市场把光伏当成"确定性的成长赛道"来定价，给到了极高的估值溢价。

**资金结构**：公募基金2021年对新能源板块的配置达到历史峰值（占全部公募持仓的20%+）。阳光电源是公募持仓最集中的光伏标的之一。北向资金2021年净买入光伏板块超￥500亿。

**龙头表现**：阳光电源（1988%）、锦浪科技（1767%）、隆基绿能（636%）、通威（520%）。

**最终结果**：2022年1月板块见顶回落。隆基从￥103跌至2023年4月￥32（回撤69%）。阳光电源从￥167跌至￥55（回撤67%）。估值泡沫破裂+行业过剩预期形成。

### 第二期：2022年5月-2022年10月——反弹窗口

**事件**：2022年Q2-Q3欧洲能源危机（俄乌冲突）冲击，欧洲光伏发电需求暴增。欧洲户用储能和分布式光伏需求极度旺盛——中国逆变器/组件出口欧洲同比增速超过100%（数据来源：[1]）。

**市场表现**：阳光电源从2022年4月低点￥55反弹至8月￥119（涨幅116%）。锦浪科技从￥110反弹至￥200（涨幅82%）。固德威从￥150反弹至￥350（涨幅133%）。组件龙头（晶科/天合/晶澳）涨幅40-60%。

**上涨原因**：欧洲能源危机带来的短脉冲需求。市场将欧洲光伏需求定性为"长期结构性增长"。

**最终结果**：2022年四季度后逆变器板块再次回落。俄乌冲突导致的能源危机并未持续驱动光伏需求。欧洲天然气价格在2022Q4见顶回落，光伏出口脉冲结束。

### 第三期：2023年至今——产能过剩与持续出清

**事件**：2023年中国光伏新增装机216GW（YoY+148%），2024年约280GW（YoY+30%），增速放缓。但产能扩张更为猛烈——2024年硅料产能超300万吨（可用量足以生产超1500GW组件），超过全球需求2倍以上。

2023年硅料价格从￥30万/吨暴跌至￥10万/吨，2024年进一步跌至￥4万/吨，跌破全行业现金成本。2024年全产业链亏损——通威亏损、隆基亏损、TCL中环亏损、爱旭亏损。行业进入痛苦的"出清"阶段。

2025年信号：部分高成本产能关停（青海丽豪/新疆协鑫等边际产能停产），行业产能利用率降至50-60%（数据来源：[1]）。龙头企业开始整合（通威收购润阳等）。

**市场表现**：光伏指数（884045.WI）从2022年高点的13000点跌至2024年低点的3500点，跌幅超73%。隆基绿能从￥68跌至￥12（回撤82%），蒸发逾4000亿市值。通威从￥62跌至￥15（回撤76%）。阳光电源从￥167跌至￥50（回撤70%），但2025年已回升至￥80（逆变器盈利韧性+储能放量）。

**资金结构**：公募清仓式减持——光伏板块持仓从2021年20%（数据来源：[1]）降至2025年<5%。北向资金持续净卖出。板块从"核心资产"沦落为"周期股"。

**经验总结与规律**：
- 第一阶段（2020-2022）：需求爆发+供给紧缺，龙头涨幅500-2000%，估值泡沫化
- 第二阶段（2022）：欧洲能源危机脉冲，涨幅60-130%，短命反弹
- 第三阶段（2023-2025）：产能过剩+全行业亏损，指数跌超70%，部分企业跌破现金成本

核心规律：光伏是"好赛道、烂生意"的典型——长期需求确定（能源转型），但制造业壁垒低、竞争同质化、产能扩张无节制。投资光伏需要坚定的周期思维：在行业亏损最惨烈的底部布局（2024-2025年），在盈利修复+估值修复的双击阶段持有，在行业全面盈利高峰卖出。

## 投资逻辑分析

### 超额收益来源

光伏行业当前处于周期底部，超额收益来自三个路径：

1. **产能出清后的格局改善**：2025年是光伏产能出清的关键年份。中小企业退出、行业整合加速后，龙头企业的市场份额和定价权将显著提升。通威在硅料/电池环节、隆基在硅片/组件环节、阳光在逆变器环节的龙头地位将进一步巩固。

2. **技术迭代的结构性机会**：N型TOPCon→BC/HJT→钙钛矿叠层的技术演进路径清晰。BC和钙钛矿叠层（2027-2028年量产预期）的突破将带来设备商（迈为/捷佳伟创/帝尔激光）和率先量产的企业（隆基BC/华晟HJT）的结构性机会。

3. **逆变器+储能的终端价值延伸**：光伏从"发电脑"向"发电+储能+能效管理"的智能电站演进。阳光电源的逆变器+储能+EPC综合方案能力构成差异化竞争壁垒。

### 景气传导

```
产能出清（2024-2025） → 产能利用率企稳 → 价格止跌
    ↓
龙头企业盈利拐点 → 行业产能利用率回升至70%+ → 毛利率修复
    ↓
新技术渗透率提升（BC/HJT/钙钛矿） → 设备更新周期
    ↓
全球光伏装机持续增长（年增10-15%）→ 需求端消化过剩产能
```

### 未来催化剂

1. **产能出清完成信号**：多晶硅/组件价格企稳回升，行业重资产折旧接近尾声
2. **BC/钙钛矿技术重大突破**：隆基BC电池量产效率突破28%（数据来源：[1]），或钙钛矿叠层电池实验室效率突破35%
3. **全球光伏装机超预期**：IRENA/CPIA上调中长期装机预测，打开估值空间
4. **碳关税落地**：欧盟CBAM正式征收碳关税，中国光伏组件的低碳优势转化为溢价
5. **电力市场化改革深化**：光伏绿电溢价交易机制完善，电站IRR提升

## 主要风险

1. **产能过剩风险**：光伏各环节产能严重过剩，价格持续承压
2. **政策退坡风险**：补贴退坡/消纳红线调整
3. **贸易摩擦风险**：欧美对中国光伏/风电产品的贸易壁垒

## 舆论风向

**雪球**：
- "光伏到底什么时候见底"——雪球上讨论最多的制造业问题之一
- "隆基绿能还能不能回本"——从2021年的"价值投资典范"沦为2025年的"套牢重灾区"
- 通威、TCL中环等企业跌破净资产的讨论——周期底部特征还是价值陷阱？

**知乎**：
- "产能过剩的制造业该怎么投资"——光伏是最经典的案例
- "钙钛矿是未来还是噱头"——技术讨论非常深入
- "中国光伏怎么越来越便宜了？"——从产业角度分析成本下降和价格战

**微博/财经大V**：
- 光伏板块消息的传播速度很快——每次硅料/组件价格波动都在大V群引发讨论
- "光伏一涨就是春天，一跌就是冬天"——投资者情绪的极端波动
- 欧洲光伏需求、美国反倾销调查等海外消息对板块情绪的影响巨大

**关键话语特征**：光伏是A股最"情绪化"的板块之一——从2021年的"YYDS"到2024年的"行业末日"，舆论在两极之间剧烈摇摆。当下对光伏的投资讨论已从"成长价值"转为"周期博弈"。
