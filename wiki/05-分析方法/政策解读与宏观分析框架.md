## 第一部分：政策新闻解读法

### 核心思路

把每一条政策新闻放回决策链条里，找到它在棋盘上的位置。你得到的不是一地碎片，而是一张完整的地图。读新闻不再是读孤立的单集，而是一部连续剧。

### 步骤一：看会场里坐着谁

最高级别的专题学习/会议，参会部门本身就是信息。例如国务院专题学习，除了总理，通常有：

- 国家发改委
- 财政部
- 自然资源部
- 生态环境部
- 科技部
- 国家能源局
- 相关领域的央企负责人
- 相关学者

**分析方法**：谁被叫来开会，说明这件事跟谁的职能范围有关。没被叫来的部门，说明这件事暂时不涉及他们的地盘。

来源：知乎 @不熬夜睡觉冠军，《该遵从什么样的思路去解读政策？》，https://www.zhihu.com/question/329365774/answer/2030905604689109518

### 步骤二：分清楚「钱袋子」和「权袋子」

这是最关键的一步。把参会的部门分成两类：

**钱袋子（管钱、管投资的部门）：**
- 财政部 — 管公共预算，决定公共资金往哪个方向拨
- 国家发改委 — 管重大项目审批和投资导向
- 央行（如涉及） — 管货币政策和信贷投放

**权袋子（管政策、管准入、管标准的部门）：**
- 国家能源局 — 掌握行业发展规划、项目核准、市场主体准入
- 自然资源部 — 管土地、矿产等资源审批
- 生态环境部 — 管环评、排放标准
- 科技部 — 管技术路线和科研经费

**分析方法**：跟踪一个细分方向时，"权袋子"出政策消息要先判断是不是真落地（有没有配套的"钱袋子"动作）。只有"权袋子"喊话没有"钱袋子"掏钱，说明还在吹风阶段。当"钱袋子"开始动了，才是实质性的政策落地信号。

来源：同上

### 步骤三：跟踪时间差——让日期自己说话

这是最硬核的技巧。**看文件底部的落款日期**，对比发布日期。

案例（原文案例）：
```
4月11日 —— 意见已在内部签发（落款日期）
4月20日 —— 国务院举行第十九次专题学习（统一思想、动员部署）
4月22日 —— 文件正式对外公布
```

这十一天的时间差说明了什么？

1. **从签发到公布中间隔着一段时间**——这段时间用来找最合适的发布时机、等内外部条件成熟、做配套的舆论和部门协调准备。
2. **专题学习有明确功能**——高层带着整个条线的负责人集体"上课"、公开表态，是在向社会释放信号：这件事是最高层级的优先级。
3. **三步连在一起构成完整动作**：内部行文定调 → 高层学习动员 → 向社会公布执行。从决策到动员再到落地。

**核心方法**：把几件事的时间线列在一起，时间差本身就表达了政策推进的内部节奏。

来源：同上

### 步骤四：把新闻串成连续剧

单条新闻没有意义，放在决策链条里才有位置。问自己三个问题：

1. 这条新闻在决策链条的第几步？（定调？动员？落地？）
2. 参与的是"钱袋子"还是"权袋子"？（实质性进展还是吹风？）
3. 往前推，什么事件触发了它？往后看，下一步应该出现什么？

来源：同上

## 第二部分：政策如何传导到股价（供需分析框架）

> 这是第一个框架的延伸——看懂政策之后，怎么用它来做投资判断。

### 框架总览：四层递进

```
第四层（最上游）：政治周期 —— 决定政策方向
     ↓
第三层：地缘冲突 —— 调节风险偏好
     ↓
第二层：金融资产供需 —— 货币水量决定估值高度
     ↓
第一层（最直接）：实体经济供需 —— 政策制造需求突变
```

贯穿四层的红线：**一切最终服务于经济。**

来源：知乎 @不熬夜睡觉冠军，《股价波动的本质系列——供需篇：政策与宏观变革如何制造供不应求》，https://zhuanlan.zhihu.com/p/2026931170202384138

### 第一层：实体经济供需路径

**逻辑链条**：宏观冲击（经济危机、气候压力、技术革命）→ 触发政策战略性响应 → 政策通过可量化目标或大规模资源配置计划 → 人为制造终端需求突变 → 在供给刚性的产业链环节制造实物资产的供不应求。

**经典案例**：

| 宏观冲击 | 政策响应 | 人为制造的需求 | 受益环节 |
|----------|----------|----------------|----------|
| 2008年全球金融危机 | 四万亿 + 棚户区改造（2014-2018） | 大规模拆迁催生的购房需求 | 房地产全产业链 |
| 气候变化压力 | 3060碳中和目标（2020年） | 新能源装机翻倍、电网改造、电动车爆发 | 光伏、风电、电池、电网设备 |
| AI技术革命 | 新一代人工智能发展规划（2017）+ AI+（2023） | 算力基础设施需求暴增 | 光模块、服务器、国产算力 |

**三个案例共享同一个底层模式**：重大宏观冲击 → 政策制定可量化的宏大目标 → 围绕目标配置资源 → 产业链某一环节出现供需失衡 → 价格/利润暴增。

来源：同上

### 第二层：金融资产供需路径

**核心逻辑**：全球货币政策周期决定"货币水量"的多少和流向。

- 美联储加息 → 无风险利率上升 → 全球风险资产估值基准下移 → 美元走强 → 资金从非美市场流出
- 美联储降息 → 无风险利率下降 → 全球风险资产估值基准上移 → 美元走弱 → 资金向非美市场分散

**两条路径叠加才有完整画面**：
- 有故事没水 → 股价涨不动（实体经济需求逻辑成立，但流动性不支持估值）
- 有水没故事 → 资金不知道该去哪里（流动性充裕，但没有产业主线）
- **有故事又有水** → 真正意义上的供不应求，主升浪

这正是2023-2025年A股AI板块的完整解释：AI基本面（第一层逻辑）存在，但2023-2024年美联储加息压制估值（第二层逻辑），直到2024年9月首次降息后，2025年降息周期延续，两条路径同时成立，板块才突破此前无法逾越的高度。

来源：同上

### 第三层：地缘冲突——风险偏好的调节变量

**核心公式**：
```
实际流向非美市场的资金量 = 全球货币水量（美联储周期） × 地缘风险偏好系数
```

- 地缘冲突加剧 → 风险偏好系数降低 → 即使降息，资金也不愿意回流非美市场
- 地缘冲突缓和 → 风险偏好系数回升 → 降息效果真正释放

2022-2024年俄乌冲突、中东紧张、中美博弈叠加加息，形成了双重压制。2025年冲突阶段性缓和叠加降息周期，才出现资金重新向非美市场分配。

来源：同上

### 第四层（最上游）：政治周期

**政治周期决定了**：宏观冲击后"政策响应"的方向、力度和持续性。

不同党派的政策偏好差异：

| 政治倾向 | 财政偏好 | 产业偏好 | 货币政策偏好 |
|----------|----------|----------|-------------|
| 民主党（偏左） | 大政府、高支出、再分配 | 清洁能源、基建、社会福利 | 倾向于宽松 |
| 共和党（偏右） | 小政府、减税、放松监管 | 传统能源、国防、制造业回流 | 关注通胀 |
| 欧洲左翼 | 扩大公共支出、福利国家、绿色转型 | 绿色新政、碳边境税 | 倾向宽松 |
| 欧洲右翼 | 财政纪律、减税 | 保护传统产业（汽车、钢铁） | 担忧通胀 |

**政治周期服务于经济周期**：经济过热时选民倾向保守派以控制通胀，经济衰退时倾向进步派以寻求刺激。政党为了连任，会在经济周期的不同阶段采取不同的政策组合。

来源：同上

### 终极认知：一切服务于经济

这不是空洞的口号，而是有切实的分析意义：

- 看地缘冲突 → 追问它服务于哪一方的经济重构（如欧洲被迫从俄气转向美LNG和新能源）
- 看政党更替 → 追问新执政联盟背后站着哪些产业利益集团
- 看政策转向 → 追问它服务于哪个经济周期的哪个阶段需求

**养成"向下追问"的习惯**：每一次事件都不是孤立的，而是经济周期运行到某一阶段的必然表现。

## 两套框架的关系

政策新闻解读法（第一部分）是**输入端**的功夫——怎么从公开信息中提取政策意图和推进节奏。
供需分析框架（第二部分）是**输出端**的功夫——理解政策意图之后，怎么判断它对资产价格的影响。

两者结合才能形成完整的分析链路：
```
看新闻（谁参会？钱/权袋子？时间差？）
  → 判断政策优先级和落地阶段
    → 评估对产业链供需的实际影响
      → 判断对资产价格的含义
```
