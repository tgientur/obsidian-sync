## 二、本周最重要产业链变化

### 第1条：AI产业链（光模块环节）

**是否有新的边际变量**：是
**是否改变盈利预期**：否（TrendForce维持全年260亿美元预测不变）
**是否改变估值体系**：否（估值框架未变，但资金在板块内重新分配）

**再定价判断**：未重新定价，但**板块内结构正在调整**

**本期状态**：上行周期（加速中后段）

**核心变化**：
1. 光模块连涨12天后出现分化——一线龙头（中际旭创/新易盛）估值高位，资金转向二线（华工科技/光迅科技）CPO技术路线
2. TrendForce数据显示：全球AI光收发模块2026年市场规模预计达260亿美元（2025年165亿美元，+57%），800G放量+1.6T加速
3. AI应用端（AI PC/AI Agent）开始活跃，显示资金从基础设施向应用端蔓延

**龙头变化**：
- 本周龙头：华工科技、光迅科技（CPO概念）
- 是否替代旧龙头：是，资金从一线光模块向二线扩散
- 补涨还是主升：补涨性质（一线估值过高后的溢出效应）

**传导分析**：
资金路径：光模块一线龙头 → 光模块二线/CPO → AI应用端
产业链传导：算力基建（光模块/GPU）→ 模型平台 → AI应用（AI PC/AI Agent）
当前处于**从基础设施向应用端传导的早期阶段**

**与历史对比**：
- 历史类似阶段：2023年7月（光模块第一波上涨后分化）
- 当前所处阶段：**主升浪后半程，板块内结构分化**
- 是否主升浪：是，但已进入后半程

### 第2条：半导体产业链

**是否有新的边际变量**：是
**是否改变盈利预期**：待观察（IPO扩充产能）
**是否改变估值体系**：否

**再定价判断**：正在定价中

**本期状态**：上行周期（温和上行）

**核心变化**：
1. 燧原科技（AI芯片）、粤芯半导体（晶圆代工）IPO过会——国产AI芯片供给端扩容
2. 日经225创历史新高，半导体设备股（爱德万测试+7.67%）领涨，全球半导体景气度延续

**龙头变化**：
- 本周无显著龙头切换

**传导分析**：
IPO扩容信号→国产替代预期维持→但短期不改变现有竞争格局

### 第3条：小金属/稀土产业链（新定价信号）

**是否有新的边际变量**：是
**是否改变盈利预期**：待观察
**是否改变估值体系**：否

**再定价判断**：正在定价中（主题驱动）

**本期状态**：主题炒作期

**核心变化**：

**与历史对比**：

- 当前阶段：**主题驱动，需观察是否转向业绩驱动**
