# 云计算

## 定义与产业定位

云计算（Cloud Computing）是通过互联网按需提供计算资源（算力/存储/网络/软件/平台）的服务模式。核心特征：**按需付费、弹性伸缩、资源池化**。用户不再需要自建IT基础设施，转而向云服务商订阅算力和服务。

对于AI产业，云计算承担着"算力水厂"的角色——AI大模型训练和推理都需要海量算力，企业通过云平台获取GPU算力，避免了自有数据中心的高昂建设和运维成本。北美四大云厂商（Amazon AWS/Microsoft Azure/Google GCP/Meta）2026年资本开支合计超3500亿美元，其中AI基础设施占比>60%（数据来源：[1]）。

AI时代云计算发生了结构性变化：**从CPU通用算力时代进入GPU智能算力时代**。智能算力占总算力比例从2023年的20%（数据来源：[1]）提升至2026年的50%+，云厂商成为最大的AI算力采购者和分配者。

## 产业链位置

```
上游（硬件基础设施） → 中游（云服务） → 下游（客户）
    GPU/AI芯片          IaaS（算力裸金属）     AI训练/推理
    服务器                PaaS（平台服务）      企业数字化转型
    光模块/交换机         MaaS（模型服务）       SaaS应用
    光模块/交换机         SaaS（软件即服务）    互联网/金融/医疗
    数据中心
```

云厂商处于AI产业链的"枢纽"位置——向上锁定GPU产能和CoWoS封装，向下通过云服务分发算力，同时自研大模型（如Google Gemini、Amazon Titan）向下游应用延伸。

## 投资逻辑简析

1. **AI驱动的结构性增量**：云厂商资本开支从2023年的~$1800亿增长至2026年~$3500亿，AI算力占比从30%（数据来源：[1]）升至60%+。这直接拉动光模块/服务器/PCB/液冷等A股AI基建标的的业绩。云厂商Capex是A股AI板块最重要的外部跟踪指标。

2. **MaaS（模型即服务）是下一个增长极**：大模型API调用量每6个月翻倍（OpenAI/通义/千问），推理算力消耗正在超过训练算力。推理算力的毛利率高于训练算力（纯软件服务边际成本近零）。

3. **国内智算中心建设**：2025-2027年各地规划的智算中心总投资超5000亿元，运营商（移动/电信/联通）是主要建设方。利好A股服务器（浪潮/曙光）、光模块、液冷、UPS电源等环节。

4. **混合云+边缘计算**：AI推理部署向边缘侧延伸（AI PC/自动驾驶/工业视觉），边缘云（优刻得/首都在线）获得增量需求。

**风险提示**：云厂商Capex指引不及预期是AI板块的最大风险。国内云厂商盈利能力弱于海外，AI变现还需验证。

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期
