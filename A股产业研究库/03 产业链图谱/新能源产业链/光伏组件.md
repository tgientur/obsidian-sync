# 光伏组件

## 定义与产业定位

光伏组件（Solar Module）是光伏发电系统的核心发电单元，由一定数量的光伏电池片通过串并联方式封装而成。组件环节位于光伏产业链的中游偏末端——上游承接电池片，下游对接逆变器/电站EPC。

## 产业链位置

```
硅料 → 硅片 → 电池片 → 组件 → 逆变器 → 电站
                        ↑
                    光伏组件（本节点）
```

组件环节是光伏产业链中**产能最分散、竞争最激烈、定价权最弱**的环节。全球组件产能中中国占比超过80%，但CR5仅约40-45%，远低于硅料（CR5~70%）和逆变器（CR5~80%）。

## 组件环节的竞争特征

1. **品牌壁垒低**：组件标准化程度高，招标中价格是最核心因素，品牌溢价仅约0.01-0.02元/W
2. **全球渠道能力关键**：海外市场（欧洲/美国/中东/印度）需要建立本地化销售和服务网络，渠道能力决定海外出货量
3. **垂直一体化是趋势**：2023-2025年产能过剩期，一体化企业（通威/隆基）的抗风险能力明显强于纯组件厂商
4. **海外贸易壁垒持续升级**：美国201/301关税、欧洲碳关税、印度ALMM清单等贸易壁垒推动中国组件企业海外建厂（东南亚/中东/美国本土）

## 主要风险

1. **产能过剩风险**：光伏各环节产能严重过剩，价格持续承压
2. **政策退坡风险**：补贴退坡/消纳红线调整
3. **贸易摩擦风险**：欧美对中国光伏/风电产品的贸易壁垒
