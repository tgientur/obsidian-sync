# 资料库模块

## 定位

建立原始资料中心。知识库优先保存资料来源，而非重复改写资料内容。只存来源链接与发布信息，不存原文内容。

## 分类结构

### 14.1 政策文件
国务院、发改委、工信部、地方政府发布的产业相关政策原文链接

### 14.2 监管文件
证监会、交易所、金融监管机构发布的监管规定与指引

### 14.3 行业标准
国家标准、行业标准、团体标准

### 14.4 行业白皮书
第三方机构发布的行业研究报告与白皮书

### 14.5 统计年鉴
国家统计局、行业协会发布的年度统计数据

### 14.6 券商报告
券商研究所发布的产业深度报告（注明来源与日期）

### 14.7 上市公司年报
重点覆盖公司的年报、半年报、季报链接

### 14.8 招股说明书
IPO 招股说明书、再融资说明书

## 每份资料记录格式

```yaml
名称: 
发布时间: 
发布机构: 
原始链接: 
核心内容摘要: 
适用产业: 
关联节点: 
```

## 原则

- 原文链接优先
- 资料保存优先
- 总结内容次之
- 不存储受版权保护的全文内容
