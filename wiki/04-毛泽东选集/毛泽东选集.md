# 毛泽东选集

> 毛泽东在一系列革命与建设时期的重要著作合集，涵盖政治、军事、哲学、经济、文化等领域，核心是**把马克思主义的普遍真理和中国具体实际相结合**——用他的话说叫"实事求是"。

## 核心内容

《毛泽东选集》共七卷，收录了从1925年到1970年代的主要著作。这不是一套"理论书"，更准确地说是一套**"解决问题的方法论实录"**——每一篇文章都是针对一个具体问题写的：党内有错误思想怎么办、仗怎么打、经济怎么搞、矛盾怎么处理。

七卷按时间线划分：

| 卷次 | 时期 | 代表篇目 |
|------|------|----------|
| 第一卷 | 1925—1937（大革命+土地革命） | 中国社会各阶级的分析、湖南农民运动考察报告、星星之火可以燎原、反对本本主义、实践论、矛盾论 |
| 第二卷 | 1937—1940（抗战前期） | 论持久战、战争和战略问题、统一战线中的独立自主 |
| 第三卷 | 1940—1945（抗战后期） | 改造我们的学习、整顿党的作风、反对党八股、在延安文艺座谈会上的讲话、为人民服务、愚公移山 |
| 第四卷 | 1945—1949（解放战争） | 目前形势和我们的任务、将革命进行到底 |
| 第五卷 | 1949—1957（建国初期） | 论人民民主专政、论十大关系、关于正确处理人民内部矛盾的问题 |
| 第六卷 | 1957—1960年代 | 社会主义建设时期文稿 |
| 第七卷 | 1960年代—1970年代 | 文化革命时期文稿 |

## 普通人能学到的实用启示

毛选不太适合当"政治课本"读，最适合当 **"解决问题的方法论"** 读。以下是最实用的几条：

### 1. 实事求是——从事实出发，而不是从概念出发

> "没有调查就没有发言权。"

这是毛选里最核心的方法论。很多人做决策是先有一个结论，然后去找论据证明自己是对的。实事求是的意思是反过来：先搞清楚实际情况什么样，再决定怎么做。

职场上的应用：新到一家公司，先别急着输出方案，花时间摸清业务逻辑、利益关系、历史遗留问题。**调查清楚了再说话，比说了一堆漂亮话但讲不中要害强得多。**

### 2. 分清主次——抓主要矛盾

> "研究任何过程，如果是存在着两个以上矛盾的复杂过程，就要用全力找出它的主要矛盾。捉住了这个主要矛盾，一切问题就迎刃而解。"

一个人每天要处理几十件事，其中真正关键的往往只有一两件。大多数人的问题是：在次要矛盾上花了80%的精力，在主要矛盾上只花了20%。**停下来想一想：我当前最大的瓶颈到底是什么？**

### 3. 矛盾无处不在——别怕冲突，要学"管理冲突"

> "矛盾是普遍的、绝对的，存在于事物发展的一切过程中。"

很多人害怕冲突，觉得有矛盾就是坏事。毛选告诉你：矛盾是必然的，没有矛盾才奇怪。关键不是消除矛盾，是**识别矛盾的性质**——是对抗性的还是非对抗性的，然后采取不同的处理方式。

职场/关系中的应用：和同事有分歧很正常，关键是分清楚这个分歧是"利益冲突"还是"认知差异"，处理方式完全不一样。

### 4. 持久战思维——承认短期困难，相信长期趋势

> "战略上藐视敌人，战术上重视敌人。"

这是极好的压力管理法则。面对一个难题：**在心态上要藐视它**——它不就是个坎吗，跨过去就完了；**但在具体操作上要非常重视**——每一小步都要认真走。很多人正好反过来：战略上把困难想得太大，吓到自己；战术上又掉以轻心。

### 5. 实践出真知——干中学

> "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。"

现代人最大的毛病是想得太多、做得太少。很多事情想是想不明白的，只有上手做了才能理解。**做完了再复盘，比做之前反复论证有效得多。**

### 6. 没有调查就没有发言权——信息才是决策的基础

在今天这个信息爆炸的时代，这句话反而更重要了。大多数人的问题不是信息太少，而是**用二手信息做决策**。听到别人说某个行业不行就信了、看了一篇帖子就觉得懂了。真正有效的信息获取方式永远是：自己去看、去问、去验证。

## 核心方法论体系

| 方法论 | 出处 | 一句话 |
|--------|------|--------|
| 实事求是 | 改造我们的学习 | 一切从实际出发 |
| 调查研究 | 反对本本主义 | 没有调查就没有发言权 |
| 矛盾分析 | 矛盾论 | 抓主要矛盾，分清主次 |
| 实践标准 | 实践论 | 实践是检验真理的唯一标准 |
| 战略藐视+战术重视 | 目前形势和我们的任务 | 心态上举重若轻，操作上如履薄冰 |
| 内因外因 | 矛盾论 | 事物发展的根本原因在内部 |
| 统一战线 | 多篇 | 把朋友搞得多多的，把敌人搞得少少的 |
