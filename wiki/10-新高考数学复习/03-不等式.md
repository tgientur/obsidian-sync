# 不等式

> 难度：★★ | 分值：5~10 分 | 题型：单选/多选/解答题的一部分
> 考查特点：单独出小题较少，更多作为工具镶嵌在函数、导数、数列、解析几何大题中

---

## 一、核心考点

### 1. 一元二次不等式

$a x^2 + b x + c > 0$（或 $< 0$）的解法：先看二次项系数正负 → 求根 → 大于取两边，小于取中间

### 2. 基本不等式（均值不等式）

$$a + b \geq 2\sqrt{ab}\quad (a,b > 0)$$

**推广（三元）**：
$$a + b + c \geq 3\sqrt[3]{abc}\quad (a,b,c > 0)$$

### 3. 不等式的性质

- 同向可加：$a > b,\; c > d \Rightarrow a + c > b + d$
- 同向可乘（正数）：$a > b > 0,\; c > d > 0 \Rightarrow ac > bd$
- 倒数：$a > b > 0 \Rightarrow \frac{1}{a} < \frac{1}{b}$

### 4. 绝对值不等式

$$|a| - |b| \leq |a \pm b| \leq |a| + |b|$$

### 5. 线性规划（考频下降，但仍有）

求 $z = ax + by$ 在线性约束条件下的最值 → 画可行域 → 平移目标函数

## 三、常用技巧

### 技巧 1：配凑法用基本不等式
核心思想：$a + b \geq 2\sqrt{ab}$ 要求 $a$ 和 $b$ 的**乘积为定值**。如果题目不是直接给 $a$ 和 $b$，需要配凑。

**例**：$x + \frac{1}{x}$ 的最小值（$x > 0$）→ 直接用，$= 2$
**例**：$x + \frac{4}{x+1}$ 的最小值 → 先写成 $(x+1) + \frac{4}{x+1} - 1 \geq 2\sqrt{(x+1)\cdot\frac{4}{x+1}} - 1 = 4-1 = 3$

### 技巧 2：1 的代换
当题目条件给出 $a+b=1$（或 $\frac1a + \frac1b = 1$）时，把目标式乘以 1 再展开。

**例**：$a > 0, b > 0, a + b = 1$，求 $\frac1a + \frac4b$ 的最小值
$$\frac1a + \frac4b = (a+b)\left(\frac1a + \frac4b\right) = 1 + \frac{4a}{b} + \frac{b}{a} + 4 = 5 + \frac{4a}{b} + \frac{b}{a} \geq 5 + 2\sqrt{4} = 9$$

### 技巧 3：判别式法
对于 $\frac{ax^2+bx+c}{dx^2+ex+f}$ 型分式函数求值域，令 $y = \frac{ax^2+bx+c}{dx^2+ex+f}$，整理成关于 $x$ 的二次方程，利用 $\Delta \geq 0$ 求 $y$ 的范围。

### 技巧 4：分离参数法解恒成立
含参不等式 $f(x) \geq a$ 恒成立 $\Leftrightarrow a \leq f(x)_{\min}$
含参不等式 $f(x) \leq a$ 恒成立 $\Leftrightarrow a \geq f(x)_{\max}$

### 技巧 5：柯西不等式（选填秒杀）

$$(a^2+b^2)(c^2+d^2) \geq (ac+bd)^2$$

当 $\frac{a}{c} = \frac{b}{d}$ 时取等。

**延伸**（三维）：
$$(a_1^2+a_2^2+a_3^2)(b_1^2+b_2^2+b_3^2) \geq (a_1b_1 + a_2b_2 + a_3b_3)^2$$

## 四、二级结论

1. **糖水不等式**（比较大小利器）：
   若 $a > b > 0, m > 0$，则 $\frac{b}{a} < \frac{b+m}{a+m}$

2. **对勾函数** $y = x + \frac{k}{x}$（$k > 0$）：
   - 在 $(0, \sqrt{k}]$ 上递减，在 $[\sqrt{k}, +\infty)$ 上递增
   - 在 $x = \sqrt{k}$ 处取最小值 $2\sqrt{k}$
   - 可用于快速求形如 $x + \frac{k}{x}$ 的最值

3. **排序不等式**（了解即可）：
   两组数同序和 $\geq$ 乱序和 $\geq$ 反序和

4. **权方和不等式**（选填用）：
   $$\frac{x^2}{a} + \frac{y^2}{b} \geq \frac{(x+y)^2}{a+b} \quad (a,b > 0)$$

## 五、易错清单

- [ ] 基本不等式使用前提：$a,b > 0$，不要忘记检验
- [ ] 基本不等式取等条件：$a = b$ 时取等，验证等号能否成立
- [ ] 含参不等式讨论时，二次项系数可能为 0
- [ ] 解分式不等式时不要直接去分母（不知道分母正负）
- [ ] $\frac{a}{b} > 1$ 不能直接得出 $a > b$（$b$ 可能为负）


> **总评**：不等式作为独立考点分值不大，但作为函数、导数、解析几何的工具无处不在。"1 的代换"和"配凑法"是性价比最高的技巧。
