# 热管理

## 定义与产业定位

热管理（Thermal Management）是指通过散热设计、导热材料和冷却系统，控制电子设备/工业系统的温度在安全范围内。核心矛盾是：芯片功率密度持续提升（单颗GPU功耗从300W升至700W+）与散热能力的赛跑——若温度失控，芯片性能下降、可靠性降低，严重时直接损毁。

AI时代热管理的特殊性：**单机柜热密度从传统数据中心的5-10kW飙升至50-100kW+**（GB200 NVL72单机柜功耗120kW）。传统风冷方案已到极限（风冷最大散热能力约30kW/柜），液冷从"可选项"变成"必选项"。

## 产业链位置

### 双主线分布

热管理横跨两条产业链，且技术路径有差异：

**主线一：AI数据中心散热**

```
导热材料 → 散热模组 → 冷板液冷 → CDU（冷量分配） → 液冷整柜
                                         ↓
                                  冷却塔/干冷器
```

**主线二：新能源汽车热管理**

```
电子水泵/油泵 → 热泵系统 → 电池热管理（BTMS） → 整车热管理集成
```

两条主线共享部分技术（液冷板、导热界面材料、水泵），但产品形态和客户结构完全不同。

## 投资逻辑简析

1. **液冷渗透率加速提升**：2025年国内液冷渗透率突破50%（数据来源：[1]），预计2027年达70-80%。每GW算力对应的液冷市场约15-20亿，2025-2027年液冷市场CAGR>50%。

2. **GPU功耗增长的结构性推动**：NVIDIA从H100（700W）到B200（1000W）到Rubin（1200W+），每代GPU功耗提升40-50%（数据来源：[1]）。液冷从"选配"变"标配"，单GPU散热价值量从$50提升至$150-200。

3. **新能源汽车热管理价值量翻倍**：新能源车热管理单车价值量从传统燃油车的1000-1500元提升至5000-8000元（热泵集成方案）。三花智控/银轮股份长期受益于电动化渗透。

4. **AI+新能源双轮驱动**：部分公司（英维克/三花智控）同时受益于AI数据中心和新能源车两条需求曲线。

**风险提示**：液冷行业竞争加剧（入局者从2022年10家增至2025年50+家），价格下行压力。浸没式液冷技术路线尚未收敛，存在路径选择风险。

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期
