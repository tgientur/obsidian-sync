# 概率统计与计数原理

> 难度：★★★ | 分值：20~25 分 | 题型：单选/多选 + 解答题（第16题或第17题）
> 考查特点：计数原理强调排列组合的技巧性；概率统计大题数据量大，阅读理解要求高，考过分布列、期望、回归分析、独立性检验等

---

## 一、核心考点

### 1. 计数原理

- **分类加法计数原理**：$N = m_1 + m_2 + \cdots + m_n$
- **分步乘法计数原理**：$N = m_1 \times m_2 \times \cdots \times m_n$
- **排列数**：$A_n^m = n(n-1)\cdots(n-m+1) = \frac{n!}{(n-m)!}$
- **组合数**：$C_n^m = \frac{A_n^m}{m!} = \frac{n!}{m!(n-m)!}$
- **二项式定理**：$(a+b)^n = \sum_{k=0}^n C_n^k a^{n-k} b^k$

### 2. 概率

- **古典概型**：$P(A) = \frac{|A|}{|\Omega|}$
- **条件概率**：$P(B|A) = \frac{P(AB)}{P(A)}$
- **全概率公式**：$P(B) = \sum_{i=1}^n P(A_i)P(B|A_i)$
- **贝叶斯公式**：$P(A_i|B) = \frac{P(A_i)P(B|A_i)}{P(B)}$

### 3. 随机变量

- **离散型随机变量的分布列、期望与方差**
  $$E(X) = \sum x_i p_i,\quad D(X) = \sum (x_i - E(X))^2 p_i$$
- **二项分布**：$X \sim B(n,p)$，$P(X=k) = C_n^k p^k (1-p)^{n-k}$
  $$E(X) = np,\quad D(X) = np(1-p)$$
- **超几何分布**：$N$ 个物品，$M$ 个次品，抽取 $n$ 个
  $$P(X=k) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n},\quad E(X) = n\frac{M}{N}$$
- **正态分布**：$N(\mu, \sigma^2)$，$P(\mu-\sigma \leq X \leq \mu+\sigma) \approx 0.6827$
  $$P(\mu-2\sigma \leq X \leq \mu+2\sigma) \approx 0.9545,\quad P(\mu-3\sigma \leq X \leq \mu+3\sigma) \approx 0.9973$$

### 4. 统计

- **抽样方法**：简单随机、系统抽样、分层抽样
- **统计图表**：频率分布直方图、茎叶图、散点图
- **样本数字特征**：平均数、中位数、众数、方差、标准差
- **回归分析**：$\hat{y} = \hat{b}x + \hat{a}$，最小二乘法
- **独立性检验**：$\chi^2 = \frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}$
- **相关系数**：$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}$

## 三、常用技巧

### 技巧 1：排列组合的"定位法"
对于限制条件较多的排列组合，**优先处理特殊元素或特殊位置**：
- 某元素必须在某位置 → 先把这个位置确定
- 相邻问题 → 捆绑法（把相邻元素看成一个整体）
- 不相邻问题 → 插空法（先排其他元素，再在空位中插入）
- 定序问题 → 除以顺序的阶乘

### 技巧 2：二项式定理的"通项法"
只用记通项公式：
$$T_{k+1} = C_n^k a^{n-k} b^k$$
求常数项 → 令 $x$ 的指数为 0
求有理项 → 令 $x$ 的指数为整数

### 技巧 3：二项分布与超几何分布的快速识别
- **二项分布**："有放回"抽取 / "独立重复"试验 / 概率 $p$ 不变
- **超几何分布**："无放回"抽取 / 不放回抽样
- 当总体 $N$ 很大而样本 $n$ 很小时（$n/N \leq 0.1$），超几何分布近似于二项分布

### 技巧 4：分布列的"枚举验证法"
列出分布列后，**验证概率之和是否为 1**。这一步能发现 90% 的分布列计算错误。

### 技巧 5：全概率公式的"分拆法"
典型场景：多步试验 / 多道工序 / 多种原因
1. 找到"原因"（划分样本空间）
2. 对每个原因，求在此原因下事件发生的概率
3. 加权求和

## 四、二级结论

1. **组合恒等式**：
   $$C_n^m = C_n^{n-m},\quad C_n^m + C_n^{m-1} = C_{n+1}^m$$
   $$C_n^0 + C_n^1 + \cdots + C_n^n = 2^n$$
   $$C_n^0 - C_n^1 + C_n^2 - \cdots + (-1)^n C_n^n = 0$$

2. **二项展开式系数和**：
   - 所有系数和 = 令 $a = b = 1$
   - 奇数项系数和 = 偶数项系数和 = $2^{n-1}$
   - 系数绝对值之和 = 令 $a = 1, b = -1$

3. **方差的计算公式**：
   $$D(X) = E(X^2) - (E(X))^2$$（有时比定义式更快）

4. **独立事件的判定与性质**：
   $A$ 与 $B$ 独立 $\Leftrightarrow P(AB) = P(A)P(B)$
   若 $A$ 与 $B$ 独立，则 $A$ 与 $\bar{B}$、$\bar{A}$ 与 $B$、$\bar{A}$ 与 $\bar{B}$ 也独立

5. **相关系数 $r$ 的意义**：
   - $|r|$ 越接近 1，线性相关越强
   - $r > 0$ 正相关，$r < 0$ 负相关
   - $r = 0$ 不相关（但可能非线性相关）

6. **$3\sigma$ 原则的快速估算**：
   正态分布中，数据在 $(\mu \pm \sigma)$ 内约 68%，在 $(\mu \pm 2\sigma)$ 内约 95%，在 $(\mu \pm 3\sigma)$ 内约 99.7%

## 五、易错清单

- [ ] 排列用 $A$，组合用 $C$，不要混淆
- [ ] 区分"有序"和"无序"（排列 vs 组合）
- [ ] 概率值在 $[0,1]$ 之间，算出来负数或大于 1 一定错了
- [ ] 二项分布与超几何分布的条件：有无放回
- [ ] $\chi^2$ 独立性检验：$\chi^2$ 值越大，越能拒绝原假设（即有关联）
- [ ] 回归分析中，回归方程过样本中心点 $(\bar{x}, \bar{y})$
- [ ] 频率分布直方图求平均数：频率 $\times$ 组中值，**不是**直接用组中值


> **总评**：概率统计的难度在于**阅读理解**和**分布列计算**。排列组合掌握"捆绑法"和"插空法"基本够用。解答题的答题模板比较固定，注意把过程写清楚，尤其是随机变量的取值和对应概率的计算过程。
