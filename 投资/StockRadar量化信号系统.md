# StockRadar 量化信号系统

> 创建：2026-05-09 | 状态：已上线运行

---

## 一、项目缘起

基于 [[TrendRadar 部署与迁移|TrendRadar]] 的二次开发经验，自建个人量化信号系统。核心理念：

- **AI 做研究员，不做交易员。** AI 辅助信息处理、代码开发、策略复盘。
- **交易由程序执行。** 确定性规则引擎，无情执行，不带情绪。
- **个人量化的优势是灵活。** 小体量可验证短周期、高换手策略，这是幻方等大私募做不到的。

## 二、架构

```
config.yaml         策略参数配置
    │
    ▼
fetch.py            数据层 —— AKShare 取 K 线 → pandas DataFrame
    │
    ▼
indicators.py       指标层 —— MA / EMA / RSI / MACD / ATR / 量比（纯 pandas）
    │
    ▼
signals.py          信号引擎 —— check_xxx() 条件判断，确定性规则
    │
    ▼
main.py             入口层 —— 读取配置 → 取数据 → 跑信号 → 打印 + 存 CSV
scan_all.py         全市场扫描 —— 并行下载 5460 只 A 股，信号筛选
```

**零外部量化框架依赖。** 每个文件独立可用，加新策略只需在 `indicators.py` 加一个函数、`signals.py` 加一个 `check_xxx()`。

## 三、内置策略

| 策略 | 逻辑 | 参数 |
|------|------|------|
| MA20 突破 | 近 3 日曾低于 MA20，今日收盘站上 | `ma_period: 20`, `ma_lookback: 3` |
| 成交量放量 | 今日量 >= 20 日均量 × 1.5 | `vol_ratio_threshold: 1.5` |
| RSI 超卖反弹 | RSI < 30 后拐头向上 | `threshold: 30` |
| 组合信号 | MA 突破 + 放量同时触发 | 高置信度 |

## 五、用法

```bash
cd /d/stockradar

# 监控自选池（config.yaml 里的 stocks）
PYTHONIOENCODING=utf-8 python main.py

# 全市场扫描（首次全量下载，后续增量）
PYTHONIOENCODING=utf-8 python scan_all.py --top 30

# 第二天起仅更新今日数据
PYTHONIOENCODING=utf-8 python scan_all.py --update

# 循环监控（盘中 N 秒扫一次）
PYTHONIOENCODING=utf-8 python main.py --loop 600
```

### 工程层
