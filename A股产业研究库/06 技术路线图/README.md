## 技术演进规律总结

### Gartner Hype Cycle在A股的应用

| 阶段 | 市场特征 | 投资策略 | 典型阶段案例 |
|------|---------|---------|-------------|
| 技术触发期 | 学术/实验室突破，资本零星关注 | 风险投资逻辑，不适用二级 | 量子计算、核聚变、室温超导 |
| 期望膨胀期 | 媒体热炒，大量小市值公司蹭概念，主题投资活跃 | 游资炒作，快进快出，严格止损 | 2021元宇宙、2023人形机器人初期 |
| 泡沫破裂期 | 技术落地不及预期，股价大幅回调，行业洗牌 | 观望，等待真正龙头浮现 | 2022自动驾驶、2019区块链 |
| 爬坡恢复期 | 技术逐步成熟，商业模式验证，订单落地 | 找有收入的标的，耐心持有 | 2023-25 AI算力、2020-22 光伏N型 |
| 生产成熟期 | 渗透率>30%，行业格局稳定，龙头清晰 | PE估值，赚EPS增长的钱 | 当前光伏组件、动力电池 |

应用原则：**在期望膨胀期不追高，在泡沫破裂期不恐慌，在爬坡恢复期重仓有壁垒的企业**

### 各产业链技术迭代节奏对比

| 产业链 | 迭代周期 | 驱动因素 | 近年关键跃迁 | 对投资的意义 |
|--------|---------|---------|-------------|-------------|
| [[A股产业研究库/03 产业链图谱/AI产业链/GPU|GPU]] | 2年一代 | 摩尔定律+AI需求暴增 | Hopper→Blackwell→Rubin | 每代带来数倍性能提升，龙头确定性高 |
| [[光伏技术|光伏]] | 1-2年一代 | 转换效率竞争+降本 | P型→TOPCon→HJT→BC | 技术迭代快，押错路线损失大 |
| [[A股产业研究库/03 产业链图谱/新能源汽车产业链/动力电池|动力电池]] | 3-5年一代 | 能量密度+安全性 | 三元→铁锂→固态 | 研发周期长，产能建设慢 |
| [[半导体制造|半导体制程]] | 2-3年一代 | 物理极限逼近+良率爬坡 | 7nm→5nm→3nm→2nm | 龙头壁垒极高，追赶者成本巨大 |
| [[A股产业研究库/03 产业链图谱/医药生物产业链/创新药|创新药]] | 5-10年 | 临床试验+监管审批 | 小分子→大分子→细胞/基因治疗 | 高投入高风险，专利悬崖残酷 |
| [[储能技术|储能]] | 3-5年 | 度电成本+循环寿命 | 抽蓄→锂电→液流→压缩空气 | 长时储能是中期方向 |
| [[智驾技术|智能驾驶]] | 1-2年 | 算法+算力+数据闭环 | L2+→L3→L4→端到端 | 软件定义硬件，OTA持续升级 |

**规律总结**：硬件迭代快的行业（GPU/光伏）龙头更替风险大但增量空间大；迭代慢的行业（医药/电池）先发优势持续更久但需要耐心。

## 交叉技术图谱

### 半导体 → AI → 汽车（跨链渗透）

```
半导体制造工艺提升 → AI芯片算力跃迁 → 自动驾驶芯片上车 → 智能汽车体验升级
      ↓                                                           ↓
  先进制程（7nm以下）                                         L3+功能量产
  Chiplet封装                                                   座舱SoC集成
  HBM/CXL存储                                                  OTA持续迭代
```

### 新能源 → AI → 机器人（跨链复用）

```
新能源产业（电池/热管理/电机）→ 技术溢出 → 人形机器人（动力/散热/执行器）
AI产业（大模型/视觉/SLAM）    → 算法移植 → 机器人（感知/决策/控制）
```
