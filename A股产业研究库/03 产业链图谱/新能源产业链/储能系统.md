# 储能系统

## 定义与产业定位

储能系统（Energy Storage System, ESS）是将储能电池、变流器、电池管理系统、能量管理系统、温控消防等子系统集成为一个完整功能的电力存储和释放单元。储能系统位于储能产业链的中游——上游连接储能电芯/PCS等设备，下游对接电源侧/电网侧/用户侧应用场景。

## 产业链位置

```
电池材料 → 储能电芯 → 储能系统集成 → 终端应用（电源/电网/用户）
                       ↑
                    储能系统（本节点）
                  BMS | PCS | EMS | 温控 | 消防
```

## 系统构成

| 子系统 | 功能 | 成本占比 | 代表性供应商 |
|:------|:----|:-------:|:-----------|
| 电池簇 | 电化学储能核心，存储/释放电能 | 55-65% | 宁德时代/亿纬锂能/比亚迪 |
| PCS（变流器） | DC/AC双向变换，并网控制 | 10-15% | 阳光电源/上能电气/盛弘股份 |
| BMS（电池管理系统） | 电池状态监测/SOC估算/均衡管理 | 3-5% | 高特电子/协能科技 |
| EMS（能量管理系统） | 充放电策略/收益优化/电网调度响应 | 3-5% | 长园深瑞/国电南瑞 |
| 温控系统 | 液冷/风冷散热，保障电池温度一致性 | 3-5% | 英维克/同飞股份 |
| 消防系统 | 热失控探测/气体灭火/防爆 | 2-3% | 国安达/青鸟消防 |
| 集装箱/结构件 | 户外防护/运输安装 | 5-8% | 标准化供应商 |
数据来源：[1][2] 相关年报/行业报告

## 主要风险

1. **产能过剩风险**：光伏各环节产能严重过剩，价格持续承压
2. **政策退坡风险**：补贴退坡/消纳红线调整
3. **贸易摩擦风险**：欧美对中国光伏/风电产品的贸易壁垒
