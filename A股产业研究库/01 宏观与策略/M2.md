## 定义

M2（广义货币供应量）是指流通中的现金（M0）+ 活期存款（M1）+ 定期存款 + 储蓄存款 + 其他存款。它衡量全社会货币供应总量，反映货币政策的松紧程度。

货币层次：**M0 < M1 < M2**
- M0 = 流通中现金
- M1 = M0 + 企业活期存款（狭义货币，反映即期购买力）
- M2 = M1 + 定期存款 + 储蓄存款 + 其他存款（广义货币，反映潜在购买力）

## 数据来源与发布

- **发布机构**：中国人民银行
- **发布时间**：每月10-15日（与社融同步发布）
- **关键对比**：M2增速 vs 社融增速，构成"社融-M2增速差"，衡量资金供需紧张程度

## A股投资意义

M2增速与A股走势存在**正相关关系**：

- **M2增速回升** → 货币宽松 → 市场流动性改善 → A股估值中枢上移
- **M2增速回落** → 货币收紧 → 估值承压

**M1-M2剪刀差**是更精细的信号：
- **M1增速 > M2增速**（剪刀差走阔）：企业和居民活期存款增加，资金由储蓄转向投资和消费，通常预示A股走强（历史上2014-2015、2020年牛市均伴随剪刀差走阔）
- **M1增速 < M2增速**（剪刀差收窄）：资金回流定期存款，投资意愿不足，A股承压
