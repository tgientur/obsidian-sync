# 开放银行

**英文全称**：Open Banking
**首次提出**：2015年 / 英国竞争与市场管理局推动 / PSD2 欧盟支付服务指令
**所属产业**：金融科技 > 数字银行 > 数据共享

## 定义
银行通过标准化的 API 接口，在用户授权下将账户、交易、支付等数据安全地开放给第三方金融服务商（FinTech），实现数据共享和业务协作。

## 核心作用
将银行从"封闭的钱袋子"变成"金融服务平台"——用户能用银行 APP 管理不同银行账户，也可以用支付宝/微信直接看自家银行卡余额。开放程度直接决定了金融数字化生态的丰富度。

## 应用场景
- 个人账户聚合管理（YNAB、支付宝银行卡助手）
- 第三方支付和资金发起（开放 API 直连银行）
- 基于账户数据的智能风控与信贷评估
- 企业财务自动化（银行账户自助对接 ERP）
