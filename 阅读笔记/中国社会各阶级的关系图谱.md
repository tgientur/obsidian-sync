## 三、小资产阶级内部分化图

原文最细致的观察就在这里——小资产阶级不是铁板一块，而是分为三个经济层次，对应三种政治态度。

```mermaid
flowchart LR
    subgraph small["小资产阶级"]
        direction TB
        upper["上层（右翼）\n约10%\n有余钱剩米\n态度：怀疑革命"]
        middle["中层（中间派）\n约50%\n经济自给\n态度：中立，看风向"]
        lower["下层（左翼）\n约40%\n生活下降\n态度：积极革命"]
    end
    
    upper -->|"怕乱了自己的好日子"| swing_att["对革命怀疑"]
    middle -->|"受社会环境波动大"| wait["对革命中立"]
    lower -->|"失去了往昔的体面"| active["对革命积极"]
    
    active -->|"革命高潮时→"| join["一定参加革命"]
    wait -->|"革命高潮时→"| join2["可以参加革命"]
    swing_att -->|"革命高潮时→"| join3["只得附和革命"]
```

## 四、中产阶级两面性动态模型

中产阶级（民族资产阶级）的态度不是静态的，而是随革命局势变化的函数：

```mermaid
stateDiagram-v2
    state "受帝国主义打击" as hit
    state "感觉痛苦" as pain
    state "赞成革命" as support
    state "无产阶级力量壮大" as power
    state "感到威胁" as threat
    state "怀疑革命" as doubt
    state "右翼跑向敌人" as betray
    state "左翼留在革命中" as stay
    
    hit --> pain
    pain --> support
    support --> power
    power --> threat
    threat --> doubt
    doubt --> betray
    doubt --> stay
    
    note right of support : "反帝反军阀"
    note right of doubt : "怕伤及自身利益"
```

## 六、核心逻辑链

```mermaid
flowchart TD
    A["生产资料占有状况\n（经济地位）"] --> B["在生产力中的位置"]
    B --> C["经济利益受谁支配"]
    C --> D["是否希望改变现状"]
    D --> E{"对革命的态度"}
    
    E --> F1["反对革命\n→ 地主·买办"]
    E --> F2["摇摆不定\n→ 中产阶级"]
    E --> F3["中立观望\n→ 小资上层·中层"]
    E --> F4["积极参加\n→ 小资下层·半无产阶级"]
    E --> F5["彻底革命\n→ 工业无产阶级"]
```

**底层逻辑一句话：** 一个人在当前秩序中是受益者还是受害者，决定了他是否想推翻这个秩序。
