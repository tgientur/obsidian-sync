# 投资框架

投资方法论、交易策略、风控体系。

## 行业选择框架

|:--:|:------:|:------:|
| 高估值 | 警惕泡沫 | 坚决回避 |

## 个股选择三因子
回报率 = 景气度 x 竞争力 x 估值匹配度

## 产业链投资法则

### 先上游后下游
技术突破 → 上游设备/材料最先受益 → 中游制造/组装受益 → 下游应用/内容受益 → 基础设施再次受益

### 投资节奏
预期阶段（概念上涨，无业绩）→ 早期验证（订单/招标出现）→ 业绩兑现（收入利润高增）→ 产能过剩（价格战）→ 格局出清（龙头胜出）

## 风控体系
- 硬止损：买入价-15%，无条件执行
- 逻辑止损：买入逻辑证伪，不管盈亏都出
- 时间止损：持有3个月逻辑未兑现，减半

## 产业链投资方法论详解

### 渗透率投资法（从0到1 vs 从1到N）

渗透率是产业链投资最底层的驱动逻辑，不同阶段的投资范式截然不同。

**从0到1（渗透率0%-10%）**
- 特征是技术路线未定、产品价格高、规模极小
- 驱动力来自技术进步和政策催化，而非市场自发需求
- 投资逻辑偏主题，核心指标看技术突破、产品迭代速度、首批标杆客户
- 股价弹性极大但波动剧烈，跌80%和涨10倍都可能发生
- 策略：看不懂技术选龙头，看得懂技术选弹性最大的环节
- 典型：2020年的[[A股产业研究库/03 产业链图谱/新能源产业链/LFP正极|LFP正极]]放量、2023年的[[A股产业研究库/03 产业链图谱/AI产业链/AI服务器|AI服务器]]、2025年的[[A股产业研究库/03 产业链图谱/机器人产业链/总纲|人形机器人]]

**从1到N（渗透率10%-50%）**
- 特征是技术路线收敛、成本快速下降、需求开始自发爆发
- 驱动力转为规模效应和成本曲线，龙头逐渐跑出
- 核心指标看产能利用率、毛利率稳定性、市场份额变化
- 板块整体上涨最流畅的阶段，但分化也在加大
- 策略：买龙头、买成本最优者、买供应链中最紧缺的环节
- 典型：2021年-2023年的[[A股产业研究库/03 产业链图谱/新能源产业链/总纲|新能源]]（渗透率从15%到40%）

**渗透率50%+**
- 增长放缓，投资逻辑从成长转为周期或价值
- 关注产能出清、格局优化、出海寻找第二曲线
- 只有真龙头和周期底部反转的机会
- 典型：2024年后的[[A股产业研究库/03 产业链图谱/新能源产业链/动力电池|动力电池]]

### 国产替代投资法（替代空间×替代速度×利润弹性）

国产替代=替代空间 × 替代速度 × 利润弹性。三个因子缺一不可。

**替代空间**
- 总市场池的规模（千亿/百亿/十亿级决定了天花板）
- 国产当前市占率（越低空间越大，10%以下最佳）
- 技术壁垒（突破后是否容易守住）

**替代速度**
- 下游客户替换意愿（出于供应链安全、成本、性能哪个驱动）
- 国产产品验证周期（消费电子3-6个月、车规12-18个月、工控18-24个月、军工3-5年）
- 产能瓶颈在哪个环节

**利润弹性**
- 进口替代后毛利率的提升幅度（从25%做到60%还是从50%做到55%）
- 规模效应带来的净利率改善
- 估值重估的可能性（从小市值到龙头溢价）

**当前关注方向**
- [[A股产业研究库/03 产业链图谱/半导体产业链/半导体设备|半导体设备]]（整体国产率15%，AI拉动扩产）
- [[A股产业研究库/03 产业链图谱/半导体产业链/半导体材料|半导体材料]]（国产率更低，验证周期正在缩短）
- [[A股产业研究库/03 产业链图谱/高端制造产业链/工业软件|工业软件]]（国产率<10%，政策强力驱动）
- 高端医疗器械（CT/MRI国产化率刚起步）

### 出海投资法（全球竞争力×海外市场空间×贸易壁垒）

出海投资三因子：全球竞争力决定了能否赚到钱，海外市场空间决定了天花板，贸易壁垒决定了持续性和路径。

**全球竞争力判断**
- 技术代差：领先海外对手多少（1-2年还是5-10年）
- 成本优势：全产业链成本比对手低多少
- 产品力：海外客户愿意支付溢价吗

**海外市场空间**
- 目标区域的总市场规模
- 渗透率提升逻辑是否存在（从0到1还是从1到N）
- 竞争对手的份额和反应

**贸易壁垒**
- 关税/反倾销：短期冲击，能力强的企业可通过建厂消化
- 技术封锁：如AI芯片出口管制，反而催生非美市场
- 认证壁垒：不算贸易壁垒但时间成本极高

**当前关注方向**
- [[A股产业研究库/03 产业链图谱/新能源产业链/锂电出海|锂电出海]]（欧洲+东南亚建厂，利润率不降反升）
- 工程机械（海外收入占比40%+，持续提升中）
- 家电出海（品牌出海，从代工到自主品牌）
- 游戏出海（全球竞争力明确，不受贸易壁垒影响）

### 周期反转投资法（产能出清信号识别）

周期反转的核心是判断产能何时出清、价格何时见底。

**出清前兆信号**
- 行业龙头开始亏现金（折旧+利息覆盖不了）
- 中小企业大面积停产/破产
- 库存周期从主动去库到被动去库的拐点
- 行业内并购整合案例增多
- 政策出手干预（供给侧改革）

**反转确认信号**
- 产品价格企稳后连续数月上行
- 龙头企业产能利用率回到80%+
- 龙头企业毛利率环比改善
- 下游补库存行为开始

**经典案例**
- 存储芯片（2019年、2023年底两次周期底部的反转）
- 面板（2021年见顶→2022年出清→2023年反转）
- 光伏（2024年全行业亏现金，产能出清正在进行中）

**策略要点**
- 左侧布局只能在龙头身上做（中小企业可能出局）
- 最安全的是右侧确认信号出现后的第一波上涨
- 周期股的反转涨幅通常在6个月内完成大部分

### 政策驱动投资法（政策周期四阶段）

政策驱动的收益集中在特定阶段，错过就变成接盘。

**第一阶段：文件/讲话**
- 高层表态、中央文件首次提及新方向
- 板块普涨，涨幅靠情绪，龙头弹性最大
- 持续性取决于后续落地节奏

**第二阶段：细则/规划**
- 具体目标数字出台（装机量/渗透率/投资额）
- 细分方向开始分化，真正受益的环节走出来
- 机构资金开始按计算器

**第三阶段：补贴/招标**
- 财政补贴到位或大规模招标启动
- 业绩开始兑现，EPS驱动取代估值驱动
- 行情最长也最确定的一段

**第四阶段：兑现/炒作**
- 业绩兑现完毕，利好出尽
- 或泡沫化炒作后破灭
- 如果没有第四阶段的泡沫，往往意味着行业不够大

**应用框架**
- 信创/数据要素：当前处于第二阶段向第三阶段过渡
- [[A股产业研究库/03 产业链图谱/AI产业链/总纲|AI]]：政策持续加码，全球共振，处于一、二、三阶段并行
- 低空经济：第一阶段（顶层文件）刚出，弹性最大但风险最高
- 车路云：第二阶段到第三阶段，需要跟踪招标订单

## 估值体系

### 各产业链适合的估值方法

不同产业链的生命周期、盈利模式和资本结构不同，适用的估值体系也不同。

| 产业链 | 核心估值方法 | 辅助指标 | 为什么 |
|:-------|:------------|:--------|:------|
| AI（大模型/算力） | PS、研发折现 | 资本开支增速、用户数/API调用量 | 早期收入增长快但大量投入研发，PE失真 |
| AI（应用层） | PS、远期PE | 用户留存、ARPU、毛利结构 | 还未到规模化赚钱阶段 |
| 半导体（设计） | PE、PEG | 产品单价/毛利率趋势 | 已进入稳定盈利，看成长持续性 |
| 半导体（设备/材料） | PEG、远期PS | 订单增速、国产化率 | 国产替代加速期，盈利释放有滞后 |
| 新能源（锂电/光伏） | PE、PB | 产能利用率、价格走势 | 成熟期成长退潮，周期属性增强 |
| 新能源（储能） | PEG | 招标增速、毛利率趋势 | 高速增长期，需验证盈利质量 |
| 银行 | PB-ROE | 不良率、息差、分红率 | 高杠杆经营，PB比PE更能反映风险 |
| 消费 | PE-G | 渠道扩张、复购率 | 现金流好，PE-G是最实质的定价框架 |
| 医药（创新药） | DCF/管线折现 | 临床进度、NDA节奏 | 无利润可用PE，管线价值决定一切 |
| 机器人 | PS、远期PEG | 订单规模、客户验证进展 | 产业化早期，收入增长先于利润 |

### A股估值体系特征

A股估值体系与美股有本质差异，理解这些特征是正确使用估值的前提。

**流动性溢价**
- A股散户占比高，换手率远超美股，流动性折价小
- 流动性好的大票估值通常高于同类小票（与港股相反）
- 北向资金进出会显著影响核心资产估值中枢

**主题溢价**
- A股对新产业主题的估值容忍度远高于成熟市场
- 只要故事够大、方向够确定，市场愿意给3-5年的远期估值
- 但主题溢价消退也很快，一旦证伪估值直接腰斩

**小盘溢价**
- A股长期存在小盘股相对大盘的估值溢价（与利率环境负相关）
- 壳价值+并购重组预期是底层原因
- 2024年以来小盘溢价在收敛，但未消失
- 注册制常态化后小盘股分化加剧，真成长和纯垃圾价差拉大

### 估值与景气度的匹配框架

估值本身是中性指标，必须放在景气度的背景下解读。

| 景气方向 | 估值位置 | 操作 |
|:--------|:--------|:----|
| 上行初期 | 低/中 | 核心加仓，享受戴维斯双击 |
| 上行中段 | 中/高 | 持有为主，关注超预期空间 |
| 上行末期 | 极高 | 减仓不追，警惕估值顶 |
| 下行初期 | 中/高 | 坚决清仓，不要以为跌了就是机会 |
| 下行中段 | 低/中 | 等待出清信号，不抄底 |
| 下行末期 | 极低 | 左侧布局，需要极强的买入逻辑 |

**核心原则**
- 低景气+低估值不是买入理由，必须伴随景气拐点信号
- 高景气+高估值也可能是买入时机（只要景气持续的时间长于市场预期）
- 真正的风险不是高估值，而是景气度见顶
- 估值是艺术不是科学，不要迷信精确数字

## 交易策略

### 产业链龙头策略（先选赛道再选龙头）

核心逻辑：产业链投资的超额收益来自赛道选择，而非个股挖掘。看好一个方向，先选对赛道，再在赛道内选龙头。

**选赛道**
- 渗透率处于从1到N阶段的最佳弹性期
- 产业链中供需最紧张的瓶颈环节
- 政策支持最强、确定性最高的方向

**选龙头**
- 成本最优者（规模化后毛利率高于同行5%+）
- 绑定最强客户者（打入顶级供应链）
- 技术壁垒最高者（对手3年追不上）

**退出信号**
- 渗透率突破40%-50%（成长退潮）
- 行业出现价格战（毛利承压）
- 新进入者暴增（供给恶化）

---

### 景气度拐点策略（右侧确认后加仓）

不做左侧博弈，等待景气度拐点确认后右侧加仓。

**拐点确认三重信号**
1. 高频数据：月度的出货量/装机量/招标量环比转正且加速
2. 产业链验证：上下游调研确认需求回暖，而不是补库
3. 业绩信号：龙头公司季报超预期或上调指引

**买入时机**
- 第一个信号出现时开始关注，建立观察仓位（3%-5%）
- 第二个信号出现时加仓到中性仓位（10%-15%）
- 第三个信号出现时加仓到核心仓位（20%-30%）
- 三个信号全部兑现后不再追高

**止损条件**
- 下一个季报低于预期（拐点可能失败）
- 高频数据重新转负（拐点证伪）
- 产业链价格异常走弱

### 事件催化策略（管制升级/政策出台/产品发布）

事件驱动交易的收益来自预期差和催化剂强度，而非事件本身。

**三大类催化事件**

**管制升级**（美对华出口管制、制裁名单）
- 受益方：国产替代相关的设备和材料
- 时间窗口：消息出来后1-3天弹性最大
- 风险：利好出尽型下跌，A股对此类事件已审美疲劳
- 操作：消息出来第一时间可以追，但2天后必须处理

**政策出台**（产业规划、补贴细则、税收优惠）
- 受益方：政策明确支持的细分方向
- 时间窗口：文件出台前潜伏收益最大，出台后视超预期程度决定去留
- 关键：评估超预期程度（低于/符合/超预期）
- 操作：如果政策内容已充分预期，出消息当天就是卖点

**产品发布**（重大新品/SOTA突破/客户导入）
- 受益方：发布方及核心供应商
- 时间窗口：发布前1-2周开始有alpha，发布当天最容易高开低走
- 关键：区分里程碑事件和真正改变行业格局的事件
- 操作：越是高关注度的发布会，越不能追（华为发布会、苹果发布会、英伟达GTC）

**事件催化交易框架**
1. 提前预判：建立事件时间表（产品发布日、财报日、政策节点）
2. 预期管理：判断市场对该事件的预期是低、中还是高
3. 仓位分配：预判持仓（预期低可埋伏）vs 即时操作（超预期追入）
4. 退出纪律：事件兑现后3-5天内必须评估去留
