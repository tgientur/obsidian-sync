# 数列

> 难度：★★★ | 分值：10~15 分 | 题型：单选/多选 + 解答题（2025年考了解答题第15题）
> 考查特点：等差等比的定义和计算是基本功，近年新高考偏爱"新定义数列"与数列综合创新题

---

## 一、核心考点

### 1. 等差数列

- **定义**：$a_{n+1} - a_n = d$（常数）
- **通项公式**：$a_n = a_1 + (n-1)d$
- **前 n 项和**：$S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$
- **中项性质**：$a_m + a_n = a_p + a_q$（$m+n = p+q$）

### 2. 等比数列

- **定义**：$\frac{a_{n+1}}{a_n} = q$（常数，$q \neq 0$）
- **通项公式**：$a_n = a_1 q^{n-1}$
- **前 n 项和**：
  $$S_n = \begin{cases} na_1 & q = 1 \\ \frac{a_1(1-q^n)}{1-q} & q \neq 1 \end{cases}$$
- **中项性质**：$a_m a_n = a_p a_q$（$m+n = p+q$）

### 3. 数列求通项

- **累加法**：$a_{n+1} - a_n = f(n)$
- **累乘法**：$\frac{a_{n+1}}{a_n} = f(n)$
- **构造法**：$a_{n+1} = pa_n + q$ → 构造 $a_{n+1} + \lambda = p(a_n + \lambda)$
- **取倒数法**：$a_{n+1} = \frac{a_n}{ka_n + b}$
- **取对数法**：$a_{n+1} = p a_n^q$

### 4. 数列求和

- **公式法**：等差等比直接用公式
- **裂项相消**：$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
- **错位相减**：等差 × 等比型数列
- **分组求和**：交替符号或分段数列
- **倒序相加**：等差数列求和的推导思想

## 三、常用技巧

### 技巧 1：$S_n$ 与 $a_n$ 的互化
$$
a_n = \begin{cases} S_1 & n=1 \\ S_n - S_{n-1} & n \geq 2 \end{cases}
$$
**关键**：用 $S_n$ 求 $a_n$ 时，**一定要分 $n=1$ 和 $n \geq 2$ 讨论**。太多人在这里丢分。

### 技巧 2：构造法的标准化流程
对于 $a_{n+1} = pa_n + q$（$p \neq 1$）：
1. 设 $a_{n+1} + \lambda = p(a_n + \lambda)$
2. 展开得 $a_{n+1} = pa_n + (p-1)\lambda$
3. 令 $(p-1)\lambda = q$ → $\lambda = \frac{q}{p-1}$
4. 则 $\{a_n + \lambda\}$ 是等比数列

**推广**：
- $a_{n+1} = pa_n + qn + r$ → 构造 $a_{n+1} + \lambda(n+1) + \mu = p(a_n + \lambda n + \mu)$
- $a_{n+1} = pa_n + q^n$ → 两边同除以 $p^{n+1}$ 或 $q^{n+1}$

### 技巧 3：裂项相消的常见模式
- $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
- $\frac{1}{n(n+k)} = \frac{1}{k}(\frac{1}{n} - \frac{1}{n+k})$
- $\frac{1}{\sqrt{n+1} + \sqrt{n}} = \sqrt{n+1} - \sqrt{n}$
- $\frac{1}{n(n+1)(n+2)} = \frac12\left[\frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)}\right]$
- $\frac{2n+1}{n^2(n+1)^2} = \frac{1}{n^2} - \frac{1}{(n+1)^2}$

### 技巧 4：错位相减的速算模板
$S_n = (An+B) \cdot q^n - B$（$q$ 为公比）可以直接写出结果。

对于 $S_n = \sum (an+b)q^{n-1}$：
1. 设 $S_n = (A n + B)q^{n} - B$
2. 代入 $n=1$ 和 $n=2$ 列方程组解 $A, B$

**例**：$a_n = (2n-1) \cdot 3^{n-1}$，求 $S_n$
设 $S_n = (A n + B) \cdot 3^n - B$
$S_1 = a_1 = 1$，$S_2 = a_1 + a_2 = 1 + 9 = 10$
$\begin{cases} (A+B) \cdot 3 - B = 1 \\ (2A+B) \cdot 9 - B = 10 \end{cases}$ → 解出 $A=1, B=-1$
$S_n = (n-1) \cdot 3^n + 1$

### 技巧 5：数列放缩（证明题用）
常用放缩方向：
- $\frac{1}{n^2} < \frac{1}{n(n-1)} = \frac{1}{n-1} - \frac{1}{n}$（$n \geq 2$）
- $\frac{1}{n^2} > \frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
- $2(\sqrt{n+1} - \sqrt{n}) < \frac{1}{\sqrt{n}} < 2(\sqrt{n} - \sqrt{n-1})$

## 四、二级结论

1. **等差/比数列的判定**：
   - $\{a_n\}$ 等差 $\Leftrightarrow S_n = An^2 + Bn$（无常数项）
   - $\{a_n\}$ 等比 $\Leftrightarrow S_n = A(q^n - 1)$ 且 $q \neq 1$

2. **下标和性质**：
   - 等差：$m+n = p+q \Rightarrow a_m + a_n = a_p + a_q$
   - 等比：$m+n = p+q \Rightarrow a_m a_n = a_p a_q$

3. **等差中项的推广**：
   若 $a, b, c$ 成等差 → $2b = a + c$，且 $b$ 是等差中项

4. **$S_n$ 与 $a_n$ 的奇偶性质**（常用在综合题）：
   等差数列中：$S_{2n-1} = (2n-1)a_n$（即前奇数项和 = 项数 × 中间项）

5. **分数型递推的常见形式**：
   $a_{n+1} = \frac{a_n}{k a_n + b}$ → 取倒数 → $\frac{1}{a_{n+1}} = \frac{k}{b} + \frac{1}{b} \cdot \frac{1}{a_n}$

## 五、易错清单

- [ ] $S_n$ 求 $a_n$ 时，忘了分 $n=1$ 单独验证
- [ ] 等比数列求和不讨论 $q=1$ 的情况
- [ ] 错位相减的减法运算容易算错符号
- [ ] 构造法解出的 $\lambda$ 忘记回代验证
- [ ] 数列公式中的下标"$n-1$"忘记定义域 $n \geq 2$
- [ ] 放缩方向反了（要证大于却放了小于）


> **总评**：数列的"等差等比"是保底分，递推和求和是中档题，放缩和综合是拔高题。构造法和裂项相消是最实用的两大技巧。
