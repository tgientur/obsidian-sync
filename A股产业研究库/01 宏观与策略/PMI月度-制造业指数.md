# PMI月度-制造业指数

PMI月度-制造业指数是官方制造业PMI的完整称呼，由国家统计局每月最后一天发布。它是衡量中国制造业景气程度最核心的月度先行指标。

## 与PMI的关系

- **PMI** 是统称，涵盖制造业PMI、非制造业PMI（商务活动指数）、综合PMI产出指数
- **PMI月度-制造业指数** 是狭义PMI，即制造业采购经理人指数

## 核心特征

- **调查样本**：覆盖31个制造业行业大类，3000+家企业
- **五大分项**：新订单、生产、从业人员、供应商配送时间、原材料库存（配送时间为逆指标）
- **权重**：新订单30%、生产25%、从业人员20%、供应商配送15%、原材料库存10%

## 细分关注

| 子指标 | 意义 |
|:-------|:-----|
| 新订单指数 | 反映未来需求，领先一般3-6个月 |
| 生产指数 | 反映当期生产活跃度，即期指标 |
| 出厂价格指数 | 与PPI联动 |
| 原材料购进价格指数 | 预判PPI原材料方向 |
| 产成品库存指数 | 判断库存周期位置 |
| 新出口订单指数 | 出口的先行指标 |
