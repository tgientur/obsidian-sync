# SiC（碳化硅）——半导体产业链视角

> 本文件聚焦**半导体产业链视角**下的SiC：衬底材料、外延、器件制造端的国产替代与技术突破。

## SiC在半导体产业链中的位置

SiC属于**第三代宽禁带半导体材料**，在半导体产业链中涵盖以下环节：

```
SiC衬底（材料） → SiC外延（材料/制造中间环节） → SiC芯片制造（晶圆制造） → 功率封装（封装测试） → 系统应用
     ↓                    ↓                         ↓                        ↓
  天岳先进             天岳先进                  士兰微/华润微            斯达半导/时代电气
  天科合达（未上市）     斯达半导                  芯联集成                 比亚迪半导体
```

## 核心技术壁垒

SiC在半导体制造端的核心壁垒在**衬底环节**：

1. **长晶难度高**：SiC单晶需在2500°C+高温下生长，控制晶体缺陷（微管/位错）极其困难。6英寸衬底良率仅60-70%（数据来源：[1]），远低于硅片的95%+
2. **尺寸升级慢**：从4英寸→6英寸用了10年，从6英寸→8英寸仍在爬坡中。Wolfspeed 8英寸衬底2024年量产，国内天岳先进2025年突破
3. **外延缺陷控制**：SiC外延层缺陷密度直接影响MOSFET栅氧化层可靠性（Vth漂移/击穿）

## 投资逻辑（半导体视角）

1. **衬底国产替代是SiC产业链最确定的主线**：SiC衬底占器件成本40-50%（数据来源：[1]），Wolfspeed/II-VI/Coherent三家占据全球>80%市场份额。天岳先进8英寸突破为国内SiC器件企业提供了"国产衬底+国产器件"的全链路自主基础。
2. **8英寸衬底是下一轮竞争焦点**：从6英寸到8英寸，芯片产出面积增加78%（数据来源：[1]），成本下降20-30%。率先量产8英寸衬底并提升良率的企业将获得下一代竞争优势。
3. **IDM模式在SiC领域的优势**：SiC器件的设计与制造工艺高度耦合（栅氧化层/离子注入/高温退火等），IDM企业（士兰微/华润微）在工艺know-how积累上有优势。

## 主要风险

1. **周期下行风险**：全球半导体周期可能进入下行阶段
2. **技术追赶风险**：国产替代进度低于预期
3. **设备管制风险**：荷兰/日本对华半导体设备出口管制加码
4. **产能过剩风险**：成熟制程产能扩张可能导致价格战
