# 文献索引模块

## 定位

建立产业研究文献元数据库。只存元数据卡片（标题/作者/DOI/链接/摘要），不存原文内容。形成可检索的研究资料体系。

## 收录范围

- 学术论文
- 行业白皮书
- 券商深度报告
- 监管机构研究报告
- 行业协会研究报告
- 上市公司研究资料

## 每篇文献记录格式

```yaml
标题: 
作者: 
机构: 
发表年份: 
DOI: 
原始链接: 
摘要: 
关键词: 
所属产业: 
引用格式: 
研究价值评价: 
```

## 引用格式标准

GB/T 7714-2015

## 原则

- 优先保留原始文献来源
- 不得仅保留二次转载内容
- 不存储受版权保护的全文
