# 大模型（大语言模型/基础模型）

## 定义与产业定位

大模型（Large Model），更准确地称为"基础模型"（Foundation Model），是指在大规模数据集上预训练、具有海量参数（百亿至万亿级）的深度学习模型。其核心特征是——通过Transformer架构的自注意力机制，在无标注数据上进行自监督预训练（预处理），获得对语言/视觉/多模态数据的通用理解能力，再通过微调、RLHF等方式适配具体应用场景。

大模型的产业爆发以2022年11月30日OpenAI发布ChatGPT为标志。ChatGPT在GPT-3.5（1750亿参数）基础上引入对话交互界面，60天月活突破1亿，刷新互联网产品增长纪录。2023年3月GPT-4发布（多模态、推理能力大幅提升），2024年5月GPT-4o发布（全模态实时交互），2025年GPT-5预期发布。这场"模型竞赛"的参与者范围从OpenAI一家扩展到全球科技巨头、创业公司、研究机构的多方博弈。

**产业定位**：大模型处于AI产业链的"应用层前端"，向下连接算力基础设施（GPU/HBM/服务器/数据中心），向上支撑AI应用（ChatGPT/搜索/办公/编程/视频/机器人）。大模型本身是"AI的基因"——它不直接创造收入，但它是所有AI应用的基础能力层。没有大模型，AI行业的所有算力投资将失去价值。

**资本市场关注度**：大模型在A股的映射方式与硬件层完全不同。A股没有OpenAI、Anthropic这样的直接标的，投资者通过以下方式参与大模型产业链：
1. **参股公司**：昆仑万维（投资OpenAI相关/自研大模型）、三六零（自研大模型）、科大讯飞（星火）
2. **算力基础设施**：训练大模型需要大量GPU → 算力租赁（中贝通信/拓维信息/鸿博股份）
3. **应用层**：大模型能力提升带动AI应用（金山办公/万兴科技/虹软科技）
4. **数据服务**：大模型训练需要高质量数据（海天瑞声/拓尔思）

大模型在A股的投资逻辑以"概念催化→实际应用验证"为主线，情绪驱动远强于业绩驱动。

## 产业链分析

### 上下游关系

```
上游（算力GPU + 数据 + 算法人才） → 大模型研发/训练 → 模型托管/推理 → AI应用
  利润份额：~45%                     利润份额：~30%       ~10%          ~15%
```

**上游**：
- **算力**：大模型训练的核心成本。GPT-4训练约消耗$1-2亿算力成本（25000张A100×90天）。Llama 3 405B训练成本约$1.5亿。算力成本是大模型行业最大的进入门槛。上游以NVIDIA GPU为核心，直接决定大模型的训练进度和成本。
- **数据**：预训练数据质量和多样性直接影响模型能力。OpenAI/Google/Anthropic使用TB级高质量文本（Common Crawl、Books、Wikipedia、Reddit、学术论文等）。2024年起高质量公开数据趋于枯竭——"数据墙"问题成为行业共识。合成数据和专有数据成为下一个竞争焦点。
- **算法人才**：顶尖AI研究员年薪$100-300万。DeepSeek V2的论文在2024年引发全球讨论——用MoE架构在更低的训练成本（约$500万）下达到接近GPT-4的性能。人才之争从"有多少研究员"转向"有多聪明的架构创新"。

**中游**：
- **大模型研发**（训练）：从零训练一个大模型需要$500万-2亿的算力成本+6-12个月的时间。全球有能力从零训练大模型的团队不超过20个（OpenAI/Google/Anthropic/Meta/Mistral/DeepSeek/通义千问/文心等）。
- **模型推理/部署**（工程）：训练完成后，将模型部署到推理服务器上对用户提供服务。推理成本随模型规模和技术进步快速下降——GPT-4推理成本从2023年的每百万tokens $30降至2025年的$2-5（降价90%+）。

**下游**：
- **AI应用**：基于大模型开发的各种应用，包括ChatGPT、Copilot、Claude、Gemini、通义千问、豆包、Kimi等。应用层的核心竞争在于产品体验和用户粘性，而非模型能力本身。Kimi（月之暗面）凭借长文本能力在2024年迅速崛起，豆包（字节跳动）通过免费策略在2025年实现日活1亿。

### 价值流动方向

大模型产业链的价值分配仍在剧烈变化中，尚未形成稳态。当前趋势：

- **算力层**（NVIDIA + 云厂商）：拿走最大份额。训练成本虽在大幅下降（硬件效率提升+模型架构优化），但总市场支出（投资）更大（更多模型+更大模型）。
- **大模型研发层**：大部分大模型公司尚未实现盈利（OpenAI 2024年收入~$37亿，亏损~$50亿）。商业化路径依赖API调用/订阅费/企业授权，但收入远小于算力投入。
- **AI应用层**：部分杀手级应用（GitHub Copilot年收入$20亿+、ChatGPT Monthly $20订阅费）已开始盈利。但整体来看，应用层的收入增长快速但竞争激烈。
- **数据层**：数据服务（标注/清洗/合成）在产业链中的价值占比在下降，因为高质量公开数据减少后，行业转向"模型自学"（自我改进/合成数据）而非人工标注。

### 利润分配格局

| 环节 | 当前利润分配 | 趋势 | 说明 |
|:----|:----------:|:----|:-----|
| 算力基础设施 | ~45% | 稳定 | NVIDIA垄断，但AMD/ASIC竞争开始 |
| 云厂商推理托管 | ~15% | 上升 | 推理需求爆发，云厂商提供模型托管（Amazon Bedrock/Azure OpenAI） |
| 大模型公司 | ~15%（大部分亏损中） | 上升 | 随着API调用上涨，规模效应将逐步显现 |
| AI应用层 | ~20% | 上升 | 垂直场景应用（代码/写作/客服）开始赚钱 |
| 数据/标注服务 | ~5% | 下降 | 合成数据+RLHF自动化降低人工标注需求 |
数据来源：[1][2] 相关年报/行业报告

### 定价权分析

大模型产业链的定价权高度集中在上游和部分应用层：

- **GPU厂商（NVIDIA）**：定价权最高。大模型训练算力需求每年增长10x+，NVIDIA GPU持续供不应求，大模型厂商几乎没有替代选择。即使DeepSeek通过MoE架构大幅降低训练成本，全球总训练算力需求仍在指数级增长。

- **云厂商（Azure/AWS/GCP）**：在"模型即服务"（MaaS）领域有定价权。OpenAI的API完全运行在Azure上，Amazon推出Bedrock托管Claude和Llama系列。云厂商收取15-20%的MaaS抽成，且控制推理基础设施的定价。

- **大模型公司（OpenAI/Anthropic）**：定价权正在增强。GPT-4发布后持续上调API价格（2024年GPT-4 API多次调价），但长期面临开源模型（Llama/Qwen/Mistral）和竞品（Claude/Gemini）的价格竞争压力。开源模型使大模型的"商品化"趋势加速。

- **应用层公司**：定价权弱，竞争以产品体验和用户粘性为主。ChatGPT月费$20是AI应用定价的天花板。

### 成本传导

大模型的成本结构呈现"前高后低"特征——训练成本极高但固定，推理成本随规模增长：

```
训练成本（一次性固定成本）：
    GPT-4：~$1-2亿
    Llama 3 405B：~$1.5亿
    DeepSeek V3（MoE）：~$500万
    Qwen 2.5-72B：~$300万
    趋势：每12-18个月单位训练成本下降50-70%（架构优化+硬件效率）

推理成本（变动成本，随用户数增长）：
    2023年：每百万tokens ~$30（GPT-4）
    2024年：每百万tokens ~$10（GPT-4o）
    2025年：每百万tokens ~$2-5（GPT-4o降本+开源模型竞争）
    趋势：每12个月推理成本下降80-90%
```

成本传导方向：GPU成本下降 + 模型架构优化（MoE/量化/蒸馏）→ 推理成本大幅下降 → API调用价格下降 → AI应用普及加速 → 用户/调用量增长 → 总成本（固定+变动）被更多用户分摊。

大模型行业有两个关键的价格弹性节点：①推理成本降到$0.1/百万tokens以下时，AI应用的TAM开始放大；②降到$0.01/百万tokens时，AI能力基本可以嵌入所有软件场景。

## 商业模式分析

### 收入来源

大模型公司的商业模式尚在探索中，当前的主要收入来源包括：

| 收入来源 | 说明 | 占比（OpenAI 2024） | 代表产品 |
|:--------|:----|:-----------------:|:--------|
| API调用费 | 按tokens计费，开发者调用模型的付费收入 | ~45% | GPT-4o API、GPT-4-turbo |
| 订阅费 | C端用户的按月/按年付费 | ~35% | ChatGPT Plus ($20/月)、Team ($25/人/月) |
| 企业授权 | 大客户定制/私有化部署 | ~15% | OpenAI Enterprise、Azure OpenAI |
| 模型微调 | 在基础模型上定制微调服务 | ~3% | GPT Fine-tuning |
| 其他 | 模型输出处理（图像生成/语音等） | ~2% | DALL-E、Whisper API |

数据来源：[1] OpenAI 2024年收入结构（公开估算）

**收入结构的关键特征**：
- API调用是最大收入来源，但毛利率低（推理成本占API收入约40-50%）
- C端订阅毛利率最高（ChatGPT Plus推理成本约$5-8/用户/月 vs 订阅费$20/月）
- 企业授权客单价高（$10-50万/年）但销售周期长（3-6个月）
- 所有大模型公司目前均未实现GAAP盈利

### 成本结构

| 成本项 | 占比（OpenAI估计） | 说明 |
|:------|:---------------:|:-----|
| 推理计算成本 | 30-40% | GPU租赁/自建推理集群，随用户增长线性增加 |
| 训练计算成本（摊销） | 15-20% | 爬坡训练+实验训练，按年摊销 |
| 研发人力（薪酬） | 20-25% | 顶尖AI研究员年薪$100-300万 |
| 云基础设施（非计算） | 10-15% | 网络+存储+带宽+CDN |
| 销售/管理 | 5-8% | 企业销售团队 |
| 数据成本 | 3-5% | 数据采购+标注+清洗 |
| 合规/安全 | 2-3% | RLHF安全训练+内容审核 |

数据来源：[2] The Information、行业综合估算

**关键成本特征**：推理成本占比最高且快速增长。随着用户数和调用量指数级增长，推理成本超过训练成本成为最大开支。OpenAI 2024年推理成本估计超过$40亿（占收入的>100%），这是大模型公司"赔本赚吆喝"的核心矛盾。

### 产业分工模式

大模型产业的参与者可划分为几个层次：

- **闭源模型巨头**（OpenAI/Google/Anthropic）：自研自训先进闭源模型，API+订阅+企业授权多通路变现。处于行业最前沿，定位于AGI研发。

- **开源模型阵营**（Meta Llama / Alibaba Qwen / Mistral / 智谱GLM）：开源模型权重，建立开发者社区和生态。通过企业服务/云托管/咨询变现。对闭源模型形成价格压力。

- **云厂商MaaS**（Azure/AWS/GCP/阿里云/百度云）：提供大模型托管服务（模型市场），收取15-20%的抽成+AWS/Azure基础资源使用费。核心逻辑是将模型作为云平台的流量入口。

- **AI应用公司**（Cursor/Perplexity/Notion AI/金山办公）：基于大模型API开发特定场景应用。产品力+数据飞轮是护城河，但不拥有核心模型能力。

- **中国大模型公司**（百度/阿里/腾讯/字节/科大讯飞/月之暗面/智谱/百川/零一万物）：中国版的大模型竞争，格局更加分散。百度文心、通义千问、豆包、讯飞星火、Kimi等各具差异化能力。

### 利润来源特征

大模型公司当前的核心矛盾：**收入增长极快，但亏损更大**。

- OpenAI 2024年收入~$37亿（YoY+200%），亏损~$50亿（推理成本+人力+研发）
- Anthropic 2024年收入~$8亿，亏损~$20亿
- 中国大模型公司（月之暗面/智谱/百川等）均处于烧钱阶段，尚未形成明确的盈利时间表

利润拐点的关键变量：推理成本下降速度 > 用户量增长速度 → 单位用户亏损收窄 → 盈平衡点。预计2026-2027年头部大模型公司将实现Operating Profit非GAAP盈利。

### 现金流特点

- **负经营现金流**：大模型公司普遍处于"烧钱"状态——Capex（GPU采购/租赁）+人力薪酬+推理成本远超API收入+订阅收入
- **融资依赖**：OpenAI累计融资超$200亿（包括微软投资），Anthropic~$80亿，月之暗面~$10亿。大模型是典型的"大资本消耗"行业
- **资本开支强度**极高：单次训练$500万-2亿，GPU采购$5-20亿/年
- **折旧压力**：GPU租赁费用/设备折旧是固定开支，即使模型使用率低也需支付
- **预付费收入**：企业订阅/API预付费模式正在推广，规模效应下现金流有望改善

## 行业壁垒分析

### 技术壁垒

**训练技术**
从零训练一个GPT-4级别的大模型需要解决数十个工程和算法难题：分布式训练（万卡GPU同步）、损失收敛（优化器/学习率/模型结构）、MoE路由负载均衡、长上下文（128k-1M tokens）、多模态对齐等技术栈。全球具备从零训练顶级大模型的团队不超过20个。

**架构创新**
2024-2025年，大模型技术路线出现重要分化：
- MoE（Mixture of Experts）架构：DeepSeek V3/R1、Qwen2.5-MoE、Mixtral等采用。在同样训练成本下性能更高，但推理时路由负载均衡复杂
- 状态空间模型（Mamba/Griffin）：替代Transformer的线性复杂度方案，长上下文任务有优势
- 扩散语言模型：非自回归生成，未来可能改变推理效率
- 多模态统一架构：GPT-4o/Claude 3/Gemini将文本/图像/音频统一处理

技术创新是大模型公司最核心的竞争维度。DeepSeek V2/V3的出现证明——200人团队+创新架构可以挑战万亿美元公司的AI能力。

**推理优化**
推理速度 = 用户体验 = 商业变现。KV Cache优化、Speculative Decoding、INT4/INT8量化、Flash Attention 2/3等推理优化技术直接影响运营成本和用户体验。头部大模型公司在这一领域建立了大量专利和工程积累。

### 资本壁垒

大模型行业已经演变为"大资本游戏"，新进入门槛极高：

- 训练一次GPT-4级别模型：$1-2亿（算力成本）
- 训练一次Llama 3 405B：~$1.5亿
- 年推理成本（百万用户级别）：$5-10亿
- 顶尖AI团队年薪：50-100人团队×$50-100万/年 = $2500万-1亿/年
- GPU集群采购：5000-25000张H100 = $2-15亿

即使通过DeepSeek级架构优化将训练成本降低至$500万，后续的推理成本和人力成本仍是长期支出。中国大模型公司的融资记录（百川~$3亿、智谱~$20亿、月之暗面~$10亿）印证了资本门槛的高企。

### 人才壁垒

大模型行业的人才稀缺性可能是所有壁垒中最硬的：
- 全球有能力设计MoE架构的研究员估计不超过300人
- 万卡级分布式训练系统工程师不超过500人
- 具备RLHF/DPO对齐训练经验的研究员不超过200人
- 年薪$100-300万在顶级AI研究员中已经成为常态

OpenAI前员工创办Anthropic/Perplexity/Cursor等公司的现象，说明人才壁垒与组织壁垒的矛盾——顶级人才无法被"垄断"。

### 数据壁垒

高质量预训练数据的稀缺性正在上升：
- 公开高质量文本数据（书籍/论文/代码/新闻）已被头部公司基本爬取完毕
- "数据墙"预计在2026-2027年出现——届时公开数据质量不足以支持下一代模型提升
- 合成数据（模型自我生成数据）成为突破方向，但验证方法尚不成熟
- 专有数据（企业生产数据/用户交互数据）成为差异化竞争的核心壁垒
- 华为/百度/阿里在中文数据集方面有独特优势（高质量中文语料）

### 生态壁垒

大模型生态正在形成类似CUDA的锁定效应：
- **开发者绑定**：OpenAI的API是所有AI应用的基础设施，切换模型需要重写prompt和微调逻辑
- **用户粘性**：ChatGPT/Copilot的用户使用习惯和数据积累形成退出成本
- **行业标准**：GPT生态正在成为"AI的操作系统"——大量工具（LangChain/AutoGPT/Ollama）围绕GPT API构建
- **但是**——开源模型（Llama/Qwen）提供的"本地部署+数据隐私"方案，在B端市场对闭源模型形成竞争。开源/闭源之争尚无定论。

## 历史行情复盘

### 第一期：2023年1月-6月——ChatGPT概念引爆

**事件**：2022年11月30日ChatGPT发布，60天月活破亿。2023年3月14日GPT-4发布。中国市场"百模大战"启动——百度文心一言（3月）、阿里通义千问（4月）、科大讯飞星火（5月）、商汤日日新（4月）相继发布。同花顺AI概念指数在2023年2-4月累计涨幅超过60%（数据来源：[1]）。

**A股市场表现**：A股AI概念全面爆发。昆仑万维从10元涨至50元（+400%），逻辑是"大模型+游戏+出海"三重概念。三六零从每日$7涨至$18（+157%），逻辑是"周鸿祎讲大模型故事+AI搜索替代预期"。科大讯飞从30元涨至80元（+167%），逻辑是"星火大模型+教育AI"。海天瑞声（数据标注）从20元涨至100元。

**上涨原因**：ChatGPT带来的"认知冲击"——市场第一次看到AI能力的质变。大模型概念没有任何基本面支撑，纯粹是"产业趋势预期的估值抬升"。A股所有宣称自研大模型的公司都被爆炒。

**资金结构**：游资+散户主导，公募在2023Q2才开始配置AI软件方向。昆仑万维/三六零的龙虎榜数据显示游资接力现象明显。

**最终结果**：2023年6月后大模型概念急剧退潮。昆仑万维从50元跌至25元（-50%），三六零腰斩。原因：①缺乏业绩兑现——所有大模型公司均未能变现；②国内大模型性能明显落后GPT-4；③监管收紧（生成式AI管理办法2023年8月施行）。

### 第二期：2024年2月-6月——Sora与Kimi催化

**事件**：2024年2月，OpenAI发布Sora（文生视频模型），震惊业界。月之暗面Kimi在2024年3月凭借200万token长文本能力走红，日活从10万飙升至100万+。百度文心、阿里通义、字节豆包跟进长文本竞赛。

2024年Q1，大模型API价格战爆发。字节豆包宣布免费调用，阿里通义千问API降价97%（数据来源：[1]），百度文心一系列API免费。降价逻辑：推理成本大幅下降（硬件效率+KV Cache优化+量化压缩），企业希望通过免费开放获取更多用户和数据反馈。

**A股市场表现**：游戏传媒板块在Sora催化下走强——文生视频概念带动。Kimi概念的掌阅科技（AI阅读+长文本）大涨。但大模型相关标的的整体涨幅远低于2023年第一波，说明市场的"新鲜感溢价"已经消退。

**资金结构**：北向资金在2024Q2大量卖出A股科技（美元升值背景下），AI板块缺乏增量资金。交易以存量博弈为主。

### 第三期：2025年至今——DeepSeek冲击与推理需求爆发

**事件**：2024年底至2025年，DeepSeek V3/R1横空出世——以不到$1000万的训练成本达到接近GPT-4o的性能，引发全球震动。DeepSeek"低成本高效率"的叙事重塑了市场对大模型的认知——"大模型不再是大公司的专利"。

2025年Q1，推理算力需求首次超过训练算力需求（占总算力55%）。国产大模型能力全面接近GPT-4o水平（DeepSeek V3/通义2.5/豆包/文心4.0）。推理成本降至$0.1-0.3/百万tokens。

**A股市场表现**：AI应用层（非模型本身）成为主线。金山办公（AI Office）从200元涨至400元（+100%），万兴科技（AI视频编辑）从40元涨至70元（+75%），虹软科技（AI图像处理）表现亮眼。

算力租赁方向（中贝通信/鸿博股份）因为推理需求爆发再次走强。大模型概念本身（昆仑万维/三六零/科大讯飞）整体横盘甚至下跌——市场认知从"概念"切换到"竞争格局"，大模型"百模大战"实际是一场烧钱消耗战，而非蓝海。

**经验总结**：
1. 大模型的二级投资与一级投资完全相反——一级融资持续活跃（OpenAI $2000亿估值），二级投资者缺少直接标的
2. A股大模型概念的投资价值在快速衰减——"百模大战"的利润最终被GPU和云厂商拿走
3. 大模型降本（推理成本下降90%+）最大受益方向是AI应用和推理算力
4. "卖铲子"的逻辑在大模型赛道同样成立——GPU/算力租赁/云托管环节确定性高于模型本身

## 投资逻辑分析

### 为什么产生超额收益

大模型在A股的投资逻辑与硬件层（GPU/HBM/光模块）有本质区别——A股大模型相关标的的超额收益来自：

1. **叙事溢价**：大模型代表的"AI时代的操作系统"级叙事，每次革命性的技术突破都带来一轮主题投资。ChatGPT发布、GPT-4发布、Sora发布、DeepSeek发布——这些事件性的认知冲击，带来了A股"大模型概念"的脉冲上涨。

2. **降本受益逻辑**：大模型推理成本的大幅下降（2023-2025年下降90%+），直接利好AI应用和推理算力——推理算力需求暴增拉动A股算力租赁/数据中心标的。

3. **国产替代预期**：DeepSeek证明了"低成本训练精品模型"是可行的，国产大模型的概念并非完全无价值。

### 景气度传导

```
模型能力突破（如DeepSeek/GPT-4o） → 推理成本下降 → AI应用普及加速
                                                   ↓
                                     推理算力需求爆发 → 算力租赁（A股标的）受益
                                                   ↓
                                     应用层公司收入增长
```

大模型能力提升→成本下降→应用普及→算力需求增长，这是大模型产业链的景气传导链。

### 业绩兑现路径

大模型产业链的业绩兑现路径各异：

- **大模型公司**（A股无直接标的）：API调用量增长→收入增长→规模效应→单位成本下降→盈利拐点。预期2026-2028年实现
- **AI应用层**：大模型降本→应用毛利率改善→用户增长→盈亏平衡。当前AI应用大部分仍处于"付费用户增长但整体亏损"的阶段
- **算力租赁**：推理GPU租赁需求爆发→出租率提升→收入增长。业绩兑现最直接

### 未来催化剂

1. **GPT-5发布**：能力再次质变，引发新一轮AI概念行情
2. **DeepSeek级开源模型更新**：低成本高性能的叙事持续冲击市场对大模型的定价认知
3. **AI应用爆发（杀手级应用出现）**：谁是"AI时代的Killer App"——代码（Cursor/Devin）、搜索（Perplexity）、办公（Copilot/M365）、Agent（AutoGPT）
4. **推理成本$0.01/百万tokens节点**：AI能力真正嵌入所有软件场景
5. **中国AI模型出口能力验证**：DeepSeek在全球的下载量和技术影响力已在验证国产大模型能力

## 主要风险

1. **技术迭代风险**：GPU架构代际升级加速，落后一代即面临市场份额大幅下滑
2. **供应链集中风险**：先进制程/CoWoS/HBM产能高度集中于台积电/SK海力士
3. **地缘政治风险**：美国对华AI芯片出口管制可能进一步升级
4. **估值泡沫风险**：AI算力板块估值处于历史高位，业绩不及预期将面临戴维斯双杀
5. **需求波动风险**：云厂商Capex增速放缓可能导致订单低于预期

## 政策法规

### 核心政策

- **《生成式人工智能服务管理暂行办法》**（国家网信办等七部门，2023年7月13日发布，2023年8月15日施行）：中国首个生成式AI监管法规。要求大模型服务提供者履行算法备案、内容审核、数据安全义务。来源：国家网信办官网（www.cac.gov.cn）
  **链接**：https://www.cac.gov.cn/2023-07/13/c_1690898327029107.htm
  [资料卡片:: [[A股产业研究库/14 资料库/01 政策文件/生成式AI管理办法]]

- **《新一代人工智能发展规划》**（国务院，2017年7月）：2020年、2025年、2030年三步走战略。到2025年AI产业规模超4000亿，带动相关产业超5万亿。来源：中国政府网

- **《关于加快场景创新以人工智能高水平应用促进经济高质量发展的指导意见》**（科技部等六部门，2022年8月）：推动AI在各行业应用场景落地。

- **《深度合成服务管理规定》**（网信办/工信部/公安部，2022年12月）：对AI生成内容（AIGC）进行标识要求和内容审查。

- **《关于科技伦理审查管理的办法》**（科技部等十部门，2023年9月）：要求AI大模型开发进行伦理审查。

- **算法备案制度**（网信办持续执行，2022年3月起）：所有大模型和AIGC产品需要完成算法备案，2024-2025年备案加速。

### 监管影响判断

中国对生成式AI的监管思路为"发展与安全并重"——在保障内容安全的前提下鼓励创新。大模型公司的合规成本在增加（算法备案+内容审核+安全训练+数据合规），但监管不确定性的最大风险期已经过去（2023年暂行办法出台后）。2025年以来监管环境趋于稳定。

## 舆论风向

### 雪球/股吧

- "大模型是泡沫还是未来"——从2023年争论到2025年，没有定论
- "DeepSeek能打破OpenAI的垄断吗"——2024-2025年最热话题
- "A股大模型概念都是骗子"——部分投资者的高度怀疑态度
- "昆仑万维能不能靠大模型翻身"——围绕具体标的的多空争论

### 知乎/技术社区

- "GPT-4o的技术细节拆解"——AI社区的技术讨论核心
- "MoE架构和稠密模型到底谁更优"——DeepSeek和Llama的技术路线之争
- "RLHF和DPO的对齐技术对比"——模型安全训练的前沿讨论
- "大模型训练的数据墙问题"——行业发展的深层挑战

### 微博/财经大V

- "AI改变世界的节奏"——长期积极派vs短期质疑派
- "买OpenAI股票（没上市）还是买英伟达"——AI赛道最经典的问题
- "DeepSeek概念能涨多久"——游资关注的核心话题

### 争议性话题

1. 大模型是否已经进入"商品化"——开源模型=免费的基础设施，闭源模型（GPT-4/Claude）的长期护城河是什么
2. DeepSeek低成本训练是否真实——训练成本$500万是否包含了隐形成本（数据/人力/GPU调试）
3. AI安全——大模型能力越强，安全风险越大，监管是否会过度限制
4. AGI时间表——2025年还是2030年还是更远，这一分歧直接影响大模型公司的估值上限
5. "百模大战"的终局——中国大模型市场最终剩几家（3家还是5家）
